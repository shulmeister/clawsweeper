## Summary

Keep open. Current `main` still intentionally enforces owner-only `0o600`/`0o700` permissions across the write paths named in the issue, and the repo’s docs/tests/changelog treat that as a security contract. There is no implemented config/env knob for shared-group modes, so this is still an open product/security tradeoff rather than something already fixed or safely auto-closeable.

## Evidence

- **json-file helper hardcodes secure modes:** `saveJsonFile()` uses `JSON_FILE_MODE = 0o600` and `JSON_DIR_MODE = 0o700`, then reapplies `chmodSync` after rename; there is no config or env lookup in this path.
  - file: [src/infra/json-file.ts:5](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/src/infra/json-file.ts#L5)
  - sha: [95b7a85f0607](https://github.com/openclaw/openclaw/commit/95b7a85f0607784d8e33b4ce7e42617174196490)
- **atomic JSON/text writer defaults to 0600:** `writeTextAtomic()` defaults `mode` to `0o600`, creates temp files with that mode, and re-chmods the final file after the atomic replace.
  - file: [src/infra/json-files.ts:56](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/src/infra/json-files.ts#L56)
  - sha: [95b7a85f0607](https://github.com/openclaw/openclaw/commit/95b7a85f0607784d8e33b4ce7e42617174196490)
- **device auth store remains owner-only:** `writeStore()` writes `device-auth.json` with `{ mode: 0o600 }` and then calls `chmodSync(filePath, 0o600)` best-effort.
  - file: [src/infra/device-auth-store.ts:37](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/src/infra/device-auth-store.ts#L37)

- **cron persistence still hardens files and dirs:** `saveCronStore()` goes through `atomicWrite()`, which chmods the cron directory to `0o700`, writes temp files with `0o600`, and chmods the final file to `0o600`.
  - file: [src/cron/store.ts:226](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/src/cron/store.ts#L226)
  - sha: [95b7a85f0607](https://github.com/openclaw/openclaw/commit/95b7a85f0607784d8e33b4ce7e42617174196490)

  - file: [CHANGELOG.md:3351](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/CHANGELOG.md#L3351)

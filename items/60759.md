## Summary

Keep open. Current main still has the reported qqbot/webhooks plugin schemas with local `$defs` and root-style `$ref` pointers, while the config schema merger copies plugin schemas into nested `plugins.entries.<id>.config` without hoisting or rewriting those definitions. The related PRs in the provided context were closed unmerged, so there is no merged canonical fix to close against.

## Summary

Current main already covers this: the chat model picker is not gated on conversation images, and Gateway/WebChat now preserves image attachments for text-only models by offloading them as `media://inbound/*` refs instead of blocking the switch path. The proving fix landed in commit `86f69ba5a06ae256906c3a26308801e0b84e0091` on April 24, 2026, after the latest released tag `v2026.4.22`, so this appears implemented on main but not yet in a shipped release tag.

## Summary

Keep #71734 open. Current main has normal priority-ordered `before_tool_call` hooks and static tool-display metadata, but it does not implement the requested trusted pre-plugin policy stage or plugin-owned tool metadata registration surface. The related RFC PR #71731 is still open and explicitly described as docs-only in the supplied GitHub context, so it does not supersede the remaining implementation work.

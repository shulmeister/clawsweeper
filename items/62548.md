This PR is a docs-only listing/setup page for a third-party OpenAI-compatible provider. Current OpenClaw already has generic configured-provider support and a provider-plugin API that can cover CheapestInference without a new core docs entry, while current scope docs route third-party plugin/provider discovery and promotion through ClawHub. The prior review comments were useful and appear addressed in the PR head, but they do not change the product boundary.

## Best Possible Solution

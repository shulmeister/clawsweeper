## Summary

PR #62548 is a docs-only provider-directory addition for CheapestInference, a third-party OpenAI-compatible inference proxy. The useful setup path is already supported by current OpenClaw through generic `models.providers` custom/base-URL config, and deeper integration can use the existing provider plugin API. Current scope docs put third-party provider/plugin discovery and promotion in ClawHub rather than core docs; the PR does not identify a missing core extension API. The prior bot review comments were useful and appear addressed in follow-up commits, so the close reason is scope, not the earlier broken snippet.

## Best Possible Solution

Close this PR as a ClawHub/community-provider fit. CheapestInference users can use the existing `models.providers` custom OpenAI-compatible config today; if CheapestInference wants stronger OpenClaw integration or discoverability, publish a small external provider plugin or setup bundle through ClawHub using the existing provider plugin API. Core docs should stay focused on the generic provider extension/configuration surface unless a maintainer intentionally curates this provider.

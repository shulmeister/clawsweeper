## Summary

Keep open. Current main still contains the reported Ollama web-search behavior: it calls the experimental endpoint, treats the Ollama Cloud model base URL as a reason to fall back to localhost, and advertises the provider as keyless while current Ollama docs require an API key for the cloud web-search API.

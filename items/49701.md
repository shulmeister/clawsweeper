## Summary

Current `main` already contains the Browserless CDP docs update in a stronger form: the English source doc uses a `wss://` example, explains that OpenClaw supports both Browserless HTTPS discovery and direct WebSocket roots, and this shipped in `v2026.4.22`. The PR’s direct `docs/zh-CN/**` edit is also incompatible with current docs policy, which says localized docs are not maintained in this repo.

## Summary

Keep open. Current main has partial prompt guidance for tool discipline, conflict handling, live verification, and a GPT-5-specific behavior overlay, but it does not implement the requested systematic Agent Behavior Principles as a global system-prompt component or equivalent all-runtime contract. The related #71937 RFC was design-only, closed as not planned, and lists #69291 as related rather than superseding it.

## Best Possible Solution

Keep #69291 open for maintainer triage. The likely path is either a compact global behavior contract in `src/agents/system-prompt.ts` with tests and docs, or an explicit maintainer decision that these remain provider overlays/workspace guidance. Any implementation should define which items are soft prompt guidance versus hard policy and preserve minimal prompt mode/cache-boundary behavior.

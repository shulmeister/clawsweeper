## Summary

Keep open. Current main still supports session reset configuration globally, by chat type, and by channel, but not per Telegram group. The related PR #8134 was closed unmerged, and #20888 covered broader stale/oversized session lifecycle work rather than this per-group override.

## Evidence

- **Telegram group config has no session field:** `TelegramGroupConfig` currently lists group-level options such as `requireMention`, `groupPolicy`, tools, topics, prompts, and error policy, but no `session` or reset override field.

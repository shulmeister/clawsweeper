## Summary

Keep open. Current `main` exposes model setup in the dashboard only through generic config editing and a debug catalog view, while the documented local-model flow still depends on CLI/onboarding and external Ollama model pulls. This is a recent enhancement request with follow-up interest, so there is not strong evidence for any conservative auto-close reason.

## Summary

Current main has adjacent memory improvements, but it does not implement the requested guaranteed short-session persistence. The pre-compaction memory flush is still gated by context/transcript thresholds and explicitly excludes heartbeat runs; session transcript search and dreaming are opt-in/background paths, not a default session-close or heartbeat durable memory flush.

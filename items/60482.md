## Summary

Keep open. Current main avoids Tailscale reset for the default gateway path, but the issue's requested invariant is still not fully true: when `gateway.tailscale.resetOnExit` / `--tailscale-reset-on-exit` is enabled, gateway restart still runs a cleanup hook that calls the global `tailscale serve reset`, which can remove non-OpenClaw Serve bindings.

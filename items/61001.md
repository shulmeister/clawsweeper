## Summary

Keep open. Current main has useful partial mitigations for bootstrap token pressure, but it does not implement the requested per-file conditional bootstrap loading by session type. Normal/direct/group sessions still receive the fixed bootstrap set, and the public config schema exposes only coarse injection/truncation knobs plus heartbeat/subagent lightweight modes.

## Summary

Keep open. Current main still does not accept WhatsApp Newsletter/Channel JIDs: `@newsletter` is absent from the WhatsApp code/docs, normalization rejects non-user/non-group JIDs containing `@`, and the message tool still advertises only E.164 or group JID targets. The related PR #13424 was closed unmerged and does not supersede the issue.

## Summary

Current main already builds OpenClaw tool inventories for embedded/main-agent turns, passes them through Codex/OpenAI schema normalization, and includes them in `openai-codex-responses` payloads. It also defaults GPT-5 Codex runs to `parallel_tool_calls=true` and adds regression coverage for both Codex payload injection and the main-session plan-only/empty-result failure mode described in the issue discussion.

## Summary

Current `main` has explicit npm-package integrity guards and install-time dist cleanup for the exact failure class reported here: prepack now writes a packaged dist inventory, postinstall prunes stale hashed `dist/**` files, global verification fails closed for `>=2026.4.15` installs with missing/invalid inventory, and tests cover missing/stale packaged files. One logged symptom (`@homebridge/ciao` missing from core) is also gone because Bonjour is now a bundled plugin with its own dependency.

## Evidence

- **Prepack now writes package dist inventory before npm packing:** `scripts/openclaw-prepack.ts` verifies prepared build artifacts, then calls `writePackageDistInventory(process.cwd())` before the pack smoke step. That creates an authoritative packaged-file list for npm installs.
  - file: [scripts/openclaw-prepack.ts:108](https://github.com/openclaw/openclaw/blob/b13545355d42701a1b2a5838b18101777c1bd808/scripts/openclaw-prepack.ts#L108)
  - command: `nl -ba scripts/openclaw-prepack.ts | sed -n '1,160p'`
  - sha: [b13545355d42](https://github.com/openclaw/openclaw/commit/b13545355d42701a1b2a5838b18101777c1bd808)
- **Postinstall prunes stale dist files from packaged installs:** `scripts/postinstall-bundled-plugins.mjs` reads `dist/postinstall-inventory.json`, lists installed `dist/**` files, removes anything not in the inventory, and logs `pruned stale dist files`. This directly targets mixed-version hashed chunk layouts like the issue's missing `config-*`, `delivery-queue-*`, and similar imports after `npm install -g --force`.
  - file: [scripts/postinstall-bundled-plugins.mjs:281](https://github.com/openclaw/openclaw/blob/b13545355d42701a1b2a5838b18101777c1bd808/scripts/postinstall-bundled-plugins.mjs#L281)

  - sha: [b13545355d42](https://github.com/openclaw/openclaw/commit/b13545355d42701a1b2a5838b18101777c1bd808)

  - file: [src/infra/package-dist-inventory.ts:154](https://github.com/openclaw/openclaw/blob/b13545355d42701a1b2a5838b18101777c1bd808/src/infra/package-dist-inventory.ts#L154)

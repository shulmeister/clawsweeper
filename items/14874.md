## Summary

Keep #14874 open. Current main does not implement optional SIE verification or warn/strict enforcement for skill loading, and the linked PR #14893 was closed unmerged. The remaining request is a core skill-loader security/policy capability, not something current ClawHub skill/plugin surfaces can fully provide without a loader hook or core/API change.

## Summary

Keep open. Current `main` still does not implement the requested same-line/partial duplicate cleanup or `</final>` boundary dedupe. The sanitizer only strips final tags and collapses exactly matching consecutive blocks separated by blank lines.

## Evidence

- **Final-tag stripping is still tag-only:** `stripFinalTagsFromText()` only removes `<final>` / `</final>` markers with `replace(FINAL_TAG_RE, "")`; it does not keep post-tag content or dedupe surrounding text.

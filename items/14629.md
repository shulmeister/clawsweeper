## Summary

Keep open. Current main still only collapses consecutive duplicate paragraph blocks separated by blank lines, while the issue requests same-line/single-newline duplicate handling, partial duplicate handling, and tag-boundary cleanup. The report is old enough to consider stale handling, but it includes a concrete affected version, model, channel, and reproduction shape, and the current source still matches the reported limitation.

## Summary

Current main already supports per-subagent tool deny/allow policy, and the correct config path is `tools.subagents.tools.{allow,deny}` rather than `agents.defaults.subagents.toolPolicy.*`. The policy is documented, typed in config schema, enforced during subagent tool-policy resolution, and covered for ACP child sessions as well.

## Summary

Current `main` already contains a concrete fix for the broken-release condition behind this report: pack/publish now fails closed if the Control UI bundle is missing, and the regression is covered by tests and release-process docs.

## Close Comment

Closing as implemented on `main` after a Codex review. There is matching regression coverage in `test/openclaw-prepack.test.ts:14` and tarball-level release coverage in `scripts/openclaw-npm-release-check.ts:62` plus `test/openclaw-npm-release-check.test.ts:292`. `CHANGELOG.md:2513` also records this as the fix for the broken Control UI release family.

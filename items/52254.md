## Summary

Current `main` still contains untranslated English strings inside the lazy-loaded Control UI tab views named in the report. The `zh-CN` bundle covers top-level tab labels/subtitles, but many strings in `Agents`, `Skills`, `Sessions`, `Nodes`, `Channels`, and `Debug` are hardcoded in the view modules instead of going through `t(...)`, so switching to Chinese will not fully localize those tabs.

## Evidence

- **Agents tab still hardcodes English UI copy:** `renderAgents` includes literal English strings such as `Copy ID`, `Set Default`, `Select an agent`, and `Pick an agent to inspect its workspace and tools.` instead of localized keys.
  - file: [ui/src/ui/views/agents.ts:177](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/ui/src/ui/views/agents.ts#L177)

  - file: [ui/src/ui/views/skills.ts:144](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/ui/src/ui/views/skills.ts#L144)

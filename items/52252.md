## Summary

Keep open. Current `main` does not implement the requested Volta path stabilization, and the PR still appears directionally useful. `resolveStableNodePath` only handles Homebrew Cellar paths today, while daemon install flow still depends on it for the active Node executable.

## Evidence

- **Current main only rewrites Homebrew Cellar paths:** `resolveStableNodePath` matches only `.../Cellar/.../bin/node` and returns all other paths unchanged; there is no Volta handling on current `main`.
  - file: [src/infra/stable-node-path.ts:11](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/src/infra/stable-node-path.ts#L11)

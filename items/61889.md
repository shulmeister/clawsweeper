## Summary

Keep open. Current main has related iMessage self-chat fixes, but the exact reported condition still drops `is_from_me=true` messages when `isSelfChat` is false. The implementation only performs the agent echo check inside the `isSelfChat` branch; the non-self-chat branch still returns `drop: from me`. Ambiguous self-thread rows are remembered so a later reflected inbound copy can be dropped, but the original `is_from_me=true` message is still not processed under the issue's failure mode.

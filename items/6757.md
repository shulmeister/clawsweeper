## Summary

Current main does not implement the requested agent-triggered self-compaction path. Manual `/compact` and the gateway `sessions.compact` RPC exist, but `session_status` has no `action: "compact"` input and the agent-facing `gateway` tool cannot call arbitrary session RPCs. The directly related PR #43401 proposed the exact `session_status(action="compact")` behavior, but it was closed unmerged in the provided GitHub context.

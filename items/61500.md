## Summary

Keep open. Current main does not implement `system.read` / `system.write`, and the standard read/write/edit tools still use gateway-local or sandbox filesystem paths rather than routing through a node host when `tools.exec.host=node`. This is core protocol/tool-routing work, not a ClawHub-only cleanup item.

Current main still treats any non-archived, non-checkpoint `.jsonl` file as a primary transcript, so `*.trajectory.jsonl` sidecars can still flow into `openclaw doctor` orphan detection and related usage/disk-budget paths. This PR is the active implementation candidate for open issue #70680; the related #72047 was closed unmerged.

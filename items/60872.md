## Summary

Keep open. Current main has ACP reply-dispatch plumbing and Telegram visibility tests that suggest this area has improved, and bundled acpx is now 0.6.0, but there is not enough high-confidence evidence that the exact Telegram + persistent ACP/acpx regression from #60872 is fixed. The generic task path can still emit a bare `Background task done` when no terminal summary is available, and the queue-owner repair implementation now lives in the external `acpx` dependency rather than inspectable repo code.

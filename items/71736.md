## Summary

Keep open. Current main does not implement the requested Control UI plugin contribution slots or the proposed `api.registerControlUiContribution(...)` surface. The related RFC PR is still open/docs-only, and the earlier Plan Mode PR cited as source behavior was closed unmerged, so meaningful requested core/plugin API work remains.

## Summary

Issue #53002 is already fixed on current `main`: the loader now normalizes alias maps so specific `openclaw/plugin-sdk/*` entries win over the bare `openclaw/plugin-sdk` alias, the CJS root alias deliberately appends the bare root alias last for prefix-match resolvers, and regression tests cover both alias ordering and `plugin-entry` subpath loading. The changelog records the shipped fix in 2026.3.28.

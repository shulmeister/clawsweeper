## Summary

Keep open. Current `main` has adjacent fixes for fragmented tool-call arguments and custom Anthropic-proxy payload normalization, but the evidence does not prove the exact reported `copilot-api` + `anthropic-messages` long-conversation regression is fixed end-to-end.

## Evidence

- **Malformed tool-call repair exists, but not for generic Anthropic proxy routes:** The embedded runner now repairs malformed streamed tool-call arguments for all `openai-completions` models and for normalized Kimi-on-Anthropic routes only. The reported setup was a generic `anthropic-messages` proxy (`copilot-api`), so this code does not directly cover the exact repro path.

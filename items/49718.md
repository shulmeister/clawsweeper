## Summary

Current `main` already implements the reported WebUI `/model` fix: the slash-command path sends the raw user input to `sessions.patch`, while Control UI model normalization preserves ambiguous raw aliases instead of forcing the current provider prefix. The same implementation is already present in release `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).

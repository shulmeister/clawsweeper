## Summary

Current `main` does not include the requested wiki plugin-loading policy or the accompanying regression coverage. The PR still appears to address a real gap: `openclaw wiki` inherits the default `loadPlugins: "never"`, startup code respects that default in both text and JSON modes, and bootstrap skips plugin registry loading when that flag is false. This is not a conservative auto-close candidate.

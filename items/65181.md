## Summary

Current `main` already persists isolated cron `sessionFile` metadata before cron session entries are written, and it has broader current coverage around isolated cron session identity and safe session-file resolution. PR #65181 is redundant against `main`; I did not find a released tag that contains this exact mainline implementation yet.

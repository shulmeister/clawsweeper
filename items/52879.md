## Summary

Current main preserves Signal group ID case in the actual delivery target and reply routing path. Session keys still lowercase group peer IDs, but replies are routed via preserved-case `To`/`OriginatingTo` and persisted `deliveryContext`, so the reported failure mode is already addressed on `main`.

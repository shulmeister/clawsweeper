## Summary

Keep open. Current `main` still has the propagation gap this PR targets: the three embedded-run call sites in `src/agents/command/attempt-execution.ts`, `src/commands/models/list.probe.ts`, and `src/hooks/llm-slug-generator.ts` call `runEmbeddedPiAgent(...)` without `allowGatewaySubagentBinding: true`, while plugin runtime late-binding still requires that explicit opt-in. The PR also still has an unresolved review concern on restoring `OPENAI_API_KEY` correctly in the new probe test.

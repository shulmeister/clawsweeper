## Summary

Keep open. Current `main` still bypasses daily/idle freshness checks for heartbeat/cron/exec system events, rewrites `updatedAt` during session init, and has explicit tests asserting that stale sessions must not reset on heartbeat. The reporter’s follow-up weakens the claimed 4.14 regression, but it does not remove the current-main behavior mismatch with documented automatic session expiry.

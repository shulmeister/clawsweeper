## Summary

Keep open. Current `main` still writes `.clobbered.*` snapshots with no per-path cap in both config-recovery paths, and the PR appears to address a real production failure mode. The author also addressed the initial warning-spam review, but a later review still raises an unresolved race on the count-then-write cap, so there is not strong evidence to close this PR as obsolete or superseded.

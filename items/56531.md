## Summary

Current `main` still rewrites `xai/grok-4-1-fast-reasoning` to `xai/grok-4-1-fast`, and the status output path is wired to surface that rewrite. The linked fix PR was closed, but the requested behavior is not implemented on `main` or in the latest release evidence available here.

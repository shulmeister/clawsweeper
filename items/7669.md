## Summary

Keep #7669 open. Current main still does not implement an npm `dev` dist-tag publish path, and the live npm registry still has no `openclaw@dev` tag.

## Best Possible Solution

Keep the issue open until release maintainers either implement a supported main-snapshot `openclaw@dev` publish path with workflow input support, release-plan validation, and postpublish verification, or intentionally retire npm `dev` and update the release-channel, update, and release docs to make dev git-only.

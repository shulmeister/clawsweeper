## Summary

Keep open. Current `main` still preserves `node dist/index.js` as a legacy direct-entry path, while the newer startup optimizations and fast paths live in `openclaw.mjs` / `src/entry.ts`. The Docker docs on `main` and in the latest release still tell users to run `node dist/index.js` inside the Docker image for several container-local tasks, so the reported slow path remains part of the documented surface and there is not strong evidence that #65031 is implemented or obsolete.

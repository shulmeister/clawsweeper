## Summary

Keep open. Current `main` does not implement `tool_result_before_model`: the public hook catalog, hook-name union, prompt-injection hook list, hook runner, and session guard wiring still only expose `tool_result_persist` for transcript writes. The PR is a substantive, still-relevant feature addition with active review iteration and related external demand, so none of the conservative close reasons apply.

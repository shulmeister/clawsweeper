## Summary

Current main, and shipped release v2026.4.23, already suppress commentary-phase assistant output in the shared channel-delivery path, prefer `final_answer` text for visible replies, and drop commentary-only assistant content from visible history. Discord with `streaming: off` uses that shared final-delivery path.

## Evidence

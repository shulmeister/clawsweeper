## Summary

Current `main` already handles the reported heartbeat session-key accumulation and overlap/backpressure paths: isolated heartbeat keys are canonicalized back to a single `:heartbeat` sibling, stale accumulated keys are removed, and the runner skips when the main queue or resolved session lane is busy. The changelog shows the session-key cleanup shipped in `v2026.4.10`.

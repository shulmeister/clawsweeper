## Summary

Keep open. Current main has a related CLI perf fix from #71739/#71746, but this gateway cold-start issue still has meaningful unresolved scope: Bedrock discovery still uses a top-level AWS SDK import, bundled plugins remain broadly default-enabled, there is no config-level disable-bundled/explicit-only switch, and discovery/manifest caches are still short in-process TTL caches rather than a cross-process persisted snapshot.

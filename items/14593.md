## Summary

Keep open. Current `main` still leaves `openai-whisper` as a brew-only skill, the onboarding flow still just recommends Homebrew and shows the install command, and the installer still hard-fails `kind: brew` installs with `brew not installed` on Linux. The Docker image/docs expose extra apt packages and other optional build features, but no supported Homebrew/bootstrap path for the official Docker flow.

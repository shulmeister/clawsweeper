## Summary

PR #60478 adds a large optional learning-loop/memory plugin. Current project scope says optional plugins should usually live outside core and be promoted/discovered through ClawHub, with a high bar for bundling optional plugins. This should be continued as a ClawHub/community plugin rather than merged into OpenClaw core.

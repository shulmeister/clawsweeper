## Summary

Current main implements the underlying bare exec allowlist pattern fix. Bare command-name patterns now match PATH-resolved executable basenames, with regression coverage and docs/changelog updates in v2026.4.24. The exact `python3 *` shape from the original report is now clarified as outside manual argument matching; `python3` is the supported bare command-name allowlist form.

## Evidence

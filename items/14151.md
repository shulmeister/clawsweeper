## Summary

Keep open. Current main does not implement a user-configurable force-prepend <think> option, and the related PR #14323 was closed unmerged. Existing code only strips/parses already-tagged reasoning or enforces <final> for tagged-reasoning providers; it does not inject an opening <think> tag before streaming/non-streaming output.

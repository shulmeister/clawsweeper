## Summary

Keep PR #72251 open. Current main still lacks the PR's volatile backup filter, tar EOF retry loop, enriched EOF failure, logger notices, and skippedVolatileCount result/summary field. The PR is a focused fix for the concrete live-backup failure in #72249, and the author already addressed the bot review findings in follow-up commits; adjacent issues #67417 and #67990 do not supersede this work.

## Best Possible Solution

Keep PR #72251 open for maintainer review. The best path is a rebased, narrowly scoped backup-archiver fix on main that adds a volatile-file filter, tar EOF retry with partial temp archive cleanup, enriched final errors, skippedVolatileCount stdout/JSON reporting, and focused regression tests, while leaving #67417 and #67990 separately tracked unless maintainers choose to consolidate backup consistency design.

## Summary

Keep #25295 open. Current main has Mattermost post update/delete helpers for preview streaming and cleanup, but the Mattermost shared message action surface still only advertises and handles `send` and `react`; `edit` and `delete` cannot reach the plugin through the shared dispatcher. The related block-streaming PRs are closed and unmerged, and the existing maintainer/Codex discussion already identified this same remaining gap.

## Best Possible Solution

Keep #25295 open, or replace it with a focused Mattermost plugin PR that wires `edit` and `delete` through `describeMattermostMessageTool()`, `supportsAction()`, and `handleAction()` in `extensions/mattermost/src/channel.ts`, reuses the existing Mattermost post update/delete helpers where appropriate, validates edit text, and adds targeted tests/docs while dropping unrelated branch cleanup.

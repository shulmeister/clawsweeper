## Summary

Keep #25295 open. Current main has Mattermost post update/delete helpers for preview streaming and cleanup, but it still does not expose Mattermost `message action=edit` or `message action=delete` through the shared message tool. The related block-streaming PRs in the provided context are closed and unmerged, and the latest maintainer/Codex discussion already identified this same remaining gap.

## Best Possible Solution

Keep #25295 open, or replace it with a focused Mattermost plugin PR that wires `edit` and `delete` through `describeMattermostMessageTool()`, `supportsAction()`, and `handleAction()` in `extensions/mattermost/src/channel.ts`, reuses or adapts the existing Mattermost post update/delete helpers in `extensions/mattermost/src/mattermost/client.ts`, adds targeted action tests and docs, validates required edit content, and removes unrelated provider/loader/test cleanup from the branch.

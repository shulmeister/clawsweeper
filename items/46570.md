## Summary

Keep open. Current main does index memory files and applies source filters to both keyword and vector search, but the reported user-visible failure is not proven fixed. In hybrid mode, keyword-only memory-file hits can still be dropped whenever any strict-scoring results exist, which can leave session results visible while exact memory-file matches fall below the default minScore/text-weight floor. The closest hybrid memory-file regression tests are currently skipped, so this is not a high-confidence cleanup close.

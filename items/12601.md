## Summary

Keep #12601 open. Current main does not implement Slack Canvas content reading, and the related PR #18752 was closed without merge. The requested capability remains meaningful for the bundled Slack plugin: there is no `read-canvas` action, no `canvases.sections.lookup` call, no `canvases:read` scope in docs, and no `channels.slack.actions.canvas` config gate on main.

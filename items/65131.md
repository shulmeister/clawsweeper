## Summary

Keep open. Current main does not provide a Control UI setting to hide system/internal chat turns, and the visible-history filtering only removes assistant commentary/silent-control replies. Internal heartbeat/memory-flush turns can still survive into transcript history, while system-role messages are rendered normally when present.

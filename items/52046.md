## Summary

Keep open. Current `main` does not implement non-blocking/background heartbeat execution; it still treats heartbeat as a main-session turn on shared queues and only mitigates contention by skipping/retrying when work is already in flight.

## Evidence

- **Docs say main heartbeats share the default main lane:** The queue docs state the default `main` lane is process-wide for inbound work and main heartbeats, while separate background lanes are for cron/subagent-style detached work.
  - file: [docs/concepts/queue.md:81](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/docs/concepts/queue.md#L81)
  - command: `nl -ba docs/concepts/queue.md | sed -n '72,92p'`
  - sha: [60f7a59f5ed5](https://github.com/openclaw/openclaw/commit/60f7a59f5ed581839cfd1d52583449c5aa1f03a4)
- **Heartbeat docs describe main-session execution, not detached/background execution:** The heartbeat guide says heartbeats run in the agent's main session by default and are skipped/retried when the main queue is busy.
  - file: [docs/gateway/heartbeat.md:253](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/docs/gateway/heartbeat.md#L253)
  - command: `nl -ba docs/gateway/heartbeat.md | sed -n '250,272p'`
  - sha: [60f7a59f5ed5](https://github.com/openclaw/openclaw/commit/60f7a59f5ed581839cfd1d52583449c5aa1f03a4)
- **Runner only checks for busy queues before starting:** `runHeartbeatOnce` returns `requests-in-flight` if `CommandLane.Main` or the resolved session lane already has queued work, which is a skip/retry guard rather than non-blocking execution.
  - file: [src/infra/heartbeat-runner.ts:759](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/src/infra/heartbeat-runner.ts#L759)
  - command: `nl -ba src/infra/heartbeat-runner.ts | sed -n '754,810p'`

- **Heartbeat turn still awaits the normal reply pipeline:** Once started, the heartbeat runner awaits `getReplyFromConfig(...)` directly, showing the heartbeat turn itself still runs synchronously through the standard agent path.
  - file: [src/infra/heartbeat-runner.ts:1080](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/src/infra/heartbeat-runner.ts#L1080)

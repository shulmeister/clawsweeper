## Summary

Keep open. Current main has some adjacent runtime pieces, including internal agent events and a trusted native plugin runtime event subscription, but it does not implement the requested RFC surface as described: no direct typed plugin `api.onAgentEvent(...)`, no plugin-owned run-scoped context storage lifecycle, no session-owned scheduler helper/cleanup contract, and no dedicated heartbeat prompt contribution or heartbeat-specific `agent_turn_prepare` hook. The linked RFC PR #71731 is still open and explicitly scoped as documentation only, while the source-behavior PR #71676 was closed unmerged.

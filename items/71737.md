## Summary

Keep open. Issue #71737 is a concrete RFC for missing core/plugin-SDK seams, and current main only has adjacent lower-level surfaces: raw runtime agent-event subscription, internal run context, generic prompt hooks, and gateway `getCron`. The requested typed/sanitized agent-event API, plugin-owned run-context storage, session-owned scheduler cleanup, and heartbeat-specific contribution hook are not implemented on main. This also should not be closed as `clawhub` because the item is asking for plugin API needed before external/plugin implementation can reach parity.

## Summary

Keep open. Current `main` still renders the Slack manifest inside the setup intro note, so the underlying copy/paste problem appears real, but this PR's approach would introduce at least two behavior regressions: it moves output into `prepare`, which runs even when the env-var shortcut suppressed the intro note, and it prints with `console.log`, which bypasses session-based wizard clients that only receive `WizardPrompter` steps.

## Evidence

## Summary

Keep open. Current main still has the exec approvals transport/schema mismatch: gateway and node-host get paths only redact socket tokens, while the public set schema rejects allowlist `source`, and `source` is still required for durable allow-always behavior. The PR is linked to the still-open #69698 and is not superseded by an implementation on main.

## Summary

Keep open. The Anthropic SDK-specific path described in the issue is partly obsolete on current main because default Anthropic runs now route through OpenClaw's boundary-aware transport, but the user-visible precision bug still appears present: Anthropic `input_json_delta` tool arguments are parsed with a JSON.parse-backed dependency parser, so bare unsafe integer IDs can still be rounded before tool execution.

## Summary

Keep open. Current main does not implement the requested async tool-result mutation hook before transcript write. The public hook contract still exposes only async/void `after_tool_call` and synchronous `tool_result_persist`, and the related PR #64907 is open and explicitly scoped to a narrower synchronous text-only hook, not the async transform requested here.

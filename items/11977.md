## Summary

Keep open. Current main still publishes Docker release images for amd64 and arm64 only; there is no `build-riscv64` job, no riscv64 digest in the manifest creation, no `Dockerfile.riscv64`, and no current Docker install docs for riscv64 images. The related PR #12033 is useful prior work, including a proposed workflow/Dockerfile/docs path, but it was closed unmerged and does not supersede the issue.

## Best Possible Solution

Keep #11977 open until a maintainer lands a Docker release workflow change that adds a non-blocking `linux/riscv64` build, includes its digest in the published manifest when available, and updates Docker docs/tests for the supported riscv64 image behavior. #12033 can serve as prior art, but it needs rebasing/review or a fresh implementation because it closed unmerged.

## Summary

Keep open. Current main still has only amd64 and arm64 Docker release jobs and manifests; there is no riscv64 build job, digest output, manifest update path, or Docker install documentation for riscv64 images. The related PR #12033 is useful prior work, but it was closed unmerged and does not supersede this issue. The existing maintainer/Codex comment on #11977 also explicitly keeps it open for the same reason.

## Best Possible Solution

Keep #11977 open until a maintainer lands a Docker release workflow change that adds a non-blocking linux/riscv64 build, includes the riscv64 digest in the published manifest when available, and updates Docker docs/tests for the supported riscv64 image behavior. PR #12033 can be used as prior art, but it needs rebasing/review or a fresh implementation because it closed unmerged.

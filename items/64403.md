## Summary

Keep open. Current main does not implement the PR's requested lazy-primary CLI metadata preference: selected plugin primaries still load the runtime plugin registry and are eagerly registered. The PR discussion contains useful bug context and several addressed review findings, but there is no strong evidence that main already contains an equivalent fix or that the PR is superseded.

## Evidence

  - file: [src/plugins/cli.ts:49](https://github.com/openclaw/openclaw/blob/0bcb4c95c185909b12d463b04d49b16ead3459d5/src/plugins/cli.ts#L49)

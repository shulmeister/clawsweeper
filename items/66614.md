## Summary

Keep #66614 open. Current main still has only fence-aware block chunking, no table-span guard, and no chunker tests for markdown table protection. The related fix PR #66568 was closed unmerged, while #66344 remains open and appears to be the active implementation path rather than shipped behavior.

## Best Possible Solution

Keep #66614 open as the bug tracker while completing #66344 or a smaller replacement. The fix should add table-span detection in the markdown/chunker layer, thread table safety through EmbeddedBlockChunker break selection and flushOnParagraph, and add regression tests for text + markdown table + text, partial streamed tables, tables inside fences, and oversized-table fallback behavior.

## Summary

Keep open. Current `main` at `8866544ffefb42ca234c66c337134fb60743d8d0` still has loop detection disabled by default, still documents/tests that disabled default, and still lacks the graceful read-offset fallback from this PR. The PR is also not an obvious conservative close because enabling loop detection by default would currently accumulate history at session scope rather than per run.

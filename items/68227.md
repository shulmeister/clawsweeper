## Summary

Keep open. Current `main` still has the `hello-ok` contract mismatch this PR fixes: the gateway handshake always emits `auth`, but the protocol schema, docs, generated Swift models, and test fixtures still treat `auth` as optional or omit it entirely.

## Evidence

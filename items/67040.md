## Summary

#67040 is now a broad umbrella. Current main has shipped/landed narrow CLI plugin-load mitigations such as #71746 for `agents list --json`, while the remaining disk-backed/cross-process plugin discovery cache problem is tracked more concretely by open issue #71789. Closing #67040 avoids splitting the same remaining Windows startup-cache work across parallel issues.

## Best Possible Solution

Close #67040 as a superseded umbrella. Keep #71789 as the focused tracker for the remaining Windows cross-process plugin discovery cache work, and rely on the already-merged #71746/#71739 path for the `agents list --json` plugin-load mitigation.

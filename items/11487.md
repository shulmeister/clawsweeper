## Summary

Keep open. Current main still lacks optional human-readable name/label metadata on group-entry schemas, including Telegram, and the generated config metadata still marks Telegram group entries as strict objects without name/label. This is a clear, still-actionable enhancement rather than an obsolete or ClawHub-only item.

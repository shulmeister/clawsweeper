## Summary

Keep open. Current `main` still resolves `--version` commit suffix from live git metadata before baked build metadata, so the reported HEAD-drift bug remains plausible and is explicitly codified in tests. The linked follow-up PR is still only discussion evidence, not proof of a shipped fix on `main`.

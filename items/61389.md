## Summary

Keep open. Current main only partially addresses the stale-threshold concern through cross-field validation, but the requested schema constraints are not implemented: both fields still use min(1), and channelMaxRestartsPerHour still cannot be set to 0. The linked PR #61392 remains open/unmerged and tracks the same requested code change, so the underlying item is not resolved on main.

## Summary

Keep open. Current main still merges and injects hook-provided prompt text without a prompt-context length cap, so this PR is not obsolete or implemented on main. The PR discussion also shows the author addressed the earlier review comments in follow-up commits, making this still actionable maintainer work rather than a conservative close candidate.

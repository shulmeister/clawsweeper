## Summary

Keep #15634 open. Current main has shipped substantial Mattermost parity improvements, including markdown preservation, reactions, threading, typing indicators, media delivery, and #19957 interactive buttons. The umbrella still includes read receipts / presence, and current Mattermost source/docs do not implement or document that remaining scope.

## Best Possible Solution

Keep #15634 open as the Mattermost parity umbrella for now, or split the unresolved read receipts / presence work into a narrower canonical Mattermost issue and close this umbrella only after that remaining scope is implemented, documented, or explicitly descoped by a maintainer. Keep #19957 as canonical evidence for the completed rich-interactions/buttons portion.

## Summary

Keep open. Current main shows active Ollama native `/api/chat` support, but it still forwards raw tool schemas on the native Ollama path and I did not find an Ollama-specific Full-profile regression test or a landed fix that clearly matches this issue's immediate `400 /api/chat` failure with GLM/Qwen when tools are enabled.

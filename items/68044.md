## Summary

Current `main` no longer silently drops pasted images on this path. Even when the gateway classifies the session model as text-only, it now offloads attachments to `media://inbound/...` refs instead of discarding them, preserves transcript ordering, and the Anthropic provider/runtime can resolve `anthropic/claude-opus-4-7` from the 4.6 template family with image input preserved.

## Evidence

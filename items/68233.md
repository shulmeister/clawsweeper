## Summary

Current `main` already surfaces bootstrap-file truncation instead of leaving it silent: it builds explicit truncation warnings, injects a bootstrap warning block into the prompt, exposes truncation in `/context`, and documents the behavior/config. The implementation landed after the reporter's 2026.4.2 build and is present on `main`, but I could not tie it to a shipped release tag yet.

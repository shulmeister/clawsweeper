## Summary

Keep open. Current `main` has partial mitigations for prompt bloat, but it does not implement the issue’s core request for automatic per-turn conditional tool injection. Tools are still built as the full runtime set and only narrowed when an explicit `toolsAllow` allowlist is passed; workspace bootstrap injection also still defaults to `always`. Skills are already compacted/budgeted, so that part of the report is partly addressed, but the main perf request remains unresolved.

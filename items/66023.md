## Summary

Keep this PR open. Current main still lacks custom cron job IDs end-to-end, and this PR is a concrete implementation candidate for the open feature request #65636, but it still needs review cleanup before landing.

## Best Possible Solution

Keep this PR open as an implementation candidate for #65636.

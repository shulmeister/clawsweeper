## Summary

Current main already contains the two fixes this report needs: isolated cron runs persist under their per-run session key/transcript instead of reusing a stale file, and Control UI history reloads now hide the synthetic transcript-repair tool result that previously broke post-stream rendering. The changelog says both shipped in 2026.4.12.

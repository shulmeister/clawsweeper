## Summary

This PR adds a new bundled skill, but current main policy and docs route new standalone skills through ClawHub, workspace skills, or plugin-shipped skills instead of expanding the core bundled skill set. Because adaptive-memory is self-contained and does not require a new core extension seam, it is better handled as an external skill distribution item than an OpenClaw core PR.

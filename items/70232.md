## Summary

Current main implements the requested non-fatal handling for the Bonjour/CIAO cancellation crash. The bundled Bonjour plugin now wires its CIAO unhandled-rejection classifier into the shared process-level rejection registry, the global rejection handler returns before fatal exit when a registered handler handles the reason, and tests cover `CIAO PROBING CANCELLED` as handled. The same-host mDNS name customization discussed in #54467 remains separate, but this issue's crash signature is addressed on main.

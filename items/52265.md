## Summary

Current `main` already covers this report: CLI RPC calls now default to the full CLI operator scope set including `operator.read`, `cron.list`/`cron.status` are explicitly read-scoped methods, and `gateway status` treats missing read-scope detail probes as degraded reachability instead of a hard connection failure. The reporter also confirmed on 2026-03-24 that upgrading to `2026.3.23-1` made the issue stop reproducing.

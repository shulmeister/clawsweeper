## Summary

Keep open. Current `main` still routes Feishu file attachments into inbound media processing, and the shared media pipeline still extracts document text/PDF content into the prompt body sent to the agent. There is also no `ignoreFileContent` field in the Feishu config schema or docs, so the reported setting is not implemented on current `main` or in the latest release `v2026.4.22`.

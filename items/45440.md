## Summary

Current main contains shipped fixes for the 2026.3.12 low-memory startup/OOM regression family: the 2026.3.13 changelog explicitly calls out the plugin-sdk shared-build fix for the recent plugin-sdk memory blow-up, current bundling builds core/plugin-sdk/bundled plugin entries in one graph to avoid duplicated runtime chunks, and current guardrails preserve a pack-size budget tied directly to the 2026.3.12 low-memory OOM reports. Related comments on #45440 already connected this report to #45064/#45065 and the 2026.3.12 bundle/chunk pressure pattern.

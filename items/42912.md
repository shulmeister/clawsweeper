## Summary

Current `main` already implements the requested behavior: mutating tool failures are retained only until the same action succeeds, after which the stale error is cleared and the misleading warning no longer survives the retry. Regression tests cover both the retry-clear path and suppression of fallback tool-error payloads when a successful answer exists.

## Summary

Keep open. Current main supports operator-configured Docker resource limits globally and per agent, but it does not provide default memory, CPU, or PID limits when those fields are absent. The requested core sandbox default policy remains missing, so this is not a conservative close candidate.

## Evidence

- **No default resource fallback:** `resolveSandboxDockerConfig` falls back to built-in defaults for image, prefix, workdir, rootfs, tmpfs, network, and capabilities, but `pidsLimit`, `memory`, `memorySwap`, and `cpus` only resolve from agent or global config with no built-in fallback.
  - file: [src/agents/sandbox/config.ts:101](https://github.com/openclaw/openclaw/blob/02f3e9cfa25f26826e8dfd79d4bf174f2f9d8f54/src/agents/sandbox/config.ts#L101)
  - command: `nl -ba src/agents/sandbox/config.ts | sed -n '1,300p'`
  - sha: [02f3e9cfa25f](https://github.com/openclaw/openclaw/commit/02f3e9cfa25f26826e8dfd79d4bf174f2f9d8f54)
- **Docker resource flags are conditional:** `buildSandboxCreateArgs` only emits `--pids-limit`, `--memory`, `--memory-swap`, and `--cpus` when the resolved config contains positive/non-empty values, so missing defaults produce unbounded Docker create args for these resources.
  - file: [src/agents/sandbox/docker.ts:433](https://github.com/openclaw/openclaw/blob/02f3e9cfa25f26826e8dfd79d4bf174f2f9d8f54/src/agents/sandbox/docker.ts#L433)
  - command: `nl -ba src/agents/sandbox/docker.ts | sed -n '330,470p'`
  - sha: [02f3e9cfa25f](https://github.com/openclaw/openclaw/commit/02f3e9cfa25f26826e8dfd79d4bf174f2f9d8f54)
- **Config schema treats limits as optional:** The Zod schema accepts `pidsLimit`, `memory`, `memorySwap`, and `cpus` as optional fields and does not assign schema defaults.
  - file: [src/config/zod-schema.agent-runtime.ts:117](https://github.com/openclaw/openclaw/blob/02f3e9cfa25f26826e8dfd79d4bf174f2f9d8f54/src/config/zod-schema.agent-runtime.ts#L117)
  - command: `nl -ba src/config/zod-schema.agent-runtime.ts | sed -n '90,135p'`
  - sha: [02f3e9cfa25f](https://github.com/openclaw/openclaw/commit/02f3e9cfa25f26826e8dfd79d4bf174f2f9d8f54)

  - file: [src/agents/sandbox-create-args.test.ts:41](https://github.com/openclaw/openclaw/blob/02f3e9cfa25f26826e8dfd79d4bf174f2f9d8f54/src/agents/sandbox-create-args.test.ts#L41)

## Summary

Current `main` has the missing runtime-dependency plumbing for bundled plugins, and `diagnostics-otel` now declares its OpenTelemetry runtime packages. The original npm-global failure mode is explicitly addressed on `main` by staging bundled-plugin dependency installs outside the plugin root before activation, with regression coverage. The latest release `v2026.4.22` predates those fixes, so this appears fixed on `main` but not yet in a tagged release.

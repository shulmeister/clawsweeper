## Summary

Keep #45501 open. Current main still hardcodes the bare `/new`/`/reset` startup prompt, tests still assert the exact disputed wording, and the config/types/schema surface does not expose `session.resetPrompt` globally or per agent. The linked #48874 is a broader multi-session RFC, not a canonical replacement for this narrow config request.

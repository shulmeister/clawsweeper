## Summary

Keep open. Current checked-out main still uses the reported OpenShell mirror-mode per-file upload path: the write bridge syncs a host file to the backend, and `syncLocalPathToRemote` uploads the full local file path directly into `dirname(remotePath)` rather than staging a flat basename like the full workspace sync path does.

## Summary

Keep open. Current `main` still has the unguarded `trim()` calls in `src/auto-reply/reply/acp-projector.ts`, while current ACP translator code and tests still produce `usage_update` session updates without a `text` field. The PR remains relevant, and the author’s follow-up commit addresses the bot’s downstream `emitSystemStatus` concern.

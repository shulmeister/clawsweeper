## Summary

Keep open. Current main still does not run the existing agentic memory flush before session teardown on `/new`, `/reset`, daily reset, or idle reset. Reset paths archive/rotate sessions and emit hooks, and startup now reads slugged reset-memory artifacts, but the requested silent tool-enabled memory-flush turn remains wired only into the normal reply/compaction pipeline.

## Summary

Keep open. Current main does not expose the requested structured token-usage object through the `session_status` agent tool. It provides a human-readable status card with latest token/cost text and there is a separate Gateway `sessions.usage` API, but that does not satisfy the requested agent-facing `session_status` raw usage fields.

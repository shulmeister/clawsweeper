## Summary

Keep open. Current main has global agent runtime timeout and global LLM idle timeout, but the requested per-model generation timeout config remains missing. The existing idle watchdog helps no-chunk hangs, but it does not cap a model that keeps streaming looping planning text, and the linked first-token/provider-skip PR was closed unmerged.

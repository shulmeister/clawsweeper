## Summary

Keep open. Current main provides a chat-header session dropdown and a separate Sessions table that links into chat, but it does not show all active channel conversations in the left Control UI sidebar. The related #46835 appears to cover the same missing behavior rather than superseding this earlier issue.

## Evidence

## Summary

Keep open. Current `main` does not implement cron `failureAlert.mode = "agent-turn"`: the shipped types, CLI validation, and docs still only allow `announce` or `webhook`, and the Gateway runtime wires failure alerts to outbound delivery rather than waking the main session. There is also an active unmerged PR (#56545) carrying this feature.

## Evidence

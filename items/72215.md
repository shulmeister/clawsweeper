## Summary

Keep #72215 open. Current main still does not allow `cron.enabled` or `agents.defaults.compaction.mode` through the agent-facing gateway config mutation guard, and the PR is a focused implementation for the still-open #72166 request rather than obsolete duplicate work.

## Best Possible Solution

Keep the PR open for maintainer review. If accepted, merge this PR or an equivalent scoped patch that adds `cron.enabled` and `agents.defaults.compaction.mode` to `ALLOWED_GATEWAY_CONFIG_PATHS` with positive guard coverage, then close/link #72166 as fixed. If maintainers consider either path too sensitive for agent-driven config edits, document that boundary explicitly and close the request with that product/security rationale.

## Summary

Keep open. Current main still does not implement Telegram allowBots: the Telegram config schema/types do not accept the field and the Telegram message context/processor paths do not pass or inspect it. The referenced PR #8310 was closed unmerged; the linked Plaiground ClawHub skill is a useful Discord workaround but does not resolve Telegram group bot-to-bot support.

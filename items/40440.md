## Summary

Keep open. Current main still records mention-gated Telegram group history as text-only entries, and the later inbound context only merges current-turn and reply media into `MediaPaths`. The related fix PRs #40470 and #40490 were closed unmerged, and the issue's requested behavior is still missing.

## Best Possible Solution

Keep #40440 open and implement the Telegram plugin fix: extend the reply-history/InboundHistory contract with media refs, preserve or persist media from skipped group messages beyond the media TTL, merge history media into the later `MediaPaths`/`MediaTypes` payload, and add a focused regression test for image-without-mention followed by @mention.

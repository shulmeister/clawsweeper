## Summary

Current `main` already provides native local Whisper-based inbound audio transcription through the shared `tools.media.audio` pipeline, including auto-detection of local `whisper`/`whisper-cli`, templated media-path arguments, and tests covering voice-note transcription flow.

## Evidence

- **User-facing docs: local Whisper is already part of audio auto-detection:** The audio docs say that when `tools.media.audio` is enabled or left in auto mode, OpenClaw auto-detects local CLIs including `whisper-cli` and Python `whisper`, and documents explicit Whisper CLI configuration for inbound audio transcription.

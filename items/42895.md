## Summary

Keep PR #42895 open. Current `origin/main` still lacks the Termux/PREFIX home-directory fallback this PR proposes, and the linked bug has concrete crash evidence. The PR remains a focused bug fix with relevant tests rather than obsolete or superseded work.

## Best Possible Solution

Keep the PR open and have a maintainer review/rebase/land the focused `src/infra/home-dir.ts` fix with its regression tests, or implement the same Termux-aware fallback directly on main. The core behavior belongs in OpenClaw because config, state, and workspace path resolution all depend on the shared home-directory helper.

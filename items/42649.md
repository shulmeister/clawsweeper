## Summary

Keep open. Current `main` already has hybrid memory retrieval, weighted vector/BM25 merging, optional temporal decay/MMR, and citation-based source attribution, but it does not yet expose per-result match explanations and does not make freshness/staleness handling a default retrieval behavior. The issue is only partially implemented.

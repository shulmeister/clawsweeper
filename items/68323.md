## Summary

Keep #68323 open. Current main still has provider selection, provider-specific TTS config, model directives, and gateway/CLI/provider controls, but not the requested provider-agnostic `messages.tts.persona` / `messages.tts.personas` contract or persona resolution layer. The linked maintainer triage comment explicitly keeps this as the canonical feature request, with #70748 still open as the closest implementation and #69051 adjacent but not equivalent.

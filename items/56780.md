## Summary

Keep open. Current `main` and the latest tagged release appear to have fixed the startup `dist/package.json`/`@lancedb/lancedb` load regression, but the recall half of the report still looks unresolved: auto-recall embeds the full prompt with no truncation or model-window guard, and `captureMaxChars` still only applies to capture filtering.

## Evidence

- **Auto-recall still embeds the full prompt:** The `before_prompt_build` hook still calls `embeddings.embed(event.prompt)` directly and only skips empty/very short prompts. There is no truncation, token budgeting, or model-specific context guard before the embedding request.

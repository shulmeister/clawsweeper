## Summary

Keep #13570 open. Current main does not implement `contextPruning.forcePruneRatio`: the config schema and effective settings omit the field, and the cache-TTL hook still returns before pruning whenever the TTL window has not expired. The related PR #13572 matches the request but was closed unmerged, so it does not supersede the issue.

## Evidence

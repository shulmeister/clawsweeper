## Summary

Keep open. Current main still intentionally blocks the requested workflow: sandboxed requester sessions cannot spawn ACP sessions, `sandbox="require"` is rejected for ACP, and the public docs continue to describe this as a current limitation. The related closed #32935 is prior context, not an implemented fix or open canonical tracker for the remaining work.

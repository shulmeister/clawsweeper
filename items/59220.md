## Summary

Keep open. Current `main` still treats DuckDuckGo bot challenges as a live failure path, and the runtime only falls back when the provider was auto-selected rather than explicitly configured. OpenClaw already has configurable alternate web-search providers, but there is no strong evidence that `main` fixes the reported DuckDuckGo breakage itself or automatically rescues a DDG-only setup.

## Evidence

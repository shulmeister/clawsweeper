## Summary

Current `main` already implements the core fix for #40363: explicit provider API keys from config take precedence over stored auth-profile credentials, and the embedded runner skips auth-profile rotation when that explicit config path is active. The issue is therefore implemented on `main`, but not yet in the latest tagged release (`v2026.4.22`).

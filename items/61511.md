## Summary

Keep #61511 open. Current main has an OpenTelemetry diagnostics plugin, but it does not implement native Sentry DSN config/env support or early gateway/agent process crash capture. A ClawHub close is not safe because the request is specifically about core startup/runtime crash handling, including uncaught exceptions, not an optional post-start diagnostic sink.

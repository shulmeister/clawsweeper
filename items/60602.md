## Summary

Keep open. Current `main` does not implement per-agent Bedrock `requestMetadata` injection, and the existing Bedrock provider/config surface still lacks both a `requestMetadata` setting and hook context that carries agent identity into the provider stream layer. The issue also already has a linked open PR (`#60698`), so conservative cleanup should not close it.

## Evidence

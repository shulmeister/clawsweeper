## Summary

Keep open. Current main does not show a clear post-report fix for Signal SSE on Node 25, and the report is still actionable. The Signal client still uses the same fetch-based SSE path, the generic fetch wrapper behavior already existed in the reported 2026.3.22 build, and the repo still documents Node >=22.14 rather than excluding Node 25.

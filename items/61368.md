## Summary

Keep open. Current main does not implement the requested full `agents.workspace.*` workspace file-management API or Control UI workspace browser; it still exposes only the bootstrap-oriented `agents.files.*` surface. The related PR #62417 is relevant and explicitly targets this issue, but it is open/unmerged in the provided GitHub context, so the requested capability remains missing on main.

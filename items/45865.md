## Summary

Keep open. Current main does not expose the requested injected `api.runtime.llm()` SDK method. Plugins do have `api.runtime.agent.runEmbeddedAgent(...)`, `api.runtime.subagent`, and lower-level simple-completion helpers, but those are not the same provider-agnostic runtime inference facade requested in #45865. The related PR #64294 appears to be the same direction of work, but it is not merged and the provided ClawSweeper context says it still has unresolved review concerns.

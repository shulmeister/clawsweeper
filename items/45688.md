## Summary

Keep #45688 open. Current main has some config UI improvements, but the main Config path still defaults to Quick Settings and its advanced mode uses an accordion surface without the top search/tab bar; large parts of config are also split into separate scoped tabs. That does not give strong evidence that the reported v2 regression, global findability/filtering, and distributed settings usability problem is solved.

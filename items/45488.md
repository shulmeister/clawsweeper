## Summary

Keep open. Current main still contains the reported post-compaction AGENTS.md section parsing and reinjection paths by default: one path queues a post-compaction context refresh for the next run, and the safeguard compaction path appends AGENTS.md sections into the compaction summary. A config opt-out exists, but that does not satisfy the issue's requested default behavior of no magic string parsing/reinjection.

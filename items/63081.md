## Summary

Keep open. Current main still has the exact `loadState` array-state bug in both device and node pairing, and no merged or clearly canonical superseding PR fixes both paths. The related issue #63035 is still open; the other related PRs are either open duplicates with different coverage or closed unmerged, so this PR still contains useful remaining work for maintainers to consolidate.

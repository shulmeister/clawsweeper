## Summary

Current `main` does not have a built-in `ApiKeyExpiringSoon` / "less than 30 days" alert path. OpenClaw's auth-health code treats API-key credentials as `static`, only marks OAuth/token credentials as expiring, and uses a 24-hour default warning window, so the reported behavior does not match reproducible current-main behavior.

## Evidence

- **No matching alert string on main:** Repo-wide search for `ApiKeyExpiringSoon`, `expire in less than 30 days`, and `less than 30 days` returned no matches in `src/`, `extensions/`, `docs/`, `ui/`, or `test/`.
  - command: `rg -n "ApiKeyExpiringSoon|expire in less than 30 days|less than 30 days" src extensions docs ui test`
  - sha: [c2139635ff82](https://github.com/openclaw/openclaw/commit/c2139635ff82b367b734ea052487258b1b33f060)
- **API keys are not treated as expiring credentials:** `buildProfileHealth` marks `api_key` profiles as `status: "static"`, and the default warning window is `DEFAULT_OAUTH_WARN_MS = 24 * 60 * 60 * 1000` (24h).
  - file: [src/agents/auth-health.ts:46](https://github.com/openclaw/openclaw/blob/c2139635ff82b367b734ea052487258b1b33f060/src/agents/auth-health.ts#L46)
  - sha: [c2139635ff82](https://github.com/openclaw/openclaw/commit/c2139635ff82b367b734ea052487258b1b33f060)
- **Expiring/missing monitoring excludes API-key providers:** Gateway auth-status code explicitly says API-key providers are excluded from expiry monitoring because their credentials do not expire on a schedule this endpoint can meaningfully monitor.
  - file: [src/gateway/server-methods/models-auth-status.ts:186](https://github.com/openclaw/openclaw/blob/c2139635ff82b367b734ea052487258b1b33f060/src/gateway/server-methods/models-auth-status.ts#L186)
  - sha: [c2139635ff82](https://github.com/openclaw/openclaw/commit/c2139635ff82b367b734ea052487258b1b33f060)
- **User-facing docs describe a 24h OAuth warning window:** `docs/concepts/models.md` documents that `openclaw models status` surfaces OAuth expiry status and warns within 24 hours by default.
  - file: [docs/concepts/models.md:203](https://github.com/openclaw/openclaw/blob/c2139635ff82b367b734ea052487258b1b33f060/docs/concepts/models.md#L203)
  - sha: [c2139635ff82](https://github.com/openclaw/openclaw/commit/c2139635ff82b367b734ea052487258b1b33f060)
- **Current UI wording does not match the reported alert text:** English UI strings use generic `Model auth expiring soon` / `{provider} ({when})` wording rather than an `ApiKeyExpiringSoon` alert or a hard-coded 30-day threshold.
  - file: [ui/src/i18n/locales/en.ts:282](https://github.com/openclaw/openclaw/blob/c2139635ff82b367b734ea052487258b1b33f060/ui/src/i18n/locales/en.ts#L282)
  - sha: [c2139635ff82](https://github.com/openclaw/openclaw/commit/c2139635ff82b367b734ea052487258b1b33f060)

## Summary

Keep #71396 open. Current main still contains the Feishu mention-cascade behavior this PR is intended to remove, and the PR discussion includes unresolved card-mode mention conversion and security-review concerns that need maintainer handling rather than cleanup closure.

## Best Possible Solution

Keep the PR open and have the Feishu plugin owner or maintainer finish the bundled plugin fix: stop unconditional automatic propagation of inbound `mentionTargets` to fallback, error, ack, and ordinary replies; keep mention context safe for the agent; preserve explicit mention behavior through structured allowlisted conversion rather than raw model-generated tags; and retain regression coverage for plain text, card, streaming-card, and fallback paths.

## Summary

Keep open. Current main implements the external-secrets baseline with SecretRefs, env/file/exec providers, runtime reload, CLI, docs, and tests, but it does not implement the requested native AWS Secrets Manager, Vault, Azure Key Vault, or GCP Secret Manager providers. The issue discussion also has an explicit maintainer status update saying #26155 shipped the baseline while this issue remains useful as the umbrella for remaining first-class provider work.

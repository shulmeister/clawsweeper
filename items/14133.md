## Summary

Current main still does not implement the requested Reminders tab or a cron metadata/tag/type contract that would let the UI distinguish reminders from other cron jobs. OpenClaw does support one-shot reminders through cron, and some related Cron card ergonomics exist, but the central feature remains missing and is a core Control UI/API concern rather than a clear ClawHub-only close.

## Summary

Keep open. Current main has a Control UI with `Cron Jobs` and `Skills` tabs, and cron explicitly supports one-shot reminders, but there is no dedicated reminders tab or reminder-specific cron metadata/filtering. The report's `Mission Control` wording does not match current repo naming, but the requested behavior is still understandable and not already implemented.

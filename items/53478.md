## Summary

Keep open. Current `main` does not implement these root re-exports, and the SDK contract currently documents and tests these helpers as narrow subpath exports instead of part of `openclaw/plugin-sdk` root. This is an unresolved API-surface decision, not a stale or already-fixed item.

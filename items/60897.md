## Summary

Current main intentionally defines the Control UI cache hit rate as cacheRead / (input + cacheRead + cacheWrite), documents that formula in the UI hint, and has regression coverage for cache writes lowering the displayed percentage. The 2026.4.24 changelog/appcast also records this Control UI /usage fix via related #47885.

## Summary

Keep open. Current `main` does not include this UI change, and the proposed condition from PR #68071 has a credible regression in single-result root/scoped views because the section hero is only rendered when `activeSection` is set. The existing review comment is consistent with current `main` code, so there is not strong evidence to auto-close this PR as implemented or obsolete.

## Summary

Keep #61673 open. Current main does not implement a pre-container bind resolver, and this is not a clean ClawHub/plugin close because the default Docker sandbox path still has no pre-create hook with channel/user metadata. The related items are useful context (#19420 confirms before_agent_start cannot solve pre-container enforcement; #22669 was bind ordering; #30334 was separate config injection), but none duplicates or supersedes the remaining request.

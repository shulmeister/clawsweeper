## Summary

Keep open. Current `main` still does not implement Feishu `groups.*.toolsBySender` support, so there is no strong basis to close this PR as already implemented or obsolete.

## Evidence

- **Feishu group schema still lacks `toolsBySender`:** `FeishuGroupSchema` only includes `requireMention`, `tools`, `skills`, `enabled`, `allowFrom`, `systemPrompt`, `groupSessionScope`, `topicSessionMode`, and `replyInThread`. There is no `toolsBySender` field, and the local `ToolPolicySchema` only permits `allow`/`deny`, not `alsoAllow`.
  - file: [extensions/feishu/src/config-schema.ts:24](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/feishu/src/config-schema.ts#L24)

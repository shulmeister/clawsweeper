## Summary

Keep open. The issue body is partly obsolete because current `main` has removed the old Chrome extension relay and migrated that flow to `existing-session`/Chrome MCP, but the requested capability is still not implemented on the replacement path. Current `existing-session` snapshot handling explicitly rejects `selector`/`frame` targeting, while iframe-scoped snapshots exist only on the Playwright-managed/raw CDP path.

## Summary

Keep this PR open. Current main still lacks the bundled-channel-entry-specific loader diagnostic and field-level bundled-entry validation logging, so the work is not obsolete or implemented on main. The branch needs revision before landing because the provided diff/review context shows it drops the existing debug module-shape diagnostic for ordinary missing-register plugin failures.

## Best Possible Solution

Keep the PR open and request a narrow revision: preserve formatMissingPluginRegisterError for normal missing-register plugin failures, detect bundled-channel-entry through the already-resolved record kind or a small type guard, keep the bundled-entry missing-field logging improvement, and add regression tests for both the bundled-channel diagnostic and the existing debug module-shape contract.

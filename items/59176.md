## Summary

Current main already ships native forked subagent spawning through the supported `sessions_spawn` contract (`context: "fork"`), with docs and tests present in the released `v2026.4.23` tree; this PR’s separate `fork*` parameter shape is obsolete and the branch itself was reviewed as broken against the repo layout.

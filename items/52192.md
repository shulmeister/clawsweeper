## Summary

Keep open. Current `main` at `60f7a59f5ed581839cfd1d52583449c5aa1f03a4` still has the duplicate-send path in `extensions/feishu/src/outbound.ts`: `sendMedia` sends text once before the media branch and again in the no-media fallback, so the PR is not implemented on main and remains useful.

## Summary

Keep open. Current `main` only ships a minimal single-pod Kubernetes install path and still requires node pairing approval; it does not implement Kubernetes service-account trust, headless worker auto-auth, worker autoscaling topology, or Helm artifacts. The linked PR #12434 was closed unmerged, and the maintainer explicitly said the simpler Kubernetes baseline landed separately while the SA-trust work would need to come back as a separate focused change.

## Evidence

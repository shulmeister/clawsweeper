## Summary

Keep open. Current `main` does not implement WhatsApp message deletion: the WhatsApp plugin only advertises `react` and `poll`, its channel action adapter only claims support for `react`, and the runtime throws for any other WhatsApp action. The request remains actionable in OpenClaw core/plugin scope rather than ClawHub.

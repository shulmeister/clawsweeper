## Summary

Keep open. Current `main` still routes plugin text-transform reads through `src/plugins/active-runtime-registry.ts`, which only `require()`s `./runtime.js` / `./runtime.ts`, and the current build graph still makes `plugins/runtime/index` the stable runtime entry. I do not see source-level evidence that issue #64921 was fixed, even though the discussion referenced PR #64924.

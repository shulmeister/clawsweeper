## Summary

Keep open. Current `main` still defaults new Cron forms to `last`, so the reported behavior is not implemented on `main`, and the proposed PR logic appears to resolve defaults from the full channel plugin catalog rather than configured channels only, which matches the blocking review concern and makes closure premature.

## Summary

Keep open. Current `main` still has a Feishu config/schema mismatch: `extensions/feishu/src/config-schema.ts` defines a strict schema that does not include `contextVisibility`, while Feishu runtime and tests still read/use `channels.feishu.contextVisibility`. The reporterâ€™s sample config includes that key, so the reported `channels.feishu: invalid config` remains plausible on `main`, and I found no doctor-era compat rule for Feishu/context-visibility migration.

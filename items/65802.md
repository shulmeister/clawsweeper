## Summary

Keep open. Current `main` still inherits ambient HTTP(S) proxy env for Feishu WebSocket connections, so the request is not already implemented. The submitted PR contains the targeted production change, but the added tests introduce an env-cleanup gap for `OPENCLAW_FEISHU_WS_USE_PROXY`, which matches the review comment and means the PR still needs follow-up rather than cleanup-closing.

The contributor should rebase onto current main, add provider docs/changelog/labeler metadata, fix disabled Transcribe behavior, align Nova Sonic with AWS bidirectional event schemas and OpenClaw realtime audio contracts, address the outstanding Polly/Nova/audio review comments, and return with green changed-surface CI for maintainer review.

## Summary

Keep open. Current `main` still has the cleanup-pending gap this PR addresses: a run can end while `cleanupCompletedAt` is still unset, but the active-run predicates, status formatting, and spawn-limit counting still treat that direct run as finished unless it also has pending descendants.

## Summary

Keep open. Current main does not implement the requested per-agent Discord webhook override for normal route-bound Discord replies: the agent and binding schemas are strict and have no `discord.webhook` or binding identity field, while the existing Discord webhook persona path is limited to bound thread records. Related #41459 overlaps on Discord visual identity, but it tracks a per-channel binding override rather than fully superseding this per-agent webhook request.

## Summary

Current `main` already contains a shipped fix for the reported cron-isolated-session deadlock/degraded-lane behavior. The code now forces cron-triggered embedded runner work onto the nested lane instead of reusing the cron lane, and the changelog says this shipped in `2026.3.13` specifically to stop isolated cron jobs from deadlocking under queued inner work.

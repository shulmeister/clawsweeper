## Summary

Keep open. Current main fixes the secondary hook-config schema mismatch, but it does not add the requested internal auto-reset hook event for bundled/custom internal hooks. Auto daily/idle rollovers now emit plugin lifecycle hooks (`session_end` with `idle`/`daily` reasons), while the bundled `session-memory` internal hook still only listens to manual `command:new` and `command:reset` events.

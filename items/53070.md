## Summary

Current `main` already covers both reported regressions: `commands.ownerDisplay` is present in the strict config schema again, Discord legacy streaming aliases are normalized instead of being treated as fatal removals, and message-tool discovery now checks that `describeMessageTool` is a function before calling it, with Discord still exporting that adapter.

## Summary

Keep open. Current main shipped the Ollama stream-level fix for native thinking/reasoning chunks, but it does not implement this PR's narrower incomplete-turn retry behavior for Ollama thinking-only or generic empty turns. The PR may still contain useful focused work if narrowed and rebased.

## Best Possible Solution

Keep the PR open for maintainer review, but require a narrow rebase on current main: add only Ollama-specific incomplete-turn retry support for unsigned-thinking and generic empty responses, keep Ollama out of strict-agentic execution-contract activation and planning/ack fast paths, and add targeted regression tests. If maintainers decide the shipped `v2026.4.24` stream-preservation fix fully satisfies the product need, close this PR manually with that rationale.

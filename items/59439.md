## Summary

Keep open. Current `main` still probes unmanaged restarts with shell-only gateway auth, so the core bug described by PR #59439 is not implemented on `main`. The PR discussion also surfaced unresolved problems in the proposed patch: the added regression test would not exercise the new code path without re-seeding `loadConfig`, and the proposed auth helper usage would need an extra guard for non-SecretRef failures.

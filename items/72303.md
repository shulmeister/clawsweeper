## Summary

Current main already fixes the reported Bonjour bridge-network failure: stuck probing/announcing advertiser restarts are bounded, known ciao process errors are suppressed instead of exiting the Gateway, Docker bridge setups disable Bonjour by default, and docs/tests cover the behavior.

## Best Possible Solution

Keep the shipped main-line Bonjour hardening: bounded advertiser repair, ciao process-error suppression, Docker bridge Bonjour disabled by default, and the documented opt-in path for mDNS-capable networks. No further core issue is needed for this report unless a new reproduction appears on a build containing these fixes.

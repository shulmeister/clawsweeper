## Summary

Issue #45471 should close as superseded by open PR #56768. The issue thread narrowed the original UI-chat symptom to gateway auth/token state, duplicate gateway instances, and provider throttling; the remaining repo-actionable behavior is the `gateway status` dashboard URL parity gap that #56768 explicitly tracks.

## Evidence

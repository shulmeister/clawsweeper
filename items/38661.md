## Summary

Keep open. Current main still lacks the TUI Ctrl+C active-run hint from #38501, so the PR is not fully obsolete. However, the other bundled fixes are already handled differently on main, and the web_search rename would conflict with current policy/catalog behavior, so this PR should be narrowed rather than merged as-is.

## Best Possible Solution

Keep the PR open only as evidence for the remaining #38501 behavior, but do not merge it as-is. The best maintainer path is a focused TUI-only patch in `src/tui/tui.ts` that adds the active-run hint, with a small regression test if practical, while preserving current main's web_search, Feishu, and bootstrap-cache implementations.

## Summary

Keep open. Current main still lacks the TUI Ctrl+C active-run hint requested by #38501, so the PR is not fully obsolete. The PR should not merge as-is because its other bundled changes are already handled on main or conflict with current web_search policy behavior.

## Best Possible Solution

Keep this PR open as evidence for #38501, but do not merge it as-is. The maintainer path should be a focused TUI-only patch in `src/tui/tui.ts` that adds `run still active (Esc or /abort to stop)` to Ctrl+C status while a run is active, ideally with a small regression test. Preserve current main's web_search, Feishu, bootstrap-cache, and Telegram implementations.

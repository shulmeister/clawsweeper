## Summary

Keep open. Current main still lacks the TUI Ctrl+C active-run hint requested by #38501, and #38661 contains a directly relevant patch for that remaining bug. The bundled branch should be narrowed before merge because its `web_search` rename conflicts with current main's managed/native web-search policy, while the Feishu, bootstrap-cache, and Telegram pieces are already handled on main.

## Best Possible Solution

Keep #38661 open as contributor evidence for #38501, but do not merge the bundled branch as-is. The maintainer path is a narrow TUI-only patch in `src/tui/tui.ts` that adds `run still active (Esc or /abort to stop)` when `activeChatRunId` is set, plus regression coverage for that status text. Preserve current main's `web_search` behavior and the existing Feishu, bootstrap-cache, and Telegram implementations.

## Summary

Keep open. Current main has reduced the startup prompt exposure by listing only escaped skill metadata and reading a selected SKILL.md on demand, but it does not implement the requested instruction-isolation architecture, role locking, behavioral monitoring, or skill capability permission model. The request is core/security-facing, not a ClawHub-only skill/plugin cleanup item.

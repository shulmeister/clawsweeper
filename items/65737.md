## Summary

Keep open. Current `main` still has the docs gap this PR fixes: `docs/concepts/active-memory.md` lists `config.cacheTtlMs` without the enforced bounds/default, while the runtime and plugin schema still enforce `1000..120000` ms with a `15000` ms default.

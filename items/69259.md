## Summary

PR #69259 is superseded. Current main already ships the central GitHub Copilot change: `claude-opus-4.7` is the bundled/default setup model, docs and tests match that behavior, and `claude-opus-4.6` is intentionally excluded after #69818. The remaining static-list additions are already covered by open canonical PR #69318, while this PR also carries unrelated docs/skill commits.

## Best Possible Solution

Close #69259. Keep the shipped `github-copilot/claude-opus-4.7` default and dynamic Copilot model fallback, avoid re-adding `claude-opus-4.6`, and use #69318 as the canonical place for any remaining static default-list additions maintainers still want.

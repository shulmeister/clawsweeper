## Summary

Keep PR #39806 open. Current main still has the Debug Snapshots UX gap described by #39777: disconnected refresh returns before any visible state is set, and the card body has no loading or disconnected callout. The PR is a focused attempt to fix a real remaining Control UI bug, though it still needs normal review for layout/test quality.

## Best Possible Solution

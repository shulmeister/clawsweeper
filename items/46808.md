## Summary

Keep open. Current main has strong underlying usage/cost plumbing and a Control UI `/usage` dashboard, but it does not implement the requested real-time running cost counter in the active chat header/sidebar or an explicit live per-sub-agent-run cost tracker. Related #12299 covers programmatic cumulative access; this issue is still the UI surface request.

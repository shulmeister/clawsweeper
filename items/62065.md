## Summary

MiniMax memory-search embeddings are not implemented on current main, but the requested capability is optional provider behavior that can be shipped through the existing memory embedding provider plugin seam. OpenClaw’s scope guidance says optional capabilities should usually live as plugins and be discovered/promoted through ClawHub, and the current SDK/manifest/runtime already supports plugin-owned memory embedding providers without a new core API.

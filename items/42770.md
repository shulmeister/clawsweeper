## Summary

Keep open. Current `main` still does not apply the repo baseline security headers to the canvas host or A2UI HTTP surfaces, so this PR is not implemented on `main` and the underlying request remains valid.

- **Baseline helper exists and is documented as safe for canvas/A2UI:** `setDefaultSecurityHeaders` sets `X-Content-Type-Options`, `Referrer-Policy`, and `Permissions-Policy`, and its doc comment explicitly says framing/CSP are omitted because canvas host and A2UI may be framed.

## Summary

Current `main` does not implement WhatsApp message deletion/revocation. The WhatsApp plugin only exposes `react` as a supported action, advertises only `react`/`poll` from message-tool discovery, and its action runtime rejects any non-`react` action. The issue is still actionable and specific, so it should remain open.

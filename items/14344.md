## Summary

Keep open. Current main still lacks WhatsApp message delete/revoke support: `delete` is a generic action name, but the WhatsApp plugin only advertises `react`/`poll`, only dispatches `react` as a plugin action, and other actions fall through to the unsupported-action error. The issue body and RaviTharuma’s follow-up both describe concrete real-world WhatsApp cleanup/retraction use cases that remain unimplemented.

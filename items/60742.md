## Summary

Keep open. Current main has related continuity machinery, but it does not implement the requested recent/top-of-mind layer with `memory/recent/latest.md`, snapshot files, write-on-`/new`/`/reset`/`session:compact:before`, and recovery-first reading. The linked PR #60743 appears to be the active implementation thread, but it is still an open draft and not merged into main, so closing this issue now would be premature.

## Summary

Keep open. Current main still matches the reported agent-section regression: the Agents page uses a plain native select for agent selection, does not render the available emoji/avatar helper in that selector, and the overview model picker remains a native select. The related model-switcher issue #45630 only covers the provider-routing part, not this agent-selector UI regression.

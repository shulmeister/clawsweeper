## Summary

Current main does not implement recursive/depth-aware `subagents list`. The tool schema only accepts `action`, `target`, `message`, and `recentMinutes`; the list path passes only directly controlled runs into `buildSubagentList`; and the slash command has no `--depth` or `--recursive` parsing. The requested depth-2 visibility remains an actionable core subagent control feature.

## Summary

Keep #72306 open. Current main still has the lifecycle gap this PR addresses: the pricing refresh timer is scheduled without unref, pricing fetches only use timeout signals, and gateway shutdown only clears the scheduled timer without aborting an in-flight refresh. The PR is a focused fix for the open #72208 regression and includes targeted tests for the missing behavior.

## Best Possible Solution

Keep the PR open and land this behavior, or an equivalent maintainer-owned implementation, in `src/gateway/model-pricing-cache.ts`: unref the 24-hour pricing refresh timer, connect pricing fetches to a shutdown `AbortController`, abort the active refresh from the returned stop callback, and keep the targeted regression tests in `src/gateway/model-pricing-cache.test.ts`.

## Summary

Keep #38626 open. Current main has scalar subagent/task tracking, completion announcements, task cancellation, and subagent kill/steer controls, but it still lacks the requested durable per-subagent lifecycle timeline, persisted tool/warning/error transitions, artifact-pointer completion summaries, first-failure/recommended-action diagnostics, and idempotent audit-logged cancel/steer/resume supervision controls. Closed related issue #62478 is complementary background-spawn work, not a canonical replacement for this observability/supervision scope.

## Best Possible Solution

Keep this open unless a maintainer deliberately narrows the scope. The best product path is to add a durable per-run subagent/task event ledger for spawn, queued/running, tool start/end, warning/error, completion, and supervision actions; persist it in the subagent/task stores; expose it through /subagents info, the subagents tool, openclaw tasks show, and gateway/plugin runtime views; attach artifact pointers and terminal diagnostics; and make cancel, steer, and resume idempotent with auditable action records. Treat #62478 as complementary background-spawn work, not the canonical tracker for this observability scope.

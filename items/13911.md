## Summary

Keep #13911 open. Current main still has no per-channel subagent announce suppression/reroute config, and the completion path still delivers to the requester channel when a deliverable origin is resolved. The broader #69492 audience-classification issue overlaps, but it is an open design proposal and explicitly says linked issues with unique symptoms/surfaces should stay open until verified fixed.

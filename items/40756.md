## Summary

Keep PR #40756 open. Current main still appends the HEARTBEAT.md workspace path hint to any resolved heartbeat prompt that mentions `heartbeat.md`, including user-configured prompts, while the docs say custom heartbeat prompts are sent verbatim. The linked bug #40255 remains open, and the related PRs in the provided context were closed as duplicates rather than merged fixes.

## Best Possible Solution

Keep the PR or the linked bug #40255 open. The best product path is a small heartbeat-runner fix that distinguishes user-configured heartbeat prompts from built-in/generated prompts, keeps custom prompts consistent with the documented verbatim contract, preserves the workspace-path hint for default/internal prompts where desired, and adds a focused regression test for custom prompts mentioning HEARTBEAT.md.

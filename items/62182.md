## Summary

Current main and the v2026.4.24 release document and implement the supported config path `agents.defaults.bootstrapMaxChars`. The reported root-level `bootstrapMaxCharsPerFile` key is not part of the strict config contract, and current repo docs/source contain no references to that obsolete name.

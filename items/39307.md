## Summary

Keep #39307 open. Current main has partial compaction surfacing, but the requested default, hard-to-miss user signal plus assistant-side reset disclosure/re-grounding remains unimplemented.

## Best Possible Solution

Keep this open for maintainer/product work. A complete fix should define the default compaction/reset visibility contract across supported chat surfaces, make the signal durable or hard to miss where feasible, add safe assistant-side disclosure and re-grounding after compaction, and update docs/tests around `agents.defaults.compaction.notifyUser`, post-compaction recovery semantics, and any operator knob needed to avoid leaking internals in customer-facing bots.

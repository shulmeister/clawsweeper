## Summary

Keep #39307 open. Current main has partial compaction surfacing, but it still does not meet the issue’s requested default, hard-to-miss user signal plus assistant self-disclosure and re-grounding behavior after mid-thread compaction.

## Best Possible Solution

Keep this open for maintainer/product work. A complete fix should make compaction/reset state visible by default where the runtime can surface system events, add safe assistant-side disclosure/re-grounding behavior after compaction, cover non-Control-UI chat surfaces or document intentional limits, and update compaction docs/tests to match the chosen product contract.

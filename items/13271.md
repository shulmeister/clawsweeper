## Summary

Keep #13271 open. Current main does not implement a per-channel tool-call budget, `channels.discord.toolBudget` is not part of the strict Discord channel schema, and the existing tool-call hook surface can block or rewrite calls but does not receive a first-class channel id or provide the requested auto-delegation/summary behavior. Related issue #14373 is closed but only covered global inline delegation, while #9912 remains open for global max tool calls; neither supersedes the per-channel gateway enforcement requested here.

## Summary

Keep #13271 open. Current main still has adjacent pieces, but not the requested channel-scoped per-turn tool-call budget with `warn`, `block`, and `auto-delegate` overflow behavior. The existing related issues do not supersede this specific per-channel enforcement request: #9912 remains a global maxToolCalls/maxTurns request, while #14373 overlaps auto-delegation but is closed and not a complete channel-scoped tracker.

## Best Possible Solution

Keep #13271 open until OpenClaw has a documented channel-scoped or agent/channel-scoped tool-call budget contract. A complete implementation should add typed config/schema/help/docs, carry reliable channel/request context into tool-call enforcement, count tool calls per turn in the runtime, define warn/block/auto-delegate overflow behavior, wire auto-delegation through sessions_spawn/sub-agent completion, and cover Discord plus at least one non-Discord channel. If maintainers prefer plugin ownership, first add the missing hook/runtime contract so a plugin can implement this cleanly.

## Summary

Current main does not implement the requested static cron payload. Cron payloads are still limited to `systemEvent` and `agentTurn`; `systemEvent` is injected into the main session/heartbeat path, and detached jobs require `agentTurn` and call `runIsolatedAgentJob`. The CLI, agent cron tool schema, Gateway protocol schema, and docs all expose only those two payload types, so a no-AI direct static channel send remains meaningful open work.

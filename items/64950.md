## Summary

Keep open. Current `main` still follows the exact unfixed paths from the report: both model-picker entry points load the full runtime catalog, and the gateway catalog used by `models.list` still proxies that same full catalog without a `models.mode === "replace"` branch.

There is no conditional path that switches to configured-only catalog data when `models.mode` is `"replace"`.

## Summary

Current `main` already changed macOS gateway restart to an `enable` + `kickstart -k` flow with bootstrap-only fallback when the LaunchAgent is actually unloaded, and the CLI restart path now repairs an installed-but-not-loaded LaunchAgent. The reported `bootout`/not-loaded failure mode no longer matches current code.

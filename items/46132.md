## Summary

Current main and the v2026.4.24 release contain the dashboard/gateway auth hardening needed for this report: dashboard token URLs are resolved from the configured gateway auth token, fragment tokens override stale UI state without persisting to localStorage, auth lockouts are not burned by missing browser credentials, token mismatches return structured recovery advice with bounded retry behavior, and daemon restart/install paths now detect or remove stale embedded service tokens.

## Summary

Keep this PR open. Current main still lacks configurable `embedding.timeoutMs` and `embedding.maxRetries` for the bundled `memory-lancedb` plugin, while the PR remains an active, scoped implementation candidate with recent author follow-up on review comments.

## Best Possible Solution

Keep this PR open. The best path is to review and land a scoped `memory-lancedb` implementation that adds validated `embedding.timeoutMs` and `embedding.maxRetries` to the runtime parser, bundled manifest schema/UI hints, OpenAI embedding constructor/request path, and focused config/runtime tests; resolve the `embedding.baseUrl` security concern and any stale LaunchAgent scope before merge.

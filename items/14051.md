## Summary

Keep open. Current main does not implement the requested activity-based heartbeat idle timeout: heartbeat config still exposes interval/activeHours/delivery/runtime-cost controls, but no `idleTimeoutMinutes` or equivalent activity gate. The related open issues are adjacent, not a canonical duplicate or superseding implementation.

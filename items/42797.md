## Summary

Keep open. Current `main` does not implement gateway-side auth-gated filtering for `models.list`; it still returns the allowlisted catalog or the full catalog. The request remains relevant for active gateway consumers, and the closest auth-filtering logic now lives in the CLI/model-row path rather than the gateway handler.

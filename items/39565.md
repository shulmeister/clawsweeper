## Summary

Keep #39565 open. Current main still has the pre-PR TUI long-token classifier, so punctuation/markdown-wrapped dotted identifiers and dotted IDs without underscores can still be chunked with inserted spaces. The linked bug #39505 remains actionable, and this PR is a focused formatter/test fix rather than obsolete work.

## Best Possible Solution

Keep this PR or the linked #39505 work open. The best path is a focused TUI formatter change in `src/tui/tui-formatters.ts` that classifies punctuation-stripped candidates, strips markdown asterisks at token edges, preserves dotted identifiers and digit-bearing hyphenated IDs, and keeps punctuation-only/prose-hyphen tokens chunkable, with regression coverage in `src/tui/tui-formatters.test.ts`.

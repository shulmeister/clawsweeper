## Summary

Keep PR #39565 open. Current main still has the pre-PR TUI sanitizer behavior that can insert spaces into punctuation-wrapped dotted IDs and long dotted IDs without underscores, while the PR remains a focused formatter/test fix tied to still-open bug #39505.

## Best Possible Solution

Keep #39565, or land a rebased/cherry-picked equivalent. The best path is the narrow TUI formatter fix in `src/tui/tui-formatters.ts`: classify the punctuation-stripped candidate, strip markdown `*` at token edges, preserve dotted identifiers and digit-bearing hyphenated IDs, keep punctuation-only and natural-language hyphenated long tokens chunkable, and retain focused regression tests in `src/tui/tui-formatters.test.ts`.

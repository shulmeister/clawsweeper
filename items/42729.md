## Summary

Keep open. Current `main` at `564f820efa2bb81ae49b8f55a93e41492ad31917` still lacks the stale-slot dedupe that PR #42729 adds, so the reported duplicate-after-restart path remains plausible and is not implemented on `main`.

## Evidence

- **Runnable check still treats any expired next run as due:** `isRunnableJob` returns `true` whenever `nextRunAtMs` is scheduled and `nowMs >= nextRunAtMs`, with no guard for `lastRunAtMs > nextRunAtMs`. That is the exact missing condition the PR adds.
  - file: [src/cron/service/timer.ts:894](https://github.com/openclaw/openclaw/blob/564f820efa2bb81ae49b8f55a93e41492ad31917/src/cron/service/timer.ts#L894)
  - command: `nl -ba src/cron/service/timer.ts | sed -n '857,920p'`
  - sha: [564f820efa2b](https://github.com/openclaw/openclaw/commit/564f820efa2bb81ae49b8f55a93e41492ad31917)
- **Startup path consumes stale expired slots before recompute:** `start()` clears stale running markers, then immediately calls `runMissedJobs()`. `planStartupCatchup()` loads with `skipRecompute: true` and calls `collectRunnableJobs(...)`, so a persisted stale expired `nextRunAtMs` is eligible for immediate replay before the later `recomputeNextRuns()` pass.
  - file: [src/cron/service/ops.ts:87](https://github.com/openclaw/openclaw/blob/564f820efa2bb81ae49b8f55a93e41492ad31917/src/cron/service/ops.ts#L87)

  - sha: [564f820efa2b](https://github.com/openclaw/openclaw/commit/564f820efa2bb81ae49b8f55a93e41492ad31917)
- **Main already has the matching stale-slot concept only in maintenance recompute:** `recomputeNextRunsForMaintenance()` already checks `lastRun >= nextRunAtMs` before advancing an expired slot, which supports the PR's diagnosis that the remaining gap is the timer/startup runnable path, not the maintenance repair path.
  - file: [src/cron/service/jobs.ts:521](https://github.com/openclaw/openclaw/blob/564f820efa2bb81ae49b8f55a93e41492ad31917/src/cron/service/jobs.ts#L521)

It does not exercise the reported stale expired slot where `nextRunAtMs` still points to an already-executed past time.

## Summary

Keep open. Current main has strong mitigations for state loss, including automatic pre-compaction memory flush, reset hooks, and user-facing context usage in /status, and the discussion also points to the useful openclaw-workspace-sync community workaround. But the requested passive agent-visible `context=X%` Runtime-line signal is still not implemented, so a meaningful part of the feature request remains missing.

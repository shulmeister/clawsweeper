## Summary

Keep this PR open. Current main still does not implement the central optimization: non-live implicit provider discovery returns unscoped before consulting configured model/provider hints. The PR is a plausible agents startup performance fix, but it should address the review-noted fallback test gap and be coordinated with overlapping open PR #63552 rather than closed as obsolete.

## Best Possible Solution

Keep this PR open for maintainer review. Ask the contributor to strengthen the fallback test, then review it together with #63552 and land or adapt one implementation that scopes non-live implicit provider discovery from configured provider/model hints while preserving broad discovery when hints are absent or unresolved.

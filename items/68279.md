## Summary

Keep open. Current `main` still exposes `openclaw/plugin-sdk` as a public package export, but the root SDK barrel does not export `stringEnum` or `optionalStringEnum` even though `src/plugin-sdk/core.ts` does. Docs now deprecate root-barrel imports, but that does not restore backward compatibility for existing external plugins, and the linked PR/comment history points to an active one-line fix rather than an obsolete report.

## Summary

Keep open. Current main still filters setup web-search providers by `onboardingScopes`, DuckDuckGo’s shared provider metadata still does not set `onboardingScopes: ["text-inference"]`, and the same author has an open linked PR (#65940) with closing syntax. That PR should be reviewed, refreshed if needed, and landed or superseded before this issue is closed.

## Best Possible Solution

Keep this issue open. Maintainers should review #65940 or ask for a refreshed patch that adds `onboardingScopes: ["text-inference"]` to the shared DuckDuckGo web-search provider metadata, including the contract/lightweight surface, and adds a focused regression assertion that the setup wizard options include DuckDuckGo. Once that PR or an equivalent fix merges, GitHub closing syntax can close this issue as implemented.

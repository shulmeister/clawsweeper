## Summary

Keep open. Current main still does not accept mediaLocalRoots under either channels.telegram or agents.defaults: the Telegram config schema and agent-defaults schema are strict and do not include that key. Current main has separate media-source root widening for concrete outbound media paths, which likely helps the related CLI-send problem, but it does not implement the config validator support requested in #47002.

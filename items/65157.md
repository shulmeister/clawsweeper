## Summary

Current `main` already wires CLI-backed runs through `before_prompt_build` and `agent_end`, the reported hook-policy flag is documented, and the shipped `v2026.4.23` release contains the same core CLI hook behavior.

## Evidence

- **CLI prompt-build hooks run on the CLI path:** `prepareCliRunContext` now checks for `before_prompt_build` / `before_agent_start` and resolves prompt-build hook results with CLI session history before executing the provider-backed run.
  - file: [src/agents/cli-runner/prepare.ts:311](https://github.com/openclaw/openclaw/blob/a5db42862d76fce18e5d7fa57f3907a543e61137/src/agents/cli-runner/prepare.ts#L311)
  - sha: [a5db42862d76](https://github.com/openclaw/openclaw/commit/a5db42862d76fce18e5d7fa57f3907a543e61137)
- **CLI runs emit agent_end lifecycle hooks:** `runPreparedCliAgent` computes CLI history, builds agent-end messages, and dispatches `agent_end` on success and failure for the shared CLI runner used by `claude-cli` and the same harness-backed path mentioned in the follow-up comment.
  - file: [src/agents/cli-runner.ts:301](https://github.com/openclaw/openclaw/blob/a5db42862d76fce18e5d7fa57f3907a543e61137/src/agents/cli-runner.ts#L301)
  - sha: [a5db42862d76](https://github.com/openclaw/openclaw/commit/a5db42862d76fce18e5d7fa57f3907a543e61137)
- **Shared lifecycle helper dispatches agent_end:** The harness helper calls `hookRunner.runAgentEnd(...)` when `agent_end` hooks are registered, which is the shared dispatch point used by the CLI runner.
  - file: [src/agents/harness/lifecycle-hook-helpers.ts:42](https://github.com/openclaw/openclaw/blob/a5db42862d76fce18e5d7fa57f3907a543e61137/src/agents/harness/lifecycle-hook-helpers.ts#L42)
  - sha: [a5db42862d76](https://github.com/openclaw/openclaw/commit/a5db42862d76fce18e5d7fa57f3907a543e61137)
- **Regression tests cover before_prompt_build on CLI preparation:** The checked-in test asserts CLI preparation invokes `runBeforePromptBuild(...)` with prior session messages and merges the returned prompt/system context.
  - file: [src/agents/cli-runner/prepare.test.ts:219](https://github.com/openclaw/openclaw/blob/a5db42862d76fce18e5d7fa57f3907a543e61137/src/agents/cli-runner/prepare.test.ts#L219)
  - sha: [a5db42862d76](https://github.com/openclaw/openclaw/commit/a5db42862d76fce18e5d7fa57f3907a543e61137)
- **Regression tests cover agent_end on successful and failed CLI runs:** The reliability tests assert `runAgentEnd(...)` fires for both successful CLI execution and failure cases, matching the two missing dispatch points reported in the issue.

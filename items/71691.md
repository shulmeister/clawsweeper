## Summary

Keep open. Current main has partial mitigations for heartbeat-only compaction and optional isolated heartbeat sessions, but the reported default mixed Telegram/session heartbeat path is not proven fixed: heartbeats still use the main session unless explicitly configured otherwise, and the compaction summary path still passes unfiltered message sets into summarization. The issue is recent, concrete, and salvageable.

## Evidence

The report used default compaction behavior, so this does not automatically fix the reported mixed-session path.

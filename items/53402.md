## Summary

Keep open. Current `main` does not implement macOS app localization: the native menu bar UI and settings still use hardcoded English strings, the macOS app bundle is configured with `CFBundleDevelopmentRegion` `en`, and the repo has no `apps/macos` localization bundles or `.strings`/`.xcstrings` resources for the app.

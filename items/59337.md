## Summary

Keep open. Current `main` still intentionally removes `openclaw nodes run`, blocks `system.run` via `nodes invoke`, and documents node shell execution only through the session-scoped `exec host=node` path. That does not satisfy this issue’s request to restore direct CLI execution on paired nodes.

## Evidence

- **Breaking change explicitly removed `nodes.run`:** The changelog for `2026.3.31` says the CLI and agent `nodes.run` shell wrapper were removed so node shell execution always goes through `exec host=node`.
  - file: [CHANGELOG.md:1686](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/CHANGELOG.md#L1686)
- **User docs confirm `nodes run` is gone:** The CLI docs state: use the `exec` tool with `host=node` instead of `openclaw nodes run`, and describe the `nodes` CLI as capability-focused rather than a shell-execution surface.
  - file: [docs/cli/nodes.md:64](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/docs/cli/nodes.md#L64)
- **Current CLI blocks shell execution through `nodes invoke`:** `register.invoke.ts` hard-blocks `system.run` and `system.run.prepare` and throws `use the exec tool with host=node instead`.
  - file: [src/cli/nodes-cli/register.invoke.ts:12](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/src/cli/nodes-cli/register.invoke.ts#L12)
- **Only session-scoped `/exec` override is documented:** The exec docs describe `/exec` as a per-session override for `host`, `security`, `ask`, and `node`, which matches the reporter’s complaint that the current workaround must be done inside a session.
  - file: [docs/tools/exec.md:144](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/docs/tools/exec.md#L144)
- **No replacement standalone CLI subcommand found:** A source search for `.command("exec")` / `.command('exec')` under `src/cli` and `src/commands` returned no matches, and the registered `nodes` subcommands are `status`, `pairing`, `invoke`, `notify`, `push`, `canvas`, `camera`, `screen`, and `location` only.
  - file: [src/cli/nodes-cli/register.ts:33](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/src/cli/nodes-cli/register.ts#L33)
  - command: `rg -n "\.command\(\"exec\"\)|\.command\('exec'\)" src/cli src/commands`

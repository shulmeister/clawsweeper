## Summary

Keep open. Current main has partial UX improvements for tool visibility, including live tool cards, collapsible tool messages, a show/hide tool-calls toggle, and a resizable details sidebar. It still does not implement the requested split dashboard where user/assistant communication and live tool/runtime activity are independently scrolling panels with separate layout/persistence semantics.

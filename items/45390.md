## Summary

Keep open. Current main has partial mitigations: daily/idle session reset policies, optional heartbeat isolated sessions, and pre-compaction memory flush. It still does not implement the requested session max-turn TTL or a pre-rotation memory persistence hook, and the issue discussion specifically calls out rotation safety/data-loss concerns that remain product/core lifecycle work.

## Evidence

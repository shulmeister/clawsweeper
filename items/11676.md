## Summary

Keep open. Current `main` still requires a Node-hosted Gateway and treats non-Node devices as companion nodes, so the request is not implemented. It also does not fit the allowed close buckets: it is coherent, not a stale bug report, and not clearly a ClawHub skill/plugin request.

## Summary

Keep open. Current main still requires a Node-hosted OpenClaw runtime, and the existing agent-harness/plugin surfaces let trusted plugins replace the low-level executor inside OpenClaw rather than provide a runtime-agnostic, no-Linux/no-Node core protocol for embedded devices. MimiClaw is a relevant external implementation, but it does not make the requested OpenClaw core portability work implemented or safely closable as ClawHub-only.

Keep PR #70205 open. Current main still has the reported behavior: `usage.cost` resolves one transcript scope through `loadCostUsageSummary` with no agent scope and caches only by date range, while `sessions.usage` already discovers across `listAgentsForGateway(config).agents`. The PR also fixes a real Control UI date-only label issue that is still present on main. Related PR #42063 was closed unmerged, and #13219 is a broader usage-logging enhancement rather than a canonical replacement for this bug fix.

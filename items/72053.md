## Summary

Keep open. Current main still has the #72019 duplication: the same subagent task is rendered into the system prompt and repeated in the first user turn. PR #72053 is a focused active fix with tests, and the related #72026/#72028 PRs are open alternate designs rather than a landed or maintainer-selected superseding fix.

## Best Possible Solution

Keep #72053 open for maintainer review. The best path is to select one canonical fix for #72019, ensure the long task appears exactly once in the first model request, preserve multiline/structured task text, and land regression coverage for both prompt assembly surfaces.

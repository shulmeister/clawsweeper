## Summary

Keep open. Current main still has no Google Chat user OAuth credential surface, and the reaction/upload/proactive-DM paths still use the same service-account `chat.bot` token path. The external related Google Chat file-upload work is useful context, but it does not supersede this OpenClaw bundled Google Chat enhancement.

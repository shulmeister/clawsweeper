## Summary

Keep open. Current main still lacks manifest-level `configPatch`: the native manifest type and docs do not define it, plugin install results do not carry it, and the install command/persist path only enables the plugin, updates allowlist/install ledger state, and writes config. The prior discussion is useful: the claude-search-proxy use case and MORNLONG’s `tools.alsoAllow` example describe real remaining friction. The related PR #43421 in the provided context was closed unmerged, so it does not supersede the issue.

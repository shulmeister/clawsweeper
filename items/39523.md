## Summary

Keep open. Current `main` still does not implement Feishu channel `mediaLocalRoots` as a user-facing config field, and the PR is not obsolete. It addresses a real Feishu config/schema/docs gap tied to #39506, but the PR as written appears incomplete because normal outbound sends already inject agent-scoped roots, which would shadow a simple `mediaLocalRoots ?? account.config?.mediaLocalRoots` fallback.

## Best Possible Solution

Keep the PR open, but require a revised Feishu-owned fix that defines the intended precedence between account-config roots and agent-scoped outbound roots, updates Feishu schema/docs/generated metadata, and adds regression tests for configured directory arrays, `"any"`, and the normal outbound message-send path.

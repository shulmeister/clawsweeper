## Summary

Keep #59088 open. It is a recent, concrete Control UI enhancement request. Current main exposes limited agent workspace file RPCs for bootstrap files, but I did not find high-confidence evidence that the requested full workspace file tree/editor for general text files, shortcuts, and read-only mode is already implemented. This also remains plausibly core Control UI work rather than a ClawHub-only plugin idea.

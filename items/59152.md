## Summary

Control UI slash autocomplete is already dynamic on current `main`: it refreshes from the gateway `commands.list` catalog, and that catalog includes `user-invocable` workspace skill commands. The same implementation is present in release `v2026.4.23`.

## Evidence

- **Control UI now loads runtime slash commands:** The slash menu is no longer static-only. `refreshSlashCommands` calls `commands.list` with `scope: "text"` and replaces the menu entries from the returned runtime catalog.
  - file: [ui/src/ui/chat/slash-commands.ts:431](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/ui/src/ui/chat/slash-commands.ts#L431)

  - file: [ui/src/ui/app-chat.ts:542](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/ui/src/ui/app-chat.ts#L542)

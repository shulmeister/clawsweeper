## Summary

Keep open. Current main still only supports one-time approval, durable allow-always, and deny for exec approvals. The requested time-bounded approval mode, duration parsing, approvedUntil/defaultDuration storage, timed UI action, and timed approvals CLI visibility are not implemented on main.

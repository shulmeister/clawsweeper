## Summary

Current `main` already contains a cron-specific fix for sandbox config merging. Isolated cron runs now preserve global `agents.defaults.sandbox` settings while still allowing per-agent sandbox overrides, and there is a dedicated regression test covering `sandbox.mode: "all"` with partial agent sandbox overrides.

## Summary

Current main already preserves Control UI/webchat uploaded image attachments by writing persisted media paths onto the user transcript entry and rendering those transcript media paths back into Control UI image previews. The same implementation is present in the shipped v2026.4.23 release.

## Evidence

- **Gateway persists user upload media onto transcript entries:** `buildChatSendTranscriptMessage` adds `MediaPath`/`MediaPaths`/`MediaType`/`MediaTypes` for saved inbound images, and `rewriteChatSendUserTurnMediaPaths` backfills those fields onto the stored user turn when needed.

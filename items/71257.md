Keep #71257 open. Current main still persists session-memory transcript text verbatim, so the user-visible bug described by #71031 is not implemented on main. The PR is a plausible focused fix for a real bundled hook path, though it needs maintainer review and cleanup before landing, especially because it overlaps with #69961 and includes ignored `.artifacts/tsgo-cache` files.

## Best Possible Solution

Keep #71257 open for maintainer review.

## Summary

Keep open. Current `main` has partial overhead reductions and visibility tooling, but #14785 is not fully implemented: the `message` tool still builds a broad monolithic schema with many cross-channel fields, tests still lock in cross-channel leakage for some schema extras, the `cron` tool still embeds a long tutorial in its description, and docs still describe tool schemas as always counting toward context.

## Evidence

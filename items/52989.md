## Summary

Keep open. Current `main` does not contain this fix: `scripts/generate-base-config-schema.ts:73` still compares `import.meta.url` against `new URL(process.argv[1] ?? "", "file://").href`, while many other repo scripts use `pathToFileURL(process.argv[1] ?? "").href` for direct-entry detection.

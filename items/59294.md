## Summary

Keep open. Current `main` has absorbed some of this work already, especially thread-aware session routing and proactive thread targeting, but it does not cover several concrete fixes this PR still carries: mention-preserving command parsing, thread-scoped pending-history/debounce partitioning, and the mixed `file.download.info` attachment fallback described in the PR.

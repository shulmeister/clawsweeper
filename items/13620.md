## Summary

Keep open. Current `main` does not implement outbound Telegram album/media-group sending for the message tool. The Telegram action path still accepts only a single `mediaUrl`, the send implementation only branches over one media item at a time, and multi-media payload support is explicitly sent as a sequence of separate messages rather than `sendMediaGroup`. Existing media-group code on `main` is inbound album handling, not the requested outbound feature.

## Evidence

- **Message tool accepts one media input:** `handleTelegramAction()` for Telegram `sendMessage` reads only `mediaUrl`/`media` as a single string and passes a single `mediaUrl` into `sendMessageTelegram()`. There is no `mediaGroup`, `album`, or string-array parameter on the message-tool path.
  - file: [extensions/telegram/src/action-runtime.ts:308](https://github.com/openclaw/openclaw/blob/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e/extensions/telegram/src/action-runtime.ts#L308)
  - command: `nl -ba extensions/telegram/src/action-runtime.ts | sed -n '305,372p'`
  - sha: [272bd59e7a8a](https://github.com/openclaw/openclaw/commit/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e)
- **Send implementation is single-media only:** `TelegramSendOpts` defines only `mediaUrl?: string`, and `sendMessageTelegram()` trims `opts.mediaUrl` then sends exactly one file through `sendAnimation`/`sendPhoto`/`sendVideo`/`sendVoice`/`sendAudio`/`sendDocument`. There is no outbound `sendMediaGroup` path.
  - file: [extensions/telegram/src/send.ts:74](https://github.com/openclaw/openclaw/blob/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e/extensions/telegram/src/send.ts#L74)

  - sha: [272bd59e7a8a](https://github.com/openclaw/openclaw/commit/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e)
- **Multiple media are sent sequentially, not as an album:** The outbound adapter resolves `mediaUrls` then calls `sendPayloadMediaSequenceOrFallback()`, invoking `send()` once per item. The regression test asserts two separate `sendMessageTelegram()` calls for two media URLs.
  - file: [extensions/telegram/src/outbound-adapter.ts:75](https://github.com/openclaw/openclaw/blob/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e/extensions/telegram/src/outbound-adapter.ts#L75)

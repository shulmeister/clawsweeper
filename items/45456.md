## Summary

Close #45456 as ClawHub/plugin scope. The issue requests optional operator policy for channel/chat token budgets, and current main already exposes plugin hooks for pre-model dispatch gating, prompt budget injection, inbound channel/chat metadata, LLM usage observation, and plugin-owned commands/CLI reporting. Current main does not implement a built-in channels.*.rateLimits schema, so this is not implemented_on_main; it is better expressed as a code plugin unless a concrete missing plugin API is identified.

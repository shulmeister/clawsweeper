## Summary

Keep #28913 open. Current main still has memory-lancedb autoCapture driven by trigger heuristics, but no captureFilter/excludeWrapped config and no wrapper/source-aware filter before auto-captured memories are embedded and stored.

## Best Possible Solution

Keep the issue open. Implement a backward-compatible memory-lancedb option such as plugins.entries.memory-lancedb.config.captureFilter.excludeWrapped, add it to the typed parser and openclaw.plugin.json schema/UI hints, apply a robust wrapper/source-aware pre-filter before shouldCapture in the agent_end auto-capture path, and add regression tests showing wrapped EXTERNAL_UNTRUSTED_CONTENT with trigger words like always is skipped while normal direct preference capture still stores memories.

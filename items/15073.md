## Summary

Keep open. Current main has per-agent model fallback lists and generic bootstrap trimming, but the observable request remains missing: model fallback retries still switch only provider/model while keeping the same session agent, workspace, agent dir, session file, and bootstrap context path. The linked closed issues (#13529, #7351, #13557) are useful related context requests, but they appear to be superseded by this broader open item rather than a canonical replacement for it.

## Summary

Current main already implements the requested compaction visibility: `agents.defaults.compaction.notifyUser` sends brief start/end compaction notices, the behavior is documented, and Discord/Telegram wire compaction lifecycle callbacks into status indicators.

Closing as implemented on main after a Codex review. Current `main` already has a built-in compaction visibility path: `agents.defaults.compaction.notifyUser` is documented in `docs/concepts/compaction.md:133` and wired in `src/auto-reply/reply/agent-runner-execution.ts:641` and `src/auto-reply/reply/agent-runner-execution.ts:1248` to send brief `🧹 Compacting context...` / completion notices during compaction. If there is still a separate gap beyond these existing notices on current builds, please open a new issue with the exact channel, config, and reproduction on a current version.

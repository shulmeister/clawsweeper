## Summary

Current `main` implements the missing WebChat/Gateway image-upload path: dashboard image attachments are normalized from nested base64 payloads, forwarded as structured image input, preserved for text-only primary models as `media://inbound/*` refs, and documented with MiniMax image-model fallback behavior. The latest tagged release still lacks part of this behavior, so this is fixed on `main` but not yet in a shipped release.

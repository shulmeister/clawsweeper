## Summary

The proposal is a broad monitoring/orchestration layer that current OpenClaw scope pushes to external plugins/skills rather than core. Current main already exposes health/status/diagnostic primitives for an external monitor, and the reporter’s latest comment explicitly agrees this belongs in ClawHub.

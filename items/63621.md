## Summary

PR #63621 should close as superseded. It adds a large zh-TW glossary under the docs i18n asset directory, but the PR’s stated goal is Control UI terminology. Current main shows the Control UI generator reads `ui/src/i18n/.i18n/glossary.zh-TW.json`, the docs pipeline does not currently generate zh-TW docs, and the actual remaining Control UI zh-TW terminology/parity work is already tracked by #59706, with #52357 as the earlier related PR.

## Best Possible Solution

Close this PR as superseded for maintainer cleanup. Keep #59706 as the canonical Control UI zh-TW terminology/parity item, and ask any durable glossary follow-up to come back as a smaller targeted PR against `ui/src/i18n/.i18n/glossary.zh-TW.json` with demonstrated Control UI pipeline misses rather than a bulk docs glossary import.

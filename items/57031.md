## Summary

Keep open. Current `main` still swallows queued write failures with no warning, and `QueuedFileWriter` now also backs default-on runtime trajectory capture, so the report is still materially relevant. The reporter’s follow-up lowers severity from data-loss bug to observability gap, but it does not show the item is fixed or obsolete.

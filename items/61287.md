## Summary

Current main already fixes the reported claude-cli sessions context inflation: session persistence now uses the last call or CLI snapshot for context totals and latest cache counters instead of cumulative run/session usage, and the CLI runner marks CLI provider usage as a fresh context snapshot. Regression tests cover the lastCallUsage and claude-cli snapshot cases. The fix is present in the v2026.4.24 tag/release commit.

## Summary

Keep open. Current `main` still has the bug: temporal decay only recognizes top-level `memory/YYYY-MM-DD.md` files, so nested dated memory files remain exempt from decay, and the PR stays a small, targeted fix with follow-up review concerns already addressed in the discussion.

## Summary

Close as duplicate/superseded. #66655 was a narrow UI-side patch against an older render helper; current main already moved the thinking picker into `ui/src/ui/chat/session-controls.ts` and reads `thinkingDefault` from session/default payloads. The remaining configured-default mismatch is now the gateway session-row/default payload, tracked by #70967 with #70981 as the current open landing path.

## Best Possible Solution

Close #66655 as superseded and keep maintainer attention on #70981/#70967, where the gateway session-row/default payload can be fixed with a config-aware `thinkingDefault` resolver and regression coverage. Keep the current UI behavior that consumes `thinkingDefault` from session/default payloads.

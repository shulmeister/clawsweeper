## Summary

Current main still lacks `requireAuth: true` on QQBot `/bot-clear-storage`, while the registry and bridge code route only `requireAuth` commands through the framework auth path. PR #71158 is a narrow, still-relevant fix rather than obsolete cleanup.

## Best Possible Solution

Keep PR #71158 open and review/merge a focused fix that adds `requireAuth: true` to `/bot-clear-storage`, adds the framework-registry regression test, and includes a user-facing security changelog entry with the required attribution.

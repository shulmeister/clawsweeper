## Summary

Current main already contains the Ollama discovery and onboarding behavior this report was asking for: minimal Ollama provider config no longer blocks implicit local discovery, local models are loaded from `/api/tags`, and cloud-only onboarding now hydrates hosted models from `https://ollama.com/api/tags`. The changelog shows this class of Ollama discovery fix shipped after the reporter's `2026.4.10` build.

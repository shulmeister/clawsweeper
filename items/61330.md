## Summary

Keep #61330 open. Current main does not expose a Feishu Bitable delete-record tool: the registered Bitable tools cover metadata, fields, list/get/create/update records, app creation, and field creation, while record deletion is only used internally to clean placeholder rows after creating a new Bitable. The issue is recent, has no protected-label or maintainer-author signal in the provided context, and the only comment is a volunteer asking to work on it, not a duplicate/superseding implementation link.

## Summary

Keep open. Current main still lacks general post-write verification for the reported false-success path, and the linked fix PR #67202 remains open rather than merged or shipped.

## Best Possible Solution

Keep #67136 open until PR #67202 or an equivalent maintainer-reviewed fix lands. The best implementation is a shared post-write verification helper used by sandbox write, sandbox edit, and unrestricted host write paths, with regression tests for bridge.writeFile returning success while stat shows missing, non-file, or wrong-size output.

## Summary

Current `main` does not implement per-account SOUL file support. Bootstrap loading still reads only `SOUL.md`, there is no `soulFile` config/schema or `--soul` CLI/onboarding support in the checked-out tree, and the author explicitly opened PR #49044 as the fresh main-based successor to this conflicted PR. This should stay open rather than be closed as implemented or obsolete.

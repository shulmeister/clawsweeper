## Summary

Current `main` already normalizes OpenAI-style `prompt_tokens` / `completion_tokens` usage and feeds that normalized data into the assistant message path that was reported as showing `0` context tokens. The latest shipped release `v2026.4.22` also contains the same implementation and regression coverage.

## Summary

Keep #7909 open. Current main still exposes only the assistant-message markdown copy button, and there is no plain-text copy menu, alternate action, or preference wired into the Control UI chat path. The existing 2026-04-26 maintainer-side Codex comment also matches current main.

## Best Possible Solution

Keep #7909 open until the Control UI adds a real assistant-message plain-text copy path. The likely implementation belongs near `ui/src/ui/chat/copy-as-markdown.ts` and `ui/src/ui/chat/grouped-render.ts`: expose a dropdown or adjacent action for Markdown vs plain text, reuse or adapt a tested markdown-to-plain-text helper, and add focused clipboard/UI tests for both modes.

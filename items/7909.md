## Summary

Current main still only renders an assistant-message copy action that copies the markdown text unchanged. There is no plain-text copy action, dropdown/menu, preference, or test coverage for the requested behavior, and the latest issue discussion already contains a maintainer-side Codex review saying to keep this open.

## Best Possible Solution

Keep #7909 open until the Control UI adds a real assistant-message plain-text copy path. The likely implementation belongs near ui/src/ui/chat/copy-as-markdown.ts and ui/src/ui/chat/grouped-render.ts: expose a Markdown vs plain-text choice, reuse or adapt a tested markdown-to-plain-text helper, and add focused UI/clipboard tests for both formats.

## Summary

Keep open. Current main has a Control UI Core Files editor for raw workspace Markdown files and runtime injection of USER.md/MEMORY.md, but it does not implement the requested pinned key-value facts panel, structured pinned-context storage, or guaranteed-context UI workflow. The issue is recent, author association is NONE, has no protected labels, and the provided related items do not supersede this request.

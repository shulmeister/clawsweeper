## Summary

Keep open. Current `main` still returns the raw parsed config object for `/config show`, so the reported secret-leak behavior remains present and is still shipped in `v2026.4.23`. The PR addresses a real bug and follows an existing redaction pattern, but the review comment is valid that its added tests, as posted, do not exercise the schema-hint branch because the mocked `uiHints` omit `sensitive: true`.

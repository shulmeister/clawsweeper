## Summary

Keep open. Current main still serializes the UI form state directly for `config.set` / `config.apply` and I do not see the focused #60971 behavior on main: dropping unresolved `__OPENCLAW_REDACTED__` placeholders while preserving placeholders backed by the original raw config.

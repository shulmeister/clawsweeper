## Summary

Keep #45982 open. Current main still does not set `encoding_format: "float"` in the `memory-lancedb` OpenAI embeddings request, and the colocated tests still assert requests with only `model`/`input` or `model`/`input`/`dimensions`. The linked PR discussion also supports that this is unresolved: #45985 was closed unmerged, while #45986 and #46069 remain open attempts to add the same missing field.

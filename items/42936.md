## Summary

Keep open. Current `main` has not implemented this PR’s boolean-or-object Feishu streaming config or the throttle/title plumbing, and auto-mode streaming is only started during `block`/`final` delivery rather than on partial text. The branch is stale relative to current dispatcher logic, but the request is not already implemented on `main` and is not safe to close as obsolete.

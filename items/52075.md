## Summary

Keep open. Current `main` has a new `docs/gateway/openai-http-api.md`, but it does not fully implement this PR’s requested guidance. The page now documents auth/security and basic routing, yet it still lacks the PR’s explicit decision guidance for private custom mobile clients, the HTTP-vs-WebChat/WS-vs-channel-plugin comparison, and the stable-session `curl` example using `user`.

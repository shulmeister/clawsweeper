## Summary

Keep open. Current `main` does not implement this PR's loopback trusted-proxy allowance; instead it explicitly rejects loopback trusted-proxy sources in code, tests, and docs. The PR discussion also surfaces unresolved security/product tradeoffs, so there is not strong evidence for any conservative close reason.

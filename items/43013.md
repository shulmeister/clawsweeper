## Summary

Keep open. Current main still does not include the PR’s core selected-option handling or targeted regression coverage for the Agents page model selector. The PR also has review feedback showing the proposed unconditional `?selected` change can regress inherited model display, so it is neither obsolete nor safely merge-ready as-is.

## Best Possible Solution

Keep #43013 open, but require a smaller maintainer-quality fix: separate the option included for display from the option selected on initial render. Default agents should select `agents.defaults.model.primary`; non-default agents with no explicit model should keep the empty `Inherit default` value; missing explicit models should remain labeled `Current (...)`. Add a focused browser/view regression test for the initial render.

## Summary

Keep open. Current `main` still runs the Android companion as a long-lived `dataSync` foreground service on `targetSdk 36`, and the affected service code is unchanged between the latest release `v2026.4.23` and current `main`. I found no implementation or test on `main` that addresses the reported `ForegroundServiceStartNotAllowedException` path.

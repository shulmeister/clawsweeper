## Summary

Current `main` does not implement model or token-usage status in the Control UI topbar. The topbar still renders only navigation, search, chat mobile controls, and theme controls; model/status data remains confined to the Debug and Usage/Overview surfaces.

## Evidence

- **Topbar markup lacks model/token display:** The current topbar renders `dashboard-header`, search, and `topbar-status`, but `topbar-status` contains only the chat mobile toggle and theme toggle. There is no model name or token-usage element in the header.

## Summary

Current `main` does not implement per-agent `compaction` or `contextPruning` overrides under `agents.list[]`. The agent-entry schema is strict and omits both keys, while those settings are only defined under `agents.defaults`.

## Evidence

- **Per-agent agent config type lacks the requested keys:** `AgentConfig` lists supported per-agent overrides like `model`, `memorySearch`, `contextLimits`, `heartbeat`, `sandbox`, and `tools`, but there is no `compaction` or `contextPruning` field.
  - file: [src/config/types.agents.ts:70](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/src/config/types.agents.ts#L70)
  - command: `nl -ba src/config/types.agents.ts | sed -n '1,180p'`
  - sha: [8866544ffefb](https://github.com/openclaw/openclaw/commit/8866544ffefb42ca234c66c337134fb60743d8d0)
- **Strict agent-entry schema would reject those keys:** `AgentEntrySchema` is `.strict()` and enumerates allowed `agents.list[]` properties. It includes `contextLimits` and other overrides, but no `compaction` or `contextPruning`, so the issue's reported validation error still matches current code.
  - file: [src/config/zod-schema.agent-runtime.ts:812](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/src/config/zod-schema.agent-runtime.ts#L812)
  - command: `nl -ba src/config/zod-schema.agent-runtime.ts | sed -n '780,900p'`
  - sha: [8866544ffefb](https://github.com/openclaw/openclaw/commit/8866544ffefb42ca234c66c337134fb60743d8d0)
- **Compaction and pruning are only defined in defaults schema:** `AgentDefaultsSchema` defines `contextPruning` and `compaction` under `agents.defaults`, confirming they are global config keys today rather than per-agent overrides.
  - file: [src/config/zod-schema.agent-defaults.ts:123](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/src/config/zod-schema.agent-defaults.ts#L123)

- **User docs only document defaults-scope keys:** The configuration docs document `agents.defaults.compaction` and `agents.defaults.contextPruning`, with no matching `agents.list[].compaction` or `agents.list[].contextPruning` section.
  - file: [docs/gateway/config-agents.md:532](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/docs/gateway/config-agents.md#L532)

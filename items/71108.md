## Summary

Keep PR #71108 open. Current main still has the live/history mismatch paths the PR describes: injected chat broadcasts are only envelope/directive-stripped, live assistant chat deltas are emitted without commentary-phase filtering, unrelated session-scoped tool output still flushes active chat stream state, and terminal chat finals can still trigger history reload/reset without the active-run guard proposed by the PR. The discussion also contains a useful bot review and an unresolved implementation concern, so this PR still warrants maintainer review rather than cleanup closure.

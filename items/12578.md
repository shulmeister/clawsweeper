## Summary

Keep open. Current main still only supports per-session `responseUsage`; the requested `agents.defaults.usageDefault` / `responseUsageDefault` behavior is not implemented on main. The GitHub context shows related open PRs #48635 and #69618, but neither is merged into the reviewed main SHA, so closing the issue now would be premature.

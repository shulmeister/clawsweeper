## Summary

Keep #46565 open. Current main has partial mitigation for the reported heartbeat/subagent collision, but the broader feature request is not implemented: there is no configurable session-type bootstrap exclude contract, no bootstrap `!command` preprocessing, and current time is intentionally not injected into the stable system prompt.

## Evidence

- **Hardcoded session filtering only:** Current main filters bootstrap files for subagent/cron sessions with a fixed allowlist, keeping AGENTS.md, TOOLS.md, SOUL.md, IDENTITY.md, and USER.md while dropping HEARTBEAT.md/BOOTSTRAP.md/MEMORY.md. That helps the reported HEARTBEAT.md collision, but it is not the requested configurable `bootstrap.exclude` or section-level session-type filtering.

## Summary

PR #64600 is a draft bundled Servepath provider. Current main does not ship Servepath, but the requested capability is an optional hosted OpenAI-compatible provider integration that the current plugin/provider API already supports through external provider plugins and ClawHub/npm distribution. The PR body itself says Servepath already works through the generic custom-provider path; the remaining value is first-class discovery/onboarding, which belongs in a provider plugin rather than core absent a missing SDK seam or maintainer signal.

## Best Possible Solution

Close the core PR and move Servepath to an external provider plugin published through ClawHub/npm. If extracting it exposes a real missing generic provider-plugin or onboarding API, open a focused SDK/core issue or PR for that seam; any unrelated runtime bug fixes from the PR should be resubmitted separately with reproduction evidence.

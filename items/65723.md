## Summary

Current `main` already uses Discord-native button approvals that stay pending until the approval expiry window and resolve via Discord interactions/Gateway, so the reported short DM approval timeout is obsolete.

## Evidence

- **Discord docs describe native DM button approvals:** The Discord docs now state that Discord supports button-based approval handling in DMs, that buttons are the primary approval UX, and that approvals expire after 30 minutes by default.
  - file: [docs/channels/discord.md:882](https://github.com/openclaw/openclaw/blob/c150110e02550eaf6c56db0c851dc62fc0ca88e1/docs/channels/discord.md#L882)

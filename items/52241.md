## Summary

Keep open. Current `main` does not show this feature as implemented, and the WeChat integration is still documented as an external Tencent-maintained plugin rather than code owned in this repo. There is not enough evidence to close this as fixed, unreproducible, ClawHub-scoped, incoherent, or stale.

## Evidence

- **WeChat integration is external to this repo:** Current docs say OpenClaw connects to WeChat through Tencent's external `@tencent-weixin/openclaw-weixin` plugin, and that the WeChat-specific runtime is owned by the external plugin rather than OpenClaw core.
  - file: [docs/channels/wechat.md:10](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/docs/channels/wechat.md#L10)
  - sha: [60f7a59f5ed5](https://github.com/openclaw/openclaw/commit/60f7a59f5ed581839cfd1d52583449c5aa1f03a4)
- **Advertised capabilities are still limited:** The same docs still describe the plugin as supporting direct chats and media, with group chats not advertised by current capability metadata. There is no mention of reading shared public-account articles.
  - file: [docs/channels/wechat.md:13](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/docs/channels/wechat.md#L13)
  - sha: [60f7a59f5ed5](https://github.com/openclaw/openclaw/commit/60f7a59f5ed581839cfd1d52583449c5aa1f03a4)
- **Core only knows the channel id/alias:** A current test only verifies that `weixin` normalizes to the `openclaw-weixin` channel id in the plugin registry; it does not indicate any built-in article-reading behavior in core.
  - file: [src/channels/registry.helpers.test.ts:43](https://github.com/openclaw/openclaw/blob/60f7a59f5ed581839cfd1d52583449c5aa1f03a4/src/channels/registry.helpers.test.ts#L43)
  - sha: [60f7a59f5ed5](https://github.com/openclaw/openclaw/commit/60f7a59f5ed581839cfd1d52583449c5aa1f03a4)
- **No bundled weixin extension exists in this checkout:** The `extensions/` tree in current `main` does not contain a `weixin` package, so there is no in-repo implementation here to inspect for a shipped fix.

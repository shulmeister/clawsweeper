## Summary

Keep #57638 open. Current main does not implement a cron-scoped defaults block for agentTurn model, delivery, contextTokens, or sessionTarget. The related reports only cover narrower pieces: #9155 remains open for delivery defaults, #46777 addressed fallback propagation but not cron.defaults, and #52706/#52769 do not provide a merged canonical fix for the broader request.

## Best Possible Solution

Keep this open as a core product/config request. A complete fix would add a typed and documented cron.defaults block for agentTurn jobs, validate it in the strict config schema, apply it with explicit per-job payload/delivery settings winning, update CLI/UI creation and previews as needed, and cover model/fallback, delivery, contextTokens, and any sessionTarget default behavior with focused regression tests.

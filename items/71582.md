## Summary

Keep PR #71582 open. Current main does not contain the requested routing fix: exec/system-event wakes still use the older scoped wake call, heartbeat wake overrides still only model `target`, and the runner still pre-merges requested heartbeat overrides before `runHeartbeatOnce`. The PR remains meaningful maintainer work rather than something already implemented, duplicate, or out of scope.

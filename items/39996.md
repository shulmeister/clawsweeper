## Summary

Keep open. Current main still lacks PR #39996's requested `Global tools profile` overview row/formatter for `openclaw status` and `openclaw status --all`, and the shared status overview surface does not carry `tools.profile` data today. The PR remains coherent and useful, but it needs a refresh for the current shared overview pipeline plus maintainer judgment on global-baseline versus effective-profile wording.

## Best Possible Solution

Keep the PR open. Refresh it against the shared status overview surface, decide whether the row should show only the global baseline or a resolved effective-profile summary, and add focused regression coverage for both `openclaw status` and `openclaw status --all`.

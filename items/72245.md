## Summary

Keep this PR open. Current main still lacks the proposed resolved provider hook-plugin cache, provider-ref cache-key normalization, negative lookup caching, and the focused tests. The related #71807 is closed unmerged and does not supersede this narrower provider-runtime cache hardening.

## Best Possible Solution

Keep this PR open for maintainer review as a narrow shared provider-runtime performance improvement.

## Close Comment

Closing after a Codex review because this request belongs to the external WeChat plugin, not `openclaw/openclaw` core. Current `main` documents WeChat as the external `@tencent-weixin/openclaw-weixin` plugin and explicitly says "The WeChat code does not live in the OpenClaw core repo" and that the WeChat-specific runtime is owned by that external plugin (`docs/channels/wechat.md:10-11`, `docs/channels/wechat.md:27-43`). The channel index also labels WeChat as an external plugin (`docs/channels/index.md:35`), and this checkout does not contain a bundled Weixin implementation to patch here. This looks like a valid feature request for the WeChat plugin itself, but not for OpenClaw core.

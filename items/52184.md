## Summary

Keep open. Current `main` still prefers the currently running Node executable for `openclaw gateway install`, and the stabilization logic only rewrites Homebrew Cellar paths. There is no Volta-specific normalization to `~/.volta/bin/node`, and the codebase still warns that version-manager Node paths can break after upgrades.

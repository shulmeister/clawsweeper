## Summary

Current `main` already prevents the reported blind `wiki.importInsights` / `wiki.palace` calls by gating those requests on advertised gateway methods and `memory-wiki` enablement, and the shipped `v2026.4.23` release contains the same behavior and regression coverage.

## Evidence

- **Dreaming controller gates optional wiki RPCs:** `canCallMemoryWikiMethod` first checks `hello.features.methods` and otherwise falls back to whether the bundled `memory-wiki` plugin is enabled; both wiki loaders bail out without making a request when that gate is false.

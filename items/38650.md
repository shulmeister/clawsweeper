## Summary

PR #38650 should close as superseded. The remaining user problem is still the workspace bootstrap injector following symlinked AGENTS/SOUL/USER/MEMORY files, and that is now tracked by the still-open #38622/#40210 reports plus the newer, targeted #40230 PR. Current main no longer reads bootstrap files through the `fs-safe` path this PR changes, and the `fs-safe` workspace-file API still has test-covered symlink rejection behavior.

## Summary

Keep open. Current `main` still appends extra bootstrap files without any path dedupe, and bootstrap context injection still iterates every entry it receives. The issue title likely misattributes the source to `boot-md`; current docs/code show `boot-md` only runs on `gateway:startup`, while `agent:bootstrap` hooks mutate injected bootstrap files. The linked PR #59344 indicates maintainers already identified the same fix direction, but that dedupe is not present on `main` at `78a431ed224b6f5b9d58672346551a5b411ad200`.

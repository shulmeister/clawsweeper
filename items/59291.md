## Summary

Current `main` already avoids the reported failure mode. The stream wrappers in the embedded Anthropic/tool-call path keep the original stream object and only wrap its async iterator, so downstream `stream.result.bind(stream)` calls still work. The same interface-preserving implementation is present in the shipped `v2026.4.23` tree, with regression coverage for Anthropic-compatible streamed tool-call handling.

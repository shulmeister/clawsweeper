## Summary

Keep #33329 open. Current main has provider-owned discovery toggles for GitHub Copilot, Hugging Face, Ollama, and Bedrock, with focused tests and Bedrock documentation. The requested documentation is still incomplete for Copilot, Hugging Face, and Ollama, and I did not find the later-requested global disable-all model discovery switch. The related PR #33365 is closed unmerged, so it is not a canonical superseding item.

## Best Possible Solution

Keep the issue open until maintainers either document the shipped plugin-owned disable paths for `plugins.entries.github-copilot.config.discovery.enabled`, `plugins.entries.huggingface.config.discovery.enabled`, and `plugins.entries.ollama.config.discovery.enabled`, or explicitly decide that manifest UI hints are sufficient. Separately, make a product call on the requested global disable-all model discovery flag; if accepted, implement it in the provider discovery/config path and document it alongside the provider-specific controls.

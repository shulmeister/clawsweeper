## Summary

Keep #33329 open. Current main has provider-owned discovery toggles for Bedrock, GitHub Copilot, Hugging Face, and Ollama, but public docs still only document the Bedrock discovery config path. I also did not find the requested global disable-all model discovery switch, and the related PR #33365 is closed unmerged rather than a canonical replacement.

## Best Possible Solution

Keep the issue open until maintainers document the shipped plugin-owned discovery toggles for GitHub Copilot, Hugging Face, and Ollama in the public provider/config docs, then either implement a global disable-all model discovery control or split that later request into a narrower follow-up with the intended config path and interaction with `models.mode`.

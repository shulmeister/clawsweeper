## Summary

Keep #33329 open. Current main implements per-provider discovery toggles for Bedrock, GitHub Copilot, Hugging Face, and Ollama, but the public docs still only document the Bedrock discovery config path, and the later requested global disable-all model discovery switch is not present.

## Best Possible Solution

Keep the issue open until the public provider/config docs document `plugins.entries.github-copilot.config.discovery.enabled`, `plugins.entries.huggingface.config.discovery.enabled`, and `plugins.entries.ollama.config.discovery.enabled`; then either add a global disable-all model discovery control or split that request into a focused follow-up with the intended config path and interaction with `models.mode`.

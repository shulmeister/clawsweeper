## Summary

Keep open. Current main still forwards `temperature` through the Anthropic Messages transport for `claude-opus-4-7` when thinking is not enabled, and the compaction path can reach that transport. The PR is a focused fix for a live regression reported in #72123, but it needs cleanup and regression coverage before merge rather than closure.

## Best Possible Solution

Keep the PR open, or replace it with an equivalent maintainer patch, that strips `temperature` for `claude-opus-4-7` in `src/agents/anthropic-transport-stream.ts` before payload construction and option propagation. Add focused tests in `src/agents/anthropic-transport-stream.test.ts` proving Opus 4.7 omits temperature while non-Opus models still forward it, then clean the indentation before merge.

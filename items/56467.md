## Summary

Keep open. Current `main` does not implement the requested consecutive-failure tracking or agent-default failure policy keys, and the issue is still a plausible product request. OpenClaw's existing plugin hooks make a ClawHub/plugin implementation plausible, but that scope signal is not strong enough for a conservative auto-close.

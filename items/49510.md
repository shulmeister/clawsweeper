## Summary

Current `main` contains the gateway handshake fix that matches this report’s workaround and symptom pattern. The issue was filed against `2026.3.13`; the changelog for `2026.3.22` records a fix that raises the default pre-auth WebSocket handshake timeout from 3s to 10s so healthy local CLI connections stop getting dropped, and current code implements that timeout plus a supported override.

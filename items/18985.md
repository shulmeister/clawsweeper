## Summary

Keep open. Current main still does not implement the requested Windows 11 MSYS2 install.sh support or fish-specific PATH guidance: the Unix installer still accepts only macOS, Linux, or WSL, and the PATH persistence/warning text remains bash/zsh-only. Docs still present install.sh as macOS/Linux/WSL and Windows as the PowerShell installer or WSL2. The related Microsoft tracker PR #49126 was closed unmerged and does not supersede this issue.

## Best Possible Solution

Keep this issue open until maintainers either implement explicit MSYS2/MINGW install.sh support plus fish PATH persistence/warning/docs/tests, or make a documented product decision that Windows Unix-like environments should use WSL2 or install.ps1 instead.

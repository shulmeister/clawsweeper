## Summary

Keep open. Current main still does not implement the requested Windows 11 MSYS2 install.sh support or fish-specific PATH guidance. The Unix installer still only recognizes macOS, Linux, or WSL, docs still route Windows users to install.ps1 or WSL2, and the related Microsoft tracker PR #49126 was closed unmerged rather than shipping this support.

## Best Possible Solution

Keep #18985 open until maintainers either implement explicit MSYS2/MINGW install.sh support plus fish PATH persistence/warning/docs/tests, or make a documented product decision that Windows Unix-like environments should use WSL2 or install.ps1 instead.

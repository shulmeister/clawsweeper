## Summary

Keep open. Current `main` still rewrites `google/gemini-3.1-pro` to preview IDs and still sources Gemini 3.1 Pro forward-compat metadata from the older `gemini-3-pro-preview` template, so the stale preview alias behavior described in #42854 is still present rather than already fixed.

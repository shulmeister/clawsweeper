## Summary

Current main does not implement the requested wildcard candidate behavior. Deep dreaming still ranks only candidates that pass the existing hard gates and score threshold; there is no `wildcardRatio` option, random sub-threshold sampling, or equivalent fallback path. The issue is recent, clear, non-protected, and describes core memory-core behavior rather than a ClawHub-only plugin idea, so it should remain open.

## Summary

Current `main` already suppresses transcript-only `openclaw` assistant messages, including `model="gateway-injected"`, in the embedded subscribe handlers. The implementation is broader than this PR because it also suppresses `delivery-mirror` messages and short-circuits all three assistant entry points on `main`.

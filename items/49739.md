## Summary

Current main already addresses the reported Windows post-update scheduled-task failure through a broader fix set: service install/restart now preserves stable dist entrypoints across package-manager updates and the Windows restart handoff falls back to launching the installed gateway script directly. This PR’s install-time dist-directory guard is both narrower than the real update/restart path and superseded by code already on main.

## Summary

Keep open. Current main still lacks the requested general, plugin-visible early inbound hook before `requireMention` gating. The reporter’s public `@dashhuang/conversation-archive` plugin covers standard-mode messages that reach official hooks, but the full-fidelity gap remains because several channel ingress paths still return/drop before the shared `message_received` hook runs.

## Evidence

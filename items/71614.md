## Summary

Current main already covers the observable runtime-dependency problems: bundled installs use an OpenClaw-managed npm cache under the install execution root, and doctor/runtime-dep scans respect explicit plugin/channel disablement with tests. The PR's channel-disable hunk would also conflict with the existing default-enabled-channel guard called out in review.

## Summary

Keep open. Current main still ships a Node-hosted Gateway with browser-facing clients, not a browser/WASM runtime for core gateway logic. The request is coherent, under 60 days old, has no protected maintainer signal requiring special closure handling, and is not safely closable as ClawHub work because it asks for core runtime portability rather than an optional plugin/skill. Related #11676 is useful no-Node portability context, but it is broader and was itself kept open rather than serving as a clear canonical duplicate for this browser/WASM proposal.

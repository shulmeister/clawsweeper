## Summary

Keep open. The PR is the current canonical follow-up to closed predecessors #71655 and #71674, and current main still lacks several claimed fixes: isolated heartbeat routing can still derive a forced live lane as the heartbeat base, Control UI avatar override paths still bypass the blob-revoking helpers and lack the local-override refresh guard, the running-without-stream reading indicator is absent, and the public-surface loader still has no module-resolution-only require-to-jiti fallback.

## Summary

Keep open. Current main partially addresses auditability with cron run logs and background task records, plus completion/failure delivery paths, but the requested event-driven cron lifecycle integration is still missing: there is no `cron.webhooks` on-start/on-complete config, no cron lifecycle typed plugin hook, and no telemetry event surface matching the issue acceptance criteria. The related PR #8855 was closed unmerged, so it does not supersede the request.

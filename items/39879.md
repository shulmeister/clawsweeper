## Summary

Current main still does not implement WhatsApp native outbound @mentions. The bundled WhatsApp send path builds text/media payloads without Baileys outbound mention metadata, the typed send options cannot carry mention JIDs, current tests still assert plain payloads, and the related implementation PR #56863 is open/unmerged in the provided GitHub context.

## Best Possible Solution

Keep #39879 open until the bundled WhatsApp plugin ships native outbound mention support. A robust fix should add a typed outbound mention path, extract or accept @+<digits>/@<digits> safely, resolve group/LID participant mappings where possible, pass Baileys mentions for text and supported media captions, handle the separate visible-text send used for voice notes, and add focused unit coverage. PR #56863 can remain the implementation vehicle if maintainers choose to revive or merge it.

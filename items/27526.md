## Summary

Keep #27526 open. Current main has partial session listing/search/labels/overrides, message-level pinning, checkpoint branching, and low-level sub-agent/ACP orchestration, but it still does not implement the requested WebChat conversation directory with folders/tags/archive/session pinning/recommendations or a WebChat multi-AI collaboration/committee UX. The issue also has explicit maintainer signal keeping it as the canonical tracker after duplicate #27603 was consolidated here, and the related #52307 REST session-listing PR was closed unmerged rather than superseding the request.

## Best Possible Solution

Continue using #27526 as the canonical maintainer/product tracker. The best path is to scope and implement the missing WebChat/Control UI product surface incrementally: additive session metadata for folders/tags/archive/session pinning and search/recommendation support, then an explicit multi-model/role collaboration UI and protocol for comparison, consensus, agenda, minutes, and project-task extraction. Narrow API work such as REST session listing can be tracked separately, but it would not close this broader request by itself.

## Summary

Keep #27526 open. Current main has session listing/search/labels, per-session overrides, session branching/checkpoints, multi-agent routing, and sub-agent orchestration foundations, but it still does not implement the requested WebChat conversation directory, folder/tag/archive/pin management, side-by-side multi-model collaboration UI, roles, consensus/conflict handling, agendas/minutes, committees, or project tracking. The issue also has explicit maintainer discussion treating it as the canonical tracker after #27603 was consolidated here.

## Best Possible Solution

Keep #27526 open as the canonical product tracker. The best path is to split it into concrete milestones: extend session protocol/storage with folder, tag, archive, persistent pin, richer search, and recommendation metadata; add WebChat/Control UI directory navigation; then design the multi-model collaboration UX and contracts for roles, side-by-side comparison, consensus/conflict handling, agendas/minutes, committee templates, and task/project tracking. Existing labels/search, parent sessions, checkpoints, multi-agent routing, and sub-agents should be treated as foundations, not completion.

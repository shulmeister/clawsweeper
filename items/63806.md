## Summary

Keep this PR open. Current main still has the exec workdir bug the PR targets: gateway/local exec passes the requested workdir into `resolveWorkdir`, and `resolveWorkdir` still stats the raw string, does not expand `~`, and silently falls back to cwd/home with a warning. The PR is not obsolete or implemented on main, though its changelog hunk needs cleanup before merge.

## Best Possible Solution

Keep the PR open as a valid implementation candidate. The best path is to review or update it so the source change expands gateway exec `workdir` tilde paths using the OS home, invalid gateway workdirs fail loudly, and focused regression tests cover valid tilde, missing path, and file-not-directory cases, while removing the unrelated changelog rewrites before merge.

## Summary

Keep open. Current main does not implement `openclaw migrate export` / `openclaw migrate import`; it exposes `backup create` and `backup verify`, and the migration guide still documents a manual tar/copy/doctor workflow. The linked PR #53520 is relevant and directly attempts this feature, but it is still open/unmerged in the provided GitHub context, so the original issue remains a valid tracker.

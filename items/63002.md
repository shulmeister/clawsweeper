## Summary

The Debian 13 2026.4.8 gateway CPU/stall report is a duplicate of the broader open 2026.4.8 gateway CPU/RPC stall tracker #63125. A newer open current-release Linux gateway CPU-spin/status-timeout report (#72338) also covers the same remaining product area. Current main has partial mitigations, but no source/history evidence that this CPU/stall family is fully fixed, so the best maintainer cleanup is to centralize diagnostics in the open canonical reports rather than keep this lower-detail duplicate open.

## Best Possible Solution

Close this duplicate and keep #63125 as the canonical 2026.4.8 gateway CPU/RPC-stall regression tracker. Use #72338 for current-release Linux gateway CPU-spin/status-timeout diagnostics, and ask affected Debian users to add profiles, logs, runtime status snapshots, Node version, configured channels/plugins, and workload details there.

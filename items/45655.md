## Summary

Keep open. Current main has separate image/video generation plugin surfaces, but it has not implemented the requested guard or clarification for Poe media-output models registered through `models.providers`: custom model config still has no output modality, inline provider models are still treated as LLM stream candidates, and there is no bundled Poe media-generation provider to route `poe/nano-banana-2`, `poe/veo-3.1`, or `poe/sora-2-pro` through the media runtime.

## Evidence

That requested behavior is not evident in current main.

## Summary

Keep this PR open. Current main has the later strict-agentic update_plan retry/blocking work, but the PR’s central dispatch-level stall fail-close behavior and memory session-delta retry/backoff are still absent. Several other hunks are also still not implemented or intentionally different, and the PR has unresolved review findings that need author or maintainer handling before landing.

## Best Possible Solution

Keep this PR open, but ask for a rebase and split/narrowing. The best path is to extract smaller reviewed fixes for the missing pieces: dispatch-level fail-closed handling for plan/control progress stalls, memory session-delta retry/backoff that preserves pending work, and any memory search fallback that preserves current hybrid/vector semantics. Treat the strict-agentic work already on main as the baseline and resolve the existing review findings before any landing decision.

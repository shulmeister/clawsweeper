## Summary

Current `main` already handles the reported WhatsApp voice-note path: inbound WhatsApp audio defaults to `audio/ogg; codecs=opus`, saved media keeps a MIME-derived extension, and the media-understanding CLI path now transcodes non-`.wav` audio to mono 16 kHz PCM WAV before invoking `whisper-cli`. Regression tests cover both WhatsApp OGG/Opus MIME handling and OGG-to-WAV transcoding.

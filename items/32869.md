## Summary

Close as ClawHub scope. The request is for a Chinese translation/catalog of OpenClaw skills; the linked repository already exists as an external Chinese skill catalog/translation, while current main explicitly routes new skills, skill discovery, and promotion through ClawHub and keeps foreign-language docs outside this source repo. ([github.com](https://github.com/clawdbot-ai/awesome-openclaw-skills-zh))

## Best Possible Solution

Keep this out of OpenClaw core. The Chinese skill catalog should stay in the linked external repository and, where installable skills are involved, be published or synced through ClawHub so users can discover/install them with the existing skills CLI. Core should only change if a concrete missing ClawHub or skills API is identified.

## Summary

Keep open. Current main still does not implement Slack App Home support: there is no app_home_opened handler, no views.publish path, and the Slack manifest/docs still omit app_home_opened from subscribed bot events. The related PRs in the provided GitHub context were closed unmerged, so they are useful prior discussion but not a shipped fix.

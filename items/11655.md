## Summary

Keep #11655 open. Current main still lacks Slack App Home support: the Slack manifest/docs only enable the App Home messages tab, the bot event subscriptions omit `app_home_opened`, the Slack monitor event registry has no App Home handler, and there is no `views.publish`, Home Tab config, or Home Tab action/tool surface. Related PRs #11699, #11716, and #23268 are useful prior attempts but were closed unmerged, and duplicate report #52020 already points back to #11655 as the canonical tracker.

## Best Possible Solution

Keep #11655 as the canonical tracker. Implement the feature in the bundled Slack plugin by subscribing to `app_home_opened`, registering an App Home event handler under `extensions/slack/src/monitor/events`, publishing a safe default Block Kit Home view with `app.client.views.publish`, updating setup/docs for existing Slack apps, and adding focused tests for event filtering, publish payloads, and any chosen config/tool customization surface.

## Summary

Keep #11655 open. Current main still does not implement Slack App Home support: the Slack plugin has no `app_home_opened` handler, no `views.publish` path, no Home Tab configuration/tool action, and the generated setup/docs still only enable the App Home messages tab. Prior attempts in PRs #11699, #11716, and #23268 are useful discussion, but they were closed unmerged and did not ship.

## Best Possible Solution

Keep this issue as the canonical tracker. The best implementation belongs in the bundled Slack plugin: add `app_home_opened` to Slack setup/docs, register an App Home event handler under `extensions/slack/src/monitor/events`, call `app.client.views.publish` with a safe default Block Kit view, and add focused tests for event filtering, publisher payloads, and any config/tool-action customization path maintainers choose.

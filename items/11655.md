## Summary

Keep #11655 open. Current main still does not implement Slack App Home support: the bundled Slack plugin enables the App Home messages tab, but it does not subscribe to `app_home_opened`, register a Home event handler, call `views.publish`, expose a Home-tab action/config surface, or document/test the behavior. The closed related PRs were not merged, and duplicate #52020 was closed back to #11655 as the canonical tracker.

## Best Possible Solution

Keep #11655 open as the canonical bundled Slack App Home tracker. Implement this in `extensions/slack` by adding `app_home_opened` to setup/docs, registering a monitor handler under `extensions/slack/src/monitor/events`, publishing a safe default Home view through Slack `views.publish`, deciding the supported customization path, and adding focused tests for event filtering, payload shape, error handling, manifest/docs coverage, and config/tool behavior.

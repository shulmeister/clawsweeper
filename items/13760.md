## Summary

Current `main` already implements the requested billing/quota circuit-breaker behavior: billing failures are persisted as profile disables with long backoff, fallback skips same-provider models while the disable is active, and tests/docs cover the behavior. The tagged release commit `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5` also contains this implementation.

## Summary

Keep #61740 open. Current main has related skill infrastructure, but the requested runtime agent-side discovery/semantic matching capability is still meaningfully missing: skills are still surfaced through a prompt snapshot/catalog, ClawHub search is CLI/library-facing, and no built-in `skill_search`-style runtime tool or structured skill capability metadata exists.

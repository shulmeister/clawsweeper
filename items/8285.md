## Summary

Keep #8285 open. The request is concrete and still appears unsatisfied on current main: OpenClaw has reaction/status acknowledgment controls, but no configurable pre-processing text acknowledgment. The current plugin hooks can observe, rewrite, claim, or short-circuit dispatch, but I did not find a stable hook that sends an interim text reply and then continues normal agent processing across Slack, WhatsApp, webchat, and Feishu.

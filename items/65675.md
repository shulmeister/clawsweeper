## Summary

Keep open. Current `main` still enforces a 2 MB local avatar limit and still drops oversized local avatars to a not-found outcome, but the user-facing docs on `main` do not document that limit. The PR’s core docs request is therefore still relevant, even though one of its target doc files has moved on current `main`.

## Summary

Keep open. Current main still audits the post-substitution config for `openclaw secrets audit`, so an authored `"${OPENAI_API_KEY}"` can become the resolved secret value before `collectConfigSecrets()` runs. The collector then treats non-ref, non-empty secret target strings as `PLAINTEXT_FOUND`. The linked #61207 was closed unmerged and was scoped to `.env` scanning, while the reported example is `openclaw.json`; related #62438 remains a similar but separate `security audit` issue.

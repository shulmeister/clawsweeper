## Summary

Keep open. Current `main` still renders Discord cross-context markers through the shared presentation path into Discord components v2, and the proposed `useComponentsV2` config does not exist on `main`. The request is still coherent, but the PR is stale against newer presentation plumbing and would need rework rather than closure.

## Summary

Keep open. Current main still has the native Discord empty-response fallback that sends "✅ Done." when a slash command produces no final/block/tool replies, and the only explicit bypass added in the Discord handler is for `/status`. That is not strong evidence that the broader report covering `/help`, `/acp`, `/model`, and other native commands is fixed.

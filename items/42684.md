## Summary

Current `main` already contains the WebChat history hardening needed for this report: gateway-side history truncation and byte caps, placeholder replacement for oversized entries, and UI-side message windowing. The changelog shows this shipped in `v2026.2.17`, so issue #42684 is obsolete on current `main`.

## Close Comment

Current main explicitly keeps optional capability in plugins (`VISION.md:54`), treats memory as a plugin slot (`VISION.md:63`), and uses ClawHub as the canonical discovery surface for community plugins (`docs/plugins/community.md:14`). The repo already supports external/community plugins plus alternate memory plugins (`docs/tools/plugin.md:11`, `docs/tools/plugin.md:109`). If OC-Argent is packaged and published through that path, it matches current project scope better than tracking it as an OpenClaw core issue.

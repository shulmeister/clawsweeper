## Summary

Current main still does not implement the requested prompt replacement hook result. The hook types, merger, prompt-build helper, runner, and public docs only support system prompt/context mutations, while the directly related implementation PR #8178 was closed unmerged. The April 26 maintainer/Codex review comment also correctly identified this as still open work.

## Best Possible Solution

Keep #8176 open until OpenClaw either implements and documents a prompt replacement field for before_prompt_build and legacy before_agent_start, with focused tests for merge order and runner submission behavior, or a maintainer explicitly rejects that API. The likely code path is the public hook result type/field list, hook merger, prompt-build helper, embedded runner prompt assembly before prependContext, docs, and regression tests.

## Summary

Keep open. Current main does not implement the requested before_agent_start promptOverride capability: the hook result types, hook merger, prompt-build helper, runner application path, and public hook docs all expose/apply context and system-prompt fields but not replacement of the current user prompt. The related #8178 PR was closed unmerged, and the later linked systemPrompt work only addresses system prompt handling, not user prompt override.

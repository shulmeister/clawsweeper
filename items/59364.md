## Summary

Keep open. Current `main` and the shipped `v2026.4.23` tag still lack the PR's generic source-vs-dist subpath export parity check, so this is not implemented on main. The discussion also does not support a `cannot_reproduce` close: one reviewer disputed the specific `v2026.4.1` motivating regression from the npm tarball, but other review comments found real correctness gaps in the PR itself, which makes this look like useful but unfinished work rather than a proven obsolete item.

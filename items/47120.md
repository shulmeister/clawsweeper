## Summary

Keep open. Current main has backend/session-tool support for last-message previews, but the Control UI Sessions list still does not request or render timestamp-plus-preview text. The open PR #47243 is relevant prior work for this exact issue, while #69845 was closed unmerged, so the user-facing request remains unlanded on main.

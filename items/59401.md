## Summary

Keep #59401 open. The request is concrete, recent, author is not a maintainer, has no protected labels, and current main still appears to treat Feishu @all (`@_all`) as satisfying mention gating while the Feishu config schema does not expose `ignoreAtAll` or `ignoreBroadcast`. The linked PR #59422 is still open, so this is active feature work rather than cleanup-close material.

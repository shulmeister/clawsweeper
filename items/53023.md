## Summary

Keep open. Current `main` still hardcodes per-session lane concurrency to 1, only exposes global lane concurrency config, and does not expose config for automatic yielding or lane-wait warning thresholds. Some issue text is outdated because queue steering/interrupt modes and `sessions_yield` now exist, but the requested configurability is not implemented.

## Summary

Keep open. Current `main` does not implement a Slack `groupPolicy: "private"` mode: the shared `GroupPolicy` type still only allows `"open" | "disabled" | "allowlist"`, Slack docs/config comments list only those three values, and Slack routing/tests still treat `"open"` as allowing unlisted public channels.

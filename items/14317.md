## Summary

Keep open. Current main does not implement Slack `groupPolicy: "private"`: the shared config type/schema only accepts `open | disabled | allowlist`, Slack channel gating only evaluates those three policies, and the Slack docs still list only those three options. The requested behavior remains a concrete Slack plugin capability, not a ClawHub-only item.

## Summary

Keep open. Current `main` still does not support recursive `skills.load.extraDirs` discovery or glob-configured extra dirs for a plugin-root layout like `plugins/<name>/skills/<name>/SKILL.md`. The loader only supports direct skill dirs, immediate `<root>/*/SKILL.md`, plus a narrow special case for `<dir>/skills/*/SKILL.md`.

## Evidence

- **Config contract is still plain directory list:** `SkillsLoadConfig.extraDirs` remains `string[]`, documented as additional skill folders where each directory should contain skill subfolders with `SKILL.md`; there is no recursive/glob option in the type.

  - file: [src/agents/skills/workspace.ts:317](https://github.com/openclaw/openclaw/blob/a6aa151653ab478e9aa23e95c6446c41adc4fce1/src/agents/skills/workspace.ts#L317)

## Summary

Keep #19482 open. Current main still recreates HEARTBEAT.md for existing workspaces that intentionally removed it, while the heartbeat runtime, tests, and docs preserve a real semantic difference between a missing file and an empty template file. The PR remains a focused live fix, though its negative regression fixture should be tightened as the review comment notes.

## Best Possible Solution

Keep #19482 open and land it after tightening the negative regression fixture, or apply the same code change directly on main: only seed HEARTBEAT.md for brand-new workspaces while preserving missing-file heartbeat semantics for existing workspaces.

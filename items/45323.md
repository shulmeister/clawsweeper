## Summary

Keep open. Current main has a reusable `agents.list` RPC with identity metadata and an implemented slash-command autocomplete path, but it does not implement the requested Slack-style `@agent` autocomplete, rich mention chips in the composer, or mention-chip rendering in chat history. This is also not a ClawHub cleanup candidate: the request targets core Control UI chat UX and Gateway/UI contract behavior rather than an optional plugin, skill, channel, or provider integration.

## Summary

Keep open. Current main mitigates part of the report by making model-pricing fire-and-forget and by binding the HTTP/WS gateway before sidecars, but the reporter-localized Gmail watcher path is still awaited inside startup sidecars and can still hold readiness/startup-gated methods while it waits on Tailscale setup or the 120s `gog watch start` timeout.

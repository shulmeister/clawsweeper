## Summary

Keep #68309 open. Current main still has the reported OpenAI-completions stream bug: `choice.delta.content` is appended as text without runtime string/array handling, while Mistral Small latest remains a reasoning-enabled bundled model. The linked PRs #68418 and #68855 are useful evidence and already target this issue, but they are open/unmerged in the provided context, so the bug is not implemented on main or shipped yet.

## Best Possible Solution

Keep #68309 open as the canonical user bug report until a complete fix lands. The best code path is #68418 or an equivalent parser-level fix that unpacks array-shaped `delta.content` into thinking/text deltas, preserves string behavior, avoids `[object Object]` coercion, and adds a regression test in `src/agents/openai-transport-stream.test.ts`.

## Summary

Keep open. Current main has token/cost visibility already, but it does not implement the requested optimization-specific dashboard, does not ship optimization tiers that tune compaction/pruning/memory search, and does not make local memory search the default. This also does not cleanly fit a `clawhub` close: the request is for native Gateway UI and core default behavior, which remains in OpenClaw scope.

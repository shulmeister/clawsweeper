## Summary

Keep open. Current `main` still does not inject subagent identity/session env vars into spawned command sessions; the request in #11172 is not implemented on `main`.

  - file: [src/agents/tools/sessions-spawn-tool.ts:186](https://github.com/openclaw/openclaw/blob/86f69ba5a06ae256906c3a26308801e0b84e0091/src/agents/tools/sessions-spawn-tool.ts#L186)

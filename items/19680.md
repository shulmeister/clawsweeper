## Summary

Keep #19680 open. Current main still has useful scope guidance in VISION.md plus operational PR/Codex review instructions, but it has not implemented the requested VISION.md restructure into a human/AI PR review decision framework. The same-day steipete Codex review comment also explicitly keeps this issue open, and I found no current-main implementation or canonical duplicate that supersedes it.

## Best Possible Solution

Keep the issue open for explicit maintainer docs/product judgment. A good implementation would update VISION.md with a concise product thesis, three or four PR gut-check questions, rationale-grounded examples for security, memory, lean core, plugins, and won't-merge categories, and clear language that helps both human contributors and AI review tools reason from principles.

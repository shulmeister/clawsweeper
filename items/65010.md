## Summary

Keep open. Current `main` and the latest release still use `isStreaming` as the steer/injection gate, so the reported tool-execution window is not implemented away. The PR also has a credible unresolved regression risk for Codex app-server teardown when a backend has no stop-state contract, so conservative cleanup should not auto-close it.

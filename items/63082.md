## Summary

Keep open. Current main still hardcodes shell completion profile paths and lacks regression coverage for ZDOTDIR/XDG_CONFIG_HOME, while this is the active same-author implementation candidate linked to open bug report #63069.

## Best Possible Solution

Keep the PR open as the implementation candidate for #63069. The maintainer path is to adapt it to current main by moving a unified, env-aware profile resolver into src/cli/completion-runtime.ts, using it for install/check/doctor messages, preserving consistent bash fallback semantics, and adding focused tests for zsh ZDOTDIR, fish XDG_CONFIG_HOME, and bash .bashrc/.bash_profile selection.

## Summary

Keep open. This is a recent, active enhancement request with corroborating comments and related issues, and current published OpenClaw docs still describe `plugins.slots.memory` as a single exclusive memory plugin slot rather than the proposed purpose-specific multi-slot architecture. No conservative close reason applies.

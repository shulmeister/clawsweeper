## Summary

Keep open. Current main does not implement Feishu calendar support, and the existing Feishu plugin still only registers doc/chat/wiki/drive/perm/bitable tool surfaces. The repo vision does push optional integrations toward plugins/ClawHub, but that guidance is not strong enough to confidently close a feature request against an already bundled, community-maintained Feishu plugin.

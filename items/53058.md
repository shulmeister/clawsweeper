## Summary

Keep open. Current `main` still does not implement the issue's main docs ask on the tools page: `docs/tools/index.md` lists `nodes` and `cron`/`gateway` in the built-in tools table without an owner-only section or setup guidance, and only later calls out `gateway` as owner-only. The underlying behavior is still present in code (`cron`, `gateway`, and `nodes` remain owner-only). `commands.ownerAllowFrom` is now documented elsewhere, so the issue is partially addressed, but not fully implemented on the requested docs surface.

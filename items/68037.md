## Summary

Keep open. Current `main` still normalizes HTTP MCP `tools/call` results down to `{ type, text }`, which drops required fields for non-text MCP content blocks. The latest release still has the same code path, the report is concrete and recent, and there is already an active fix PR (#68046), so this is not conservative-close material.

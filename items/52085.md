## Summary

Current main already contains the operator-scope compatibility and gateway auth coverage that address this report: operator-admin now satisfies operator.read checks, direct-local gateway-client shared-auth sessions retain their requested scopes, and there is an integration test for operator.read connect when the paired device baseline is operator.admin.

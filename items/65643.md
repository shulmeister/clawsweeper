## Summary

Keep open: current `main` at `c150110e02550eaf6c56db0c851dc62fc0ca88e1` still matches the reported bug. `runCli()`/`entry.ts` apply `--profile`, but `applyCliProfileEnv()` only sets `OPENCLAW_PROFILE` and related defaults; it never clears `OPENCLAW_LAUNCHD_LABEL`. `resolveLaunchAgentLabel()` still gives `OPENCLAW_LAUNCHD_LABEL` first priority, so an inherited gateway label can still override the requested profile.

## Summary

Current main only partially implements #30389. WebChat has an attachment button and a per-session model picker, but user uploads remain image-only and there is no chat-composer skill/capability menu for finance/coding/writing/image/reasoning-style routing.

## Best Possible Solution

Keep #30389 open as the umbrella WebChat UX request unless maintainers split it into narrower canonical issues. The remaining product work is to add non-image document/file upload across the picker, validation, chat.send serialization, gateway/model attachment handling, and docs, and to define whether a chat-composer skill/capability picker should route by tools, skills, agents, or model/task profiles. The shipped attachment button and model picker should be treated as partial progress, not a close condition.

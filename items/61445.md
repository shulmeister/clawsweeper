## Summary

Keep open. Current main does not implement a startup rowid fence, persisted echo cache, or startup grace filter for iMessage self-chat echoes. The self-chat path still dispatches an `is_from_me=true` message when the echo cache is empty or misses, which matches the reported restart-warmed-cache failure mode.

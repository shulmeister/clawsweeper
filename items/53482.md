## Summary

Current `main` still ignores legacy `CLAWDBOT_*` env names without a dedicated gateway deprecation warning or doctor guidance. The breaking removal is documented in the changelog, the current gateway error paths mention only `OPENCLAW_*`, and the proposed warning implementation exists only in still-open PRs (#53626, #53667), not on `main`.

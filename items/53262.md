## Summary

Keep open. Current `main` does not implement this PR’s schema change: `sessions_spawn.agentId` is still optional, the ACP path still intentionally falls back to `acp.defaultAgent`, docs/prompts still tell users omission is allowed in that case, and tests explicitly preserve omitted-`agentId` behavior unless `subagents.requireAgentId` is configured.

## Summary

Keep open. Current main localizes the general Control UI shell into zh-CN, but the Settings/Config form still gets section labels, field labels, and help text from English-only metadata and schema hints. The requested Chinese localization of the Settings configuration items remains missing and is core UI work, not a ClawHub plugin fit.

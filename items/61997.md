## Summary

Current main implements the observable LM Studio/local-model timeout path: the session-memory slug generator now honors `agents.defaults.timeoutSeconds`, and the bundled LM Studio provider/onboarding path defaults to OpenAI-compatible chat completions with native LM Studio discovery/preload. The related #65492 comment was a useful pointer, but that PR closed unmerged and the current main behavior covers the user-facing timeout knob through the general timeout setting.

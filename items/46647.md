## Summary

Current main still has the core prompt bug reported in #46647. `buildAgentSystemPrompt` only treats `inlineButtons` as enabling inline-button guidance, while Slack's directive pipeline accepts object-form `{ interactiveReplies: true }`; that object form is not converted into runtime capabilities, and Slack's message-tool capability discovery adds only `presentation`, not `interactiveReplies` or `inlineButtons`. The result can still produce contradictory Slack guidance rather than the requested "Inline buttons supported" behavior. The related click-routing work is separately tracked by #46683, but the original prompt-gating issue itself is not implemented on main.

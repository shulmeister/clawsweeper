## Summary

Keep open. Current `main` still uses generic `openclaw mcp` wording, so the mcporter-vs-`mcp.servers` confusion reported in #65209 remains. Docs explain the split more clearly, but the CLI/help/test changes described in closed PR #69726 are not present at `ae5c6573670b3742b156899bd2b9c36b8677f191`.

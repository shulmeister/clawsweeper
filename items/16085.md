Keep #16085 open. Current main still has the bundled Signal plugin wired to native signal-cli JSON-RPC plus SSE only, while this PR is the active implementation candidate for bbernhard/signal-cli-rest-api REST/WebSocket container support after #10240 was closed as tracked by this PR.

## Best Possible Solution

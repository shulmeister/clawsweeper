## Summary

Current `main` already keeps plugin-provided browser tools in the agent tool list, independent of the core `coding` profile, and ships docs that point the remaining 'browser missing' case to `plugins.allow` excluding `browser`. The same agent-side code and regression test are already present in release `v2026.4.23`.

## Summary

Keep #62453 open. The reported local `agent --local --json` stdout/stderr bug is still present on current `main`: the local delivery path still emits the JSON envelope through `runtime.log(JSON.stringify(...))`, while JSON mode routes console/log output to stderr. The PR is useful, but its current patch has unresolved review feedback about an unsafe `OutputRuntimeEnv` cast; `writeRuntimeJson` is the established helper and the gateway JSON path already uses it.

## Summary

Current `main` already fixes this. `openclaw models list` resolves configured rows through the runtime model resolver, and the `openai-codex` provider now overrides `gpt-5.4` to a `1_050_000` context window instead of the stale ~266k metadata seen in older releases.

## Evidence

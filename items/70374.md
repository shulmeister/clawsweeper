## Summary

Keep open. Current main has gateway execution-mode support for generic plugin actions, but `message.send` and `message.poll` still route through `executeSendAction`/`executePollAction`, whose plugin-owned path directly calls local plugin dispatch. Telegram on main also does not opt `send` into gateway execution, so the PR’s requested behavior is not implemented on main.

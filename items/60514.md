## Summary

Keep open. Current main has adjacent implementation: plugin `session_end`/`session_start` hooks, opt-in `memory-core` dreaming, forked subagent transcript context, auto-compaction, and cache-TTL pruning. It does not fully satisfy the issue as filed because the user-requested internal `session:end` hook is still absent, `/stop` still emits only the existing `command:stop` path, and the requested stop/disconnect/timeout-style lifecycle plus exact autoDream trigger semantics are not source-backed on current main.

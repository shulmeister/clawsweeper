## Summary

Keep open. Current `main` still swallows `resolveWorkflowFile` errors in the Lobster runner, so workflow-like missing paths still fall back to inline pipeline parsing instead of surfacing the file error. The PR remains relevant, and the provided review history shows its follow-up commit addressed the main regression concern about slash-containing inline commands.

## Evidence

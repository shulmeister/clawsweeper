## Summary

Close #60446 as a duplicate of the older open #39031, which tracks the same remaining iMessage tapback-to-agent feature. Current main does not implement the requested imsg reaction event path; related #60274/#60677 cover filtering/dropping tapbacks only and explicitly leave #60446/#39031 as follow-up feature work.

## Evidence

## Summary

Keep #14619 open. Current main still builds a human-readable `## Tooling` list from active runtime tools, still passes the same tools as callable schemas, and still has no scoped `skipToolListText` or equivalent config. The existing `systemPromptOverride` and `promptMode` alternatives are broader than the requested tool-list-only omission.

## Best Possible Solution

Keep #14619 open until OpenClaw adds a scoped agent/default option, for example `agents.defaults.skipToolListText`, that suppresses only the human-readable Tooling list while preserving callable tool schemas. The implementation should update prompt construction, run plumbing, config types/zod/generated schema/help/docs, `/context` accounting, and focused tests proving the prompt text is omitted while tools remain available.

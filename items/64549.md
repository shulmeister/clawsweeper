## Summary

Current main implements the requested auto-index path for builtin memory: the memory-core watcher now watches concrete `workspace/memory` and `memorySearch.extraPaths` directory roots, filters file types, and schedules a watch-triggered sync that re-lists those roots recursively before indexing. The linked #64654 PR was closed unmerged, but the later current-main implementation addresses the same watcher/glob failure mode reported here.

## Best Possible Solution

Close as implemented on current main. Keep the concrete-root chokidar watcher and regression tests; users on 2026.4.12 or v2026.4.24 need a build containing the current main change. The separate #71335 question about gateway-mode watch defaults should remain tracked separately.

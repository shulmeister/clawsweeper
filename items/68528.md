## Summary

Keep PR #68528 open. Current main still defaults `resolveChannelGroupRequireMention` to `true` when no explicit group or wildcard `requireMention` is configured, and Telegram delegates to that core helper. The linked issue #68476 describes the exact remaining user problem for Telegram groups, while the PR discussion and Greptile review support the proposed direction with only minor nits noted.

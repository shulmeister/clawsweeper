## Summary

Current `main` does not show a landed fix for #42644. The global hook runner is still reinitialized whenever `loadOpenClawPlugins()` serves an activating registry from cache, and outbound delivery still snapshots `hasMessageSendingHooks` once before sending. The latest release and current `main` are identical in the relevant hook-runner and outbound-delivery files, so there is not strong evidence to close this as implemented or obsolete.

## Summary

Current main already surfaces the requested `/status` visibility: heartbeat cadence appears as a dedicated Heartbeat overview row, and the configured/default model plus context window appears in the Sessions overview. The related PR #7671 proposed a more explicit Model row but was closed unmerged; the current shipped output still solves the user-facing need without requiring config-file inspection.

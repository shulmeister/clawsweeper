## Summary

Keep open. Current main has the plugin seam needed for a `before_tool_call` trust/discovery layer, but there is no landed implementation of `/.well-known/agent.json` discovery, trust-registry verification, or revoked-operator blocking in core. The request is coherent, recent, and there is related open follow-up work in the provided GitHub timeline, so evidence is not strong enough for a conservative close.

## Summary

Current `main` still contains the MiniMax-specific behavior that disables thinking by default on the Anthropic-compatible path and then retries reasoning-constraint failures with `minimal`, so the reported direct-fail -> fallback/retry pattern is not clearly fixed on `main`. This issue should stay open.

Current main still maps `minimal` and `low` to `low` for reasoning-enabled adaptive Claude Opus 4.7 requests, then sends that value as `output_config.effort`; the GitHub Copilot wrapper does not rewrite it. The paired same-author bug #70322 remains open with concrete Copilot rejection logs, so this PR is still a live implementation candidate rather than cleanup.

## Summary

Current `main` already contains a targeted recovery for stale persisted ACP session ids, plus regression coverage for the exact `Persistent ACP session ... could not be resumed: Resource not found` failure and stale bound-session cleanup. The same logic is present in the shipped `v2026.4.23` release tag.

## Evidence

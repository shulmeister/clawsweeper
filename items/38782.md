## Summary

Current main already fixes the reported `/new` and `/reset` greeting bug by appending a runtime `Current time:` line to the bare session-reset prompt, and both auto-reply and gateway reset flows use that prompt. Regression tests and the changelog indicate this shipped in `v2026.3.13`.

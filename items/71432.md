## Summary

Keep open. Current main has useful partial multi-agent pieces, especially isolated multi-agent routing, A2A `sessions_send`, and channel-specific broadcast groups, but it does not implement the requested shared multi-agent group session where agents observe each other in one visible thread with dispatcher/budget/isolation orchestration. The KinthAI comment is relevant evidence of demand and of an external layer built because those core coordination pieces are still absent.

## Summary

Current `main` still ships a real app/gateway version lock on the macOS local-gateway path, so this is not closeable as fixed or non-reproducible. The report overstates scope because the generic Gateway handshake only hard-fails on protocol-version mismatch and same-origin Control UI already mirrors the server version, but there is enough shipped behavior here to keep the item open as a valid enhancement/request.

## Summary

Keep open. Current main still does not extend Telegram's user-visible dispatch-error fallback to Discord, Slack, Signal, and WhatsApp; those paths still log, rethrow, or return without sending a fallback message when dispatch fails or yields no visible delivery.

## Evidence

## Summary

Keep #46733 open. Current main still intentionally strips the Anthropic 1M context beta from `sk-ant-oat-*` OAuth/setup-token requests, and the docs/tests explicitly describe that fallback. The runtime can resolve 1M tokens when `params.context1m: true` is configured, but the OAuth/header part of the reported bug remains current behavior rather than implemented cleanup.

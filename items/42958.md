## Summary

Current `main` already implements the DashScope usage-tracking path: DashScope-compatible base URLs are recognized as native-compatible endpoints, model compat preserves `supportsUsageInStreaming` for them, the OpenAI-completions transport requests `include_usage` and records usage chunks, and regression tests plus the 2026.3.22 changelog match this exact symptom.

## Summary

Keep open. Current `main` still has the reported Feishu bug: `checkBotMentioned()` returns `true` for any message whose raw content contains `"@_all"`, and group handling drops only when `requireMention && !ctx.mentionedBot`. That means `@_all` still satisfies mention gating for every bot on current `main`, so this PR addresses a real unfixed issue.

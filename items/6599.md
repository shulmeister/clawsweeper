## Summary

Keep open. Current main has fallback runtime support and `openclaw models status --probe` auth probes, but it does not implement the requested `/models test-fallback` slash command or an equivalent command that intentionally skips the primary model and verifies the fallback chain. The related PR #43342 directly attempted this feature, but it was closed unmerged, so it does not supersede the issue.

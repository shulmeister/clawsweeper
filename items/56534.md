## Summary

Current main does not contain a Control UI or gateway path that rewrites numeric chat input into dotted/decimal values. The Control UI sends the trimmed text verbatim, the gateway only applies NFC/control-character sanitization, and current tests assert the raw body and persisted user transcript stay unchanged. The reported dotted values are more consistent with model-generated tool arguments than with a UI/input transformation bug.

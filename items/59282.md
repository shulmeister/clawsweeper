## Summary

Keep open. Current `main` does not contain this fix: `mergeProviderModels` still overwrites explicit `input` with implicit model metadata, and the existing test suite still asserts that old behavior. The PR remains relevant, especially for explicit image-capable provider configs like the documented MiniMax materialization path.

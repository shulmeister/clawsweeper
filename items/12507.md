## Summary

Keep open. The request is concrete, non-protected by author/label rules, and the main branch still lacks the requested skill signature/hash verification and unsigned-skill capability gating. Current code has path, size, and ClawHub version tracking protections, but those do not solve content integrity for SKILL.md at install or runtime load.

## Summary

Keep open. Current `main` has moved provider auth env lookup to the manifest-driven `src/secrets/provider-env-vars.ts` path and documents canonical `qwen`/`alibaba` providers, but there is still no `bailian` provider id or `BAILIAN_API_KEY` support on `main`. That means this PR is not clearly implemented already, and the proposed one-line change targets an older code shape that would need rework rather than straightforward closure.

## Summary

Keep this PR open. Current main still does not ship the proposed Wear OS companion, phone-side Wear proxy, shared Android gateway module, Wear Data Layer integration, tile/complication services, or Wear documentation. The PR is large and the author has said others may pick up the latest working state, but it is concrete Android platform work, not implemented on main, not superseded by another open canonical item, and not a ClawHub-only plugin request.

## Best Possible Solution

Keep this PR open unless maintainers explicitly decide not to ship Wear OS in core. The practical path is to make the work maintainer-owned and split it into reviewable stages: shared Android gateway helpers, phone-side Wear proxy, standalone Wear app module, then tile/complication/notification polish with focused tests and fresh device proof.

## Summary

Keep open. Current `main` still lacks the shared discovery-time `realpathCache` threading and the reverse canonical-path cache entry that this PR proposes, so the reported plugin-discovery startup optimization from #65733 is not implemented on `main` or in the latest release commit provided.

## Evidence

## Summary

Keep open. Current `main` still computes `vectorScore` and `textScore` internally for hybrid memory search but drops them from the returned result object, and the public `MemorySearchResult` types on `main` still do not expose those fields. The latest shipped release in the checkout also lacks the change, so this PR is not implemented on `main` yet.

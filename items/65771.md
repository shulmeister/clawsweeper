## Summary

Keep open. Current main at 631552c554095b0c520376db9131dac0f5427f94 still does not implement the PR's central behavior: failure alerts do not surface the classified timeout cause, CLI JSON output is passed through unchanged, cron run-log entries still lack errorReason, and the gateway run-log schema has no errorReason field.

## Summary

Keep open. Current main does not already implement the PR's routed-reply normalization work, and the PR discussion contains unresolved review feedback that the proposed global threadId-to-replyToId fallback could regress Telegram forum-topic routing. This is a salvageable PR with useful tests and a concrete fix direction, not a conservative cleanup close.

## Summary

Keep #11955 open. Current main has shipped adjacent memory search, optional session transcript indexing, usage/cost/session aggregation, startupContext for bare /new and /reset, and an opt-in BOOT.md startup hook, but the central umbrella still has missing product/API work: an agent-facing self-evaluation metrics API, automatic conversation chaining or compact-summary handoff, and default gateway-restart preload before the first user message. The provided issue discussion also includes a recent maintainer comment explicitly keeping this issue open as the canonical umbrella after duplicate #11956 was closed.

## Best Possible Solution

Keep #11955 open as the canonical umbrella unless a maintainer splits it. Preserve the shipped memory_search, extraPaths, optional sessionMemory, usage/cost/session aggregation, startupContext, and BOOT.md behavior, then track narrower work for an agent-facing self-evaluation metrics API, documented conversation continuation or compact-summary handoff, and explicit gateway-startup preload/prewarm of core workspace context before first user input.

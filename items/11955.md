## Summary

Keep #11955 open. Current main has substantial adjacent implementation for memory search, session-memory indexing, Active Memory, usage/cost summaries, startupContext, and BOOT.md, but the umbrella still has meaningful missing pieces: agent-facing self-evaluation metrics, documented conversation chaining or compact-summary handoff, and gateway restart preloading before the first user message. The provided issue discussion also includes a same-day maintainer/Codex comment explicitly keeping this issue open as the live umbrella.

## Best Possible Solution

Keep #11955 open as the canonical umbrella unless a maintainer splits it. Preserve the shipped memory and usage surfaces, then implement or track the remaining product/API work: agent-facing self-evaluation metrics with task-completion/satisfaction/success-rate semantics, documented conversation continuation or compact-summary handoff, and explicit gateway-startup context preload/prewarm behavior before first user input.

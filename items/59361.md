## Summary

Keep open. The report is recent, non-maintainer-authored, and not protected by labels, but current main still leaves enough plausible surface that this cannot be conservatively closed: `openclaw agent` still requires an explicit session selector, gateway mode still catches gateway-path errors and falls back to embedded execution, the public README still shows a selector-less `openclaw agent --message ...` quick-start example, and the issue also reports `--agent` hangs plus missing plugin CLI command registration that I could not disprove from current main.

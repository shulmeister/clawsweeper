## Summary

PR #60502 should be closed as ClawHub/plugin scope rather than merged into OpenClaw core. Current project direction keeps optional protocol/runtime capabilities out of core when plugins can express them, plugin APIs include HTTP route registration, and an existing external A2A gateway plugin already covers this use case. The PR also still has material unresolved review concerns in its core gateway handler path, so it is not a good core merge candidate as-is.

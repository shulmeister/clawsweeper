## Summary

Keep open. Current `main` still does best-effort sender-name resolution via Feishu contact APIs by default and still uses live contact directory listing paths. Recent changes only suppress stale invalid-scope noise and rewrite grant URLs from `contact:contact.base:readonly` to `contact:user.base:readonly`; they do not remove the underlying dependency requested in #13751.

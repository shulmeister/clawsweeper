## Summary

Keep open. Current main has partial mitigations for Feishu sender-name lookup noise, but it still performs sender-name resolution through the Feishu contact API by default. The requested capability, basic bot operation without default contact-directory/profile lookup dependency, remains meaningfully unresolved.

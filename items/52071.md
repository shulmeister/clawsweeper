## Summary

Current main already covers the reported failure mode: plain `openclaw status --json` no longer performs the gateway probe, local authenticated gateway probes stay device-bound so read/detail RPCs are not scope-limited, and deep security audit probes now resolve local probe auth from the configured local auth path. The shipped `v2026.4.22` tree contains those behaviors.

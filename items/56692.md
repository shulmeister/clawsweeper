## Summary

Current `main` does not implement per-history multi-agent targeting metadata for Telegram group context. The runtime still records and injects prior group messages as generic `sender`/`body`/`timestamp` entries, with only current-turn `WasMentioned` and reply-context signals, so the reported context-blur problem remains plausible and unclosed.

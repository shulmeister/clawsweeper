## Summary

Keep #34986 open. Current main still lacks a configurable heartbeat pre-flight script/command that can return HEARTBEAT_OK and short-circuit agent session creation. The related PR #35504 captured the requested shape but was closed unmerged, and current alternatives only skip in built-in cases or reduce context cost.

## Best Possible Solution

Keep the issue open for maintainer product judgment or implementation. A complete fix would add an optional heartbeat pre-flight hook such as agents.defaults.heartbeat.preFlightCommand, wire it through exported config types, strict Zod schema, labels/help/generated docs, and runHeartbeatOnce before getReplyFromConfig, with tests for HEARTBEAT_OK skip, non-OK output, failure, timeout, per-agent/default merging, and unchanged existing heartbeat behavior.

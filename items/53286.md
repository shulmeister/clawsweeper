## Summary

Current `main` still gates estimated-cost display on auth mode, so pure `aws-sdk`/Bedrock providers do not show configured cost estimates in `/usage full` or `/status`. The request is not implemented on `main`.

## Evidence

- **`/usage full` footer still limits cost display to API-key auth:** In the reply footer path, `showCost` is set only when `resolveModelAuthMode(providerUsed, cfg)` returns `"api-key"`. `resolveModelCostConfig(...)` is skipped otherwise, so `aws-sdk` providers cannot show estimated cost even when model cost is configured.
  - file: [src/auto-reply/reply/agent-runner.ts:1459](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/auto-reply/reply/agent-runner.ts#L1459)
  - command: `nl -ba src/auto-reply/reply/agent-runner.ts | sed -n '1410,1475p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **`/status` still excludes `aws-sdk` from cost display:** The status message path only enables cost when the effective auth mode is `"api-key"` or `"mixed"`; `"aws-sdk"` is not included.
  - file: [src/status/status-message.ts:783](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/status/status-message.ts#L783)
  - command: `nl -ba src/status/status-message.ts | sed -n '780,806p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Bedrock resolves to `aws-sdk` auth on current main:** `resolveModelAuthMode(...)` returns `"aws-sdk"` for an explicit `aws-sdk` provider auth override and for Bedrock aliases, which means the `/usage full` and `/status` gates above exclude Bedrock by default.
  - file: [src/agents/model-auth.ts:620](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/agents/model-auth.ts#L620)
  - command: `nl -ba src/agents/model-auth.ts | sed -n '609,680p'`

- **Current docs still describe the limitation as API-key-only:** The user-facing docs say `/status` estimated cost and `/usage full` estimated cost are API-key-only, matching the current code rather than the requested behavior.
  - file: [docs/reference/api-usage-costs.md:20](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/docs/reference/api-usage-costs.md#L20)

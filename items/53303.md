## Summary

Keep open. Current `main` still gates cost display on auth mode, so non-`api-key` providers with configured pricing do not get the requested cost estimate. The PR also needs rework on the current tree because the status implementation moved, and its stated gating approach is too broad unless it distinguishes explicit user config from catalog/cache pricing.

## Summary

Keep open. Current `main` does not implement Slack RTS support from assistant action tokens: the Slack plugin advertises no `search` action, its action dispatcher has no Slack search branch, its monitor does not register assistant thread events, and there are no matches in the repo for `action_token`, `assistant_thread_started`, `assistant_thread_context_changed`, or `assistant.search.context`.

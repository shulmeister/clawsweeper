## Summary

Keep open. This is a recent, detailed core gateway-host exec report, and current main still has the same relevant behavior: YOLO gateway exec skips script preflight, runs each host exec through a shell child process, and has no visible stale-read/cache-coherence mitigation or regression coverage for the reported macOS/APFS cross-exec read race.

## Best Possible Solution

Keep this issue open for maintainer investigation. The next useful path is a focused macOS/APFS repro comparing v2026.4.19, v2026.4.20, v2026.4.24, and current main under gateway-host exec with security=full and ask=off. If confirmed, the fix likely belongs in the core host-exec/runtime path or in a documented gateway-host filesystem consistency diagnostic, with regression coverage around atomic rename followed by reads from separate exec calls.

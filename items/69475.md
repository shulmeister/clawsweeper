## Summary

Keep #69475 open. Current main still has the four reported bundled skill frontmatter descriptions without the requested WHEN/trigger language, and `skills/gh-issues/SKILL.md` is still a single large file at 885 lines / 5006 words with no `references/` split. The related #69489 fix was closed unmerged, so the observable cleanup has not landed on main.

## Best Possible Solution

Keep the issue open and land a focused maintainer-reviewed cleanup: update the four shipped skill frontmatter descriptions with precise WHAT+WHEN trigger language, including the Contacts and Obsidian-specific refinements noted in the reporter's follow-up, then split `skills/gh-issues/SKILL.md` into a concise orchestrator plus referenced phase/troubleshooting files. If maintainers decide bundled-skill polish should move out of core, record that product decision explicitly rather than auto-closing this as already handled.

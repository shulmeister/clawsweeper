## Summary

Keep open. Current `main` still uses the misleading `--tz` help text in both `cron add` and `cron edit`, while the implementation, tests, and docs already treat `--tz` as applying to offset-less `--at` values. The reporter also noted an active open fix PR (`#69975`) on 2026-04-23, so this is neither implemented on `main` nor stale.

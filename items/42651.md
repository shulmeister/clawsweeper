## Summary

Keep open. Current main has a substantial memory surface, but it does not implement the issue as written: the operator-facing CLI is `openclaw memory ...` with `status/index/search/promote/promote-explain/rem-harness/rem-backfill`, and the agent tool surface is `memory_search` plus `memory_get`. I did not find a first-class `remember`/`recall`/`review`/`edit`/`forget` CLI or bundled memory skill surface on main.

## Summary

Close #68388 as superseded. Its central request, restoring CLI-backed compaction for #68329 and preventing stale external CLI resume state, is now handled by #71916, which explicitly superseded #68388 and was merged on 2026-04-26. The landed replacement takes the cleaner direction of compacting OpenClaw's persisted CLI transcript, clearing external CLI resume bindings after compaction, and reseeding fresh CLI sessions from the compacted OpenClaw transcript instead of writing provider-private Claude JSONL session files.

## Best Possible Solution

Close #68388 and treat merged #71916 as the canonical implementation for #68329. Any remaining CLI compaction regressions should be filed against or followed up from #71916's landed OpenClaw-transcript compaction path, not this older Claude-private JSONL rotation approach.

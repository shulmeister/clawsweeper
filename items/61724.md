## Summary

Keep open. The checked-out main at 4069c81b159087cb71f6b64676d533cf0d897d68 still contains the exact runtime guard reported in #61724: `sessions_spawn` exposes `streamTo`, normalizes it before branching, and returns `streamTo is only supported for runtime=acp` when `runtime` is `subagent`. The existing test suite also asserts that `runtime: "subagent"` plus `streamTo: "parent"` is rejected. An open PR (#68397) appears to track the proposed fix, but it is not merged into this checkout, so the underlying bug remains on main.

## Summary

Current `main` already implements friendlier Control UI session-dropdown labeling: it groups sessions by agent, derives group labels from `agents.list[].identity.name`/`name`, prefers human-readable session `label`/`displayName` over raw keys, and has regression tests covering the raw `webchat:g-agent-*` case and duplicate-main disambiguation.

## Summary

Current `main` does not already land this PR’s headline surfaces. The repo still lacks native `sessions.search`/`sessions.recall`, agent tools `sessions_search`/`sessions_recall`/`skills_manage`, and the proposed `src/replay/control` + `src/research` trees. Main has adjacent but narrower alternatives (`sessions_history`, trajectory export bundles, the experimental `skill_workshop` plugin, and the Honcho memory plugin), so there is not strong evidence to close this PR as implemented, obsolete, or plugin-scope-only.

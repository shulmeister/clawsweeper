## Summary

Current `main` already wires the Agent Overview primary-model selector to persisted agent config. The overview select reads the resolved primary model, `onModelChange` writes the value into `agents.list[].model` (creating an override entry when needed), and the save path submits via `config.set` then reloads config. The implementation appears to have landed in `fc5920fb51346a682b269fdbeeec0971ae4ae723`, which is newer than the latest tagged release `v2026.4.23`, so the fix is on `main` but not yet attributable to a shipped release tag.

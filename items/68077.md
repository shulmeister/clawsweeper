## Summary

Keep open. Current `main` already shipped the `.cdpUrl` redaction portion of this PR, but the MS Teams Windows OAuth fix is still absent on `main`. The proposed `cmd /c start` change should not be used as-is: the repo already has a safer Windows browser-open path (`rundll32.exe url.dll,FileProtocolHandler`) with dedicated tests, while the PR's new path would add a separate Windows opener and its test coverage remains weak.

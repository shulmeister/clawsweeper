## Summary

Keep open. Current main reduces the TUI/gateway model picker to the configured allowlist when one exists, but it does not implement the requested auth-status filtering. If no explicit model allowlist is configured, the gateway and embedded TUI backend still fall back to returning the full catalog, so the reported unauthenticated-model picker problem can still exist for auth-only setups.

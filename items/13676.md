## Summary

Keep open. Current `main` does not implement a first-class Gateway dashboard Project object or Project RPC/API surface. The Control UI still exposes sessions, agents, skills, nodes, and related controls separately, so the requested project list, project↔session binding, and project metadata surface are not present as described in #13676.

## Evidence

- **Control UI has no Projects tab or route:** `TAB_GROUPS` and `Tab` only include `overview`, `channels`, `instances`, `sessions`, `usage`, `cron`, `agents`, `skills`, `nodes`, `chat`, and settings tabs; there is no `projects` entry or path.

## Summary

Keep open. Current main has partial adjacent primitives, especially agents, per-agent workspaces/skills, sessions, and session runtime overrides, but it does not implement the requested first-class Project object or dashboard Projects workflow. The supplied related issues #12333 and #13535 overlap, but they are still open and do not clearly supersede all of #13676's acceptance criteria, especially project-scoped constraints and the commenter-added port/service registry idea.

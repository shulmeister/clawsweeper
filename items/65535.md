## Summary

Keep open. Current main still lacks a plugin hook for Telegram streaming preview edits/finalization: `message_edit_sending` is not a registered hook, Telegram preview streaming calls `sendMessage`/`editMessageText` directly, and `message_sending`/`message_sent` are only wired through normal `deliverReplies` delivery. The related #38340 PR is open and Discord-scoped, so it does not supersede this Telegram streaming interception request.

## Summary

Keep open. Current `main` documents Windows companion apps as planned, not shipped, while the only built-in status-widget-style surface is the macOS menu bar app. OpenClaw also already exposes `status` and `system-presence` surfaces that could support an external prototype, so this is a plausible feature request rather than a conservative close candidate.

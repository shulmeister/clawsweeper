## Summary

Issue #68184 is a useful Windows desktop/status-widget prototype, but the remaining product work is already covered by the open maintainer-owned Windows/Linux companion app tracker #75. Current main still has no Windows companion app; docs explicitly say it is planned, and the existing macOS menu bar docs show the analogous first-party status surface that #75 is meant to cover.

## Best Possible Solution

Close #68184 in favor of #75 as the canonical Windows/Linux companion-app tracker. Keep #75 open for product/design discussion, and fold the prototype's generic status-widget/project-card ideas into that thread or a future focused PR when Windows companion app work is ready.

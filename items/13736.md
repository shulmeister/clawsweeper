## Summary

Current `main` does not support per-agent `compaction` or `contextPruning` overrides in `agents.list[]`. The request remains unimplemented in the typed config surface, strict agent-entry schema, runtime resolution, and docs/help paths.

## Evidence

- **Agent config type lacks per-agent compaction/context pruning:** `AgentConfig` includes per-agent fields like `heartbeat`, `sandbox`, `params`, and `tools`, but no `compaction` or `contextPruning` fields.
  - file: [src/config/types.agents.ts:70](https://github.com/openclaw/openclaw/blob/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e/src/config/types.agents.ts#L70)
  - command: `nl -ba src/config/types.agents.ts | sed -n '60,140p'`
  - sha: [272bd59e7a8a](https://github.com/openclaw/openclaw/commit/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e)
- **Strict agent-entry schema rejects those keys:** `AgentEntrySchema` is strict and enumerates allowed `agents.list[]` properties; it does not define `compaction` or `contextPruning`, so per-agent overrides are not loadable through the runtime schema.
  - file: [src/config/zod-schema.agent-runtime.ts:812](https://github.com/openclaw/openclaw/blob/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e/src/config/zod-schema.agent-runtime.ts#L812)
  - command: `nl -ba src/config/zod-schema.agent-runtime.ts | sed -n '812,900p'`
  - sha: [272bd59e7a8a](https://github.com/openclaw/openclaw/commit/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e)
- **Compaction/context pruning schemas exist only under defaults:** The config schemas define both `contextPruning` and `compaction`, but only on `AgentDefaultsSchema`, i.e. under `agents.defaults.*`.
  - file: [src/config/zod-schema.agent-defaults.ts:123](https://github.com/openclaw/openclaw/blob/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e/src/config/zod-schema.agent-defaults.ts#L123)
  - command: `nl -ba src/config/zod-schema.agent-defaults.ts | sed -n '120,220p'`
  - sha: [272bd59e7a8a](https://github.com/openclaw/openclaw/commit/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e)
- **Runtime reads only global defaults:** Current compaction and context-pruning runtime setup reads `cfg?.agents?.defaults?.contextPruning` and `cfg?.agents?.defaults?.compaction`; no per-agent override path is consulted there.
  - file: [src/agents/pi-embedded-runner/extensions.ts:40](https://github.com/openclaw/openclaw/blob/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e/src/agents/pi-embedded-runner/extensions.ts#L40)
  - command: `nl -ba src/agents/pi-embedded-runner/extensions.ts | sed -n '36,110p'`
  - sha: [272bd59e7a8a](https://github.com/openclaw/openclaw/commit/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e)

  - file: [src/agents/pi-settings.ts:43](https://github.com/openclaw/openclaw/blob/272bd59e7a8a4d6ef6e03b9bdc1068a02986d56e/src/agents/pi-settings.ts#L43)

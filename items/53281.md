## Summary

Keep open. Current `main` still accepts `attachAs.mountPath` for `runtime="subagent"`, but only uses it as child guidance and does not add the caller-facing accepted-result note proposed by this PR. The PR branch is also unmerged and heavily diverged from current `main`, so there is no strong basis to close it as already implemented or obsolete.

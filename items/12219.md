## Summary

Keep open. Current main has adjacent safety infrastructure, including plugin `before_tool_call`/`before_install` hooks and a built-in scanner for gateway-backed skill dependency installs, but it does not implement the requested `skill.yaml` permission manifest standard, `openclaw skills audit`, pre-install permission review for `openclaw skills install`, signing/attestation, or ClawHub trust badges. The related #50090 is a broader skill ecosystem umbrella, not a clear canonical replacement for this specific proposal.

## Summary

Keep open. Current `main` still scopes `openclaw wiki` CLI loading through manifest command activation, `memory-core` still does not advertise `wiki` activation, and `memory-wiki` still depends on memory capability/public-artifact state. The PR matches current code and the discussion contains independent reproductions with real data-loss impact.

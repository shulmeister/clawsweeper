## Summary

Current `main` already provides supported shutdown commands for OpenClaw-managed background processes: `openclaw gateway stop`, legacy alias `openclaw daemon stop`, and `openclaw node stop`. The stop path is wired in CLI code, implemented in shared lifecycle handling, and documented in the shipped CLI docs.

## Evidence

## Close Comment

Closing after Codex review.

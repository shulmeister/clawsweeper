## Summary

Keep open. Current main has partial generic fallback hardening for thrown unclassified model errors, but the specific Anthropic streaming JSON parse path is still strict, can still convert SyntaxError messages into assistant error text, and short unclassified errors can still be surfaced rather than replaced with safe user-facing copy. The issue is recent and concrete, so it should not be closed conservatively.

## Summary

PR #60605 adds an optional standalone wrapper around the external `clawhub` CLI and Cisco AI Skill Scanner. That is useful third-party tooling, but it does not need a new OpenClaw core surface and fits the project's ClawHub/npm distribution model better than a repo-level script.

## Best Possible Solution

Close this PR from the core repository and direct the wrapper to ClawHub/npm as external tooling. If a core security change is desired, open a separate design issue for native `openclaw skills install/update` scanning using OpenClaw-owned scanner and install-path semantics.

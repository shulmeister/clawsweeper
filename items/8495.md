## Summary

Current main still lacks the requested WhatsApp group admission cleanup surface: no auto-leave option, no group-join webhook/event handling, and no WhatsApp leave-group message action. The related #11489 is useful and broader, but it does not exactly supersede the WhatsApp-specific auto-leave/manual-leave request, so conservative cleanup should keep #8495 open.

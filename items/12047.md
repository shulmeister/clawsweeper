## Summary

Keep #12047 open. Current main has a generic inbound file-extraction path that covers Telegram PDF/text-style document attachments, but the request explicitly includes .docx and similar Office documents, and current main deliberately skips application/vnd.* Office-style payloads. That leaves a meaningful requested capability unresolved.

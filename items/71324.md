## Summary

PR #71324 should stay open. Current main contains the earlier history sanitizer from #71229, but it still only strips normal line-delimited internal runtime context and heartbeat/internal-only rows. It does not implement this PR's additional legacy projection for quote-wrapped internal-context rows, `System: [...] Background task ...` rows, or `Gateway restart` status blocks. The closed #71312 was an earlier unmerged version of this same follow-up, not a canonical replacement.

## Best Possible Solution

Keep #71324 open and have maintainers review or merge a focused version of the Gateway history projection fix. The implementation should cover quote-wrapped internal runtime context, background-task status lines, gateway restart status blocks, and mixed status-plus-user text while preserving raw transcript sequence tracking and avoiding assistant-content over-stripping.

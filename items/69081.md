## Summary

Keep #69081 open. Current main still matches the reported bug shape: the TUI watchdog is armed from chat delta handling, but tool events do not refresh it, and reconnect handling reloads history without clearing the stale active run/watchdog state. The related PR #69125 proposed the right shape of fix but is closed unmerged in the provided GitHub context, and the current source does not contain those changes.

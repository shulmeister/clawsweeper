## Summary

Current main no longer lets the MiniMax OAuth onboarding patch's empty models array block model resolution. The persisted patch is still minimal, but the portal provider catalog is plugin-owned, activated by stored OAuth profiles, and merge logic falls back to implicit catalog models when an explicit provider has an empty models array.

## Summary

Issue #59148 is best treated as already implemented on `main`: the later cross-referenced metadata-leak fix for WebChat history replay shipped, the current Control UI loader filters the synthetic transcript-repair marker before rendering chat history, and regression coverage was added for that exact leak.

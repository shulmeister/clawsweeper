## Summary

Current main has a provider-owned forward-compat resolver for `claude-opus-4-7`, test coverage for the exact dynamic-resolution path, and the changelog shows the fix shipped in v2026.4.22. The issue matches a real regression in 2026.4.15, but it is no longer open work.

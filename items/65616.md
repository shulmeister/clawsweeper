## Summary

Close as `clawhub`: this PR adds a service-specific optional bundled plugin, but current `main` explicitly prefers shipping optional/community plugins outside core via ClawHub, and the existing public `before_tool_call` hook already supports this behavior without a core change.

## Evidence

That is plugin-owned functionality for a specific external guard service rather than a missing core runtime seam.

## Summary

Keep open. Current `main` does not implement the PR's two behavioral changes: `infer model run --local` still routes through the normal agent main-session path unless the caller supplies a session id/key, and transcript fallback path resolution still uses the agent/global resolver instead of the store/profile-local sessions directory. I also found no local history or docs evidence that this draft PR was already merged or superseded on `main`.

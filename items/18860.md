## Summary

Keep #18860 open. Current main still does not implement the requested `after_tools_resolved` plugin hook or an equivalent plugin-facing resolved-tool/schema discovery API. The linked feature request #21330 describes the same missing core API and is closed now, so this PR remains useful review context rather than being superseded by an open canonical item.

## Best Possible Solution

Keep this item open until OpenClaw either lands a narrow core plugin/runtime seam that exposes the resolved post-policy tool list and schemas, or a maintainer explicitly rejects the feature/API direction. The implementation likely belongs in `src/plugins/hook-types.ts`, `src/plugins/hooks.ts`, and the embedded runner after `effectiveTools` is finalized, with docs and focused contract tests.

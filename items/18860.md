## Summary

Keep #18860 open. Current main does not implement the requested `after_tools_resolved` plugin hook or any equivalent resolved-tool/schema discovery API, and the linked #21330 tracks the same missing core plugin API rather than superseding the PR. This is not a ClawHub/plugin-only cleanup because the needed capability is a core SDK/runtime seam.

## Summary

Keep open. Current main still hardcodes reasoning messages as italic Markdown and does not expose or thread a per-channel/per-agent reasoning format option. The related PR #25336 was closed unmerged and, per the provided discussion, was also flagged as incomplete; related bug #24205 is about suppressing leaked reasoning, not making visible reasoning format configurable.

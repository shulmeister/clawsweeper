## Summary

Do not close. Current main still has a docs/runtime mismatch for the sandbox-active case: the repo and deployed docs say per-call host=node is allowed from auto, while the runtime explicitly rejects node overrides from auto when a sandbox runtime is active. The later issue comment that this was resolved omitted that sandbox-active rejection, so there is not strong evidence for an implemented-on-main close.

## Summary

Current `main` already exposes the raw assistant payload to `llm_output` hooks, so provider-added rate-limit metadata is already reachable through `event.lastAssistant`. This PR’s extra top-level `rateLimits` field would be redundant and, per the PR description, still would not produce data until upstream starts attaching headers.

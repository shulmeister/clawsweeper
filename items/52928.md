## Summary

Current `main` does not implement an `agents.setDefault` Gateway RPC or any equivalent dedicated default-agent mutation. The wire schema still exposes only `agents.list/create/update/delete`, `agents.update` still cannot accept a `default` field, and docs still point clients to generic `config.patch` for partial config writes. This issue remains a valid feature request and should stay open.

## Summary

Current `main` does not implement per-agent `agentToAgent` or per-agent session-visibility scoping. The runtime still reads `tools.agentToAgent` and `tools.sessions.visibility` only from the global config, and per-agent `agents.list[].tools` does not accept those keys. This issue is still actionable, and the linked PR #59170 appears to be the active implementation attempt rather than something already shipped.

## Evidence

- **Per-agent agent config cannot express this:** `AgentConfig.tools` exists, but it is typed as `AgentToolsConfig`, which only covers generic tool policy, elevated, exec, fs, loopDetection, and sandbox overrides. There is no per-agent `sessions` or `agentToAgent` field in that schema.
  - file: [src/config/types.agents.ts:119](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/src/config/types.agents.ts#L119)

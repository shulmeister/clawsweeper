## Summary

Current main still includes full direct-channel inbound metadata for non-webchat DMs and has no visible `envelopeMode`/minimal metadata configuration. The issue is recent, unprotected, externally authored, and describes a remaining core prompt-composition behavior rather than something already implemented or clearly suitable for ClawHub-only closure.

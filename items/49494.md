## Summary

Current `main` does not expose usage/cost metadata on `agent.wait`. The terminal snapshot type, gateway response, client-side normalization, and docs all still describe only status/timing/error fields, so this issue should remain open.

## Evidence

- **Terminal snapshot type still lacks metadata:** `AgentWaitTerminalSnapshot` only defines `status`, `startedAt`, `endedAt`, and `error`; there is no `meta`, `usage`, or `cost` field.
  - file: [src/gateway/server-methods/agent-wait-dedupe.ts:3](https://github.com/openclaw/openclaw/blob/320d52a23e41a326287826219a20f1f29f334554/src/gateway/server-methods/agent-wait-dedupe.ts#L3)
  - sha: [320d52a23e41](https://github.com/openclaw/openclaw/commit/320d52a23e41a326287826219a20f1f29f334554)
- **Gateway `agent.wait` response still omits metadata:** The `agent.wait` handler responds with only `runId`, `status`, `startedAt`, `endedAt`, and `error`. No usage/cost payload is threaded into the wire response.
  - file: [src/gateway/server-methods/agent.ts:1113](https://github.com/openclaw/openclaw/blob/320d52a23e41a326287826219a20f1f29f334554/src/gateway/server-methods/agent.ts#L1113)
  - sha: [320d52a23e41](https://github.com/openclaw/openclaw/commit/320d52a23e41a326287826219a20f1f29f334554)
- **Client wait contract still strips to status/timing/error:** `AgentWaitResult` and `normalizeAgentWaitResult()` only preserve `status`, `error`, `startedAt`, and `endedAt`, confirming the current contract does not include usage/cost metadata.
  - file: [src/agents/run-wait.ts:20](https://github.com/openclaw/openclaw/blob/320d52a23e41a326287826219a20f1f29f334554/src/agents/run-wait.ts#L20)
  - sha: [320d52a23e41](https://github.com/openclaw/openclaw/commit/320d52a23e41a326287826219a20f1f29f334554)
- **Docs still describe the old terminal snapshot:** The current docs say `agent.wait` returns `{ status: ok|error|timeout, startedAt, endedAt, error? }`, matching the code and not the requested metadata extension.
  - file: [docs/concepts/agent-loop.md:42](https://github.com/openclaw/openclaw/blob/320d52a23e41a326287826219a20f1f29f334554/docs/concepts/agent-loop.md#L42)
  - sha: [320d52a23e41](https://github.com/openclaw/openclaw/commit/320d52a23e41a326287826219a20f1f29f334554)

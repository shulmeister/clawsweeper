## Summary

Keep open. Current main does not implement Windows service support for gateway status/control; the win32 adapter still hard-codes Scheduled Task behavior, docs still describe Scheduled Task plus Startup-folder fallback only, and there is no Windows service module on main. This PR is still a live feature change rather than something already shipped or obsolete.

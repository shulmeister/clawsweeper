## Summary

Keep open. Current `main` does not contain the PR’s orphan-chain recovery behavior: it still repairs only the current orphaned user leaf by text-merging that single message into the next prompt, with no multi-turn recovery, no recovered image carry-forward, and no raw-prompt dedupe path matching the PR description.

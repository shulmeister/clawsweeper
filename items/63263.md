## Summary

Keep open. Current main only has one of the proposed reliability fixes: image describe already passes agentDir, but the local infer session isolation, image --prompt CLI wiring, structured web-search error fallback, and openai-codex image auto-priority are still absent. The PR is open and unmerged, and the review comments identify concrete changes that should be addressed before landing rather than closing the PR as obsolete.

## Best Possible Solution

Keep this PR open as an implementation candidate. The best path is to revise it so structured web-search fallback is limited to auth/availability-style failures, make image --prompt reliably override per-request prompts even when media model entries define prompt, keep the already-shipped agentDir behavior aligned, and then run the targeted CLI, web-search, and media-understanding tests before maintainer review/landing.

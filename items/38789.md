## Summary

Keep open. Current `main` does not contain this fix: embedded sandbox runs still prefer the saved skills snapshot prompt, and that snapshot is built from the host workspace, while non-`rw` sandbox sessions mirror skills into the sandbox workspace. The PR’s goal remains valid, although the submitted patch still appears to need follow-up for reviewer-raised edge cases.

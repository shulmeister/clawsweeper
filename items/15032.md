## Summary

Keep open. Current main still has only global sub-agent tool policy; `sessions_spawn` does not expose a per-spawn `tools` allow/deny parameter, and the linked implementation PR #23506 was closed unmerged. The Claw Drive comment and PR attempt are useful supporting evidence that this is a real core API need, not a completed cleanup item.

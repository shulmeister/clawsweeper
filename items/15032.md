## Summary

Current `main` does not implement per-spawn tool restrictions for `sessions_spawn`. The spawn tool schema and `SpawnSubagentParams` still have no `tools`/`allow`/`deny` field, and the active subagent tool filtering only reads global config (`tools.subagents.tools`) plus stored session role/depth, not per-call overrides.

## Evidence

## Summary

Keep open. Current main still writes and bootstraps the macOS gateway LaunchAgent under the resolved HOME path, so a HOME on /Volumes/<externalDrive> would still target the external volume path described in the issue. The issue is recent, has concrete reproduction details, and is not superseded by source or history evidence available in this checkout.

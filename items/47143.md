## Summary

Keep open. Current main still treats `compactionCount: 0` and `memoryFlushCompactionCount: 0` as already flushed, and both the token-threshold path and forced transcript-size path still depend on that helper. The exact fix PRs mentioned in the issue context were closed unmerged, while a later commenter also confirmed the same failure mode plus related preflight gaps.

## Summary

Close PR #68334 as implemented on main. Current main fixes the observable memory-wiki public-artifact loss without the proposed additive merge split: cache hits restore a complete cached memory state including memoryCapability, snapshot/rollback paths preserve previousMemoryCapability, and restoreMemoryPluginState remains a destructive swap for rollback correctness.

## Best Possible Solution

Close PR #68334 as already implemented on main. Keep the shipped loader behavior: full cached memory state is restored on cache hits, previousMemoryCapability is included in snapshot/rollback restores, and restoreMemoryPluginState remains destructive for rollback/cache correctness. Do not merge the additive mergeMemoryPluginState split; open a new focused issue only if current main still reproduces disappearing public artifacts.

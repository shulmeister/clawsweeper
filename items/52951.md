## Summary

Current `main` does not implement `tools.fs.roots` or the per-root read/write enforcement described in PR #52951. The checked-out tree still exposes only the binary `tools.fs.workspaceOnly` model in config, tool policy, runtime FS tool wiring, and outbound media plumbing, so this PR should remain open.

## Summary

Close PR #64444 as obsolete on current main. The PR documents an s390x global npm install failure caused by root installation of @lancedb/lancedb, but current main and the latest release no longer install LanceDB as a root OpenClaw dependency; LanceDB is plugin-local to memory-lancedb and staged lazily only when that plugin is enabled. The proposed --force install guide should not be merged as-is.

## Best Possible Solution

Close this stale PR rather than merge the --force IBM Z guide. Keep the current root package and lazy plugin-local runtime dependency design. If IBM Z/s390x docs are still desired, create a fresh short docs change against current behavior: normal OpenClaw install follows the standard Linux/Node path, with a separate note that memory-lancedb may be unavailable when users opt into that plugin.

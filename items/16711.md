## Summary

Keep #16711 open. Current main still has only the capped inline Control UI chat composer and does not implement the requested Expand button, larger composer modal, Apply/Cancel/Send modal workflow, or modal keyboard behavior where Enter inserts a newline and Ctrl/Cmd+Enter sends. The existing Codex review comment on the issue also explicitly kept it open, and the related items are adjacent composer/session/modal work rather than a canonical tracker for this long-prompt editor request.

## Best Possible Solution

Keep this open for first-class Control UI work. The likely implementation belongs around `ui/src/ui/views/chat.ts`, `ui/src/ui/chat/run-controls.ts`, `ui/src/ui/app-render.ts`, and `ui/src/styles/chat/layout.css`: add an Expand control near the composer actions, modal draft state with Close/Cancel, Apply, and Send semantics, modal keyboard behavior where Enter inserts a newline and Ctrl/Cmd+Enter sends, plus focused UI tests. Do not close unless the feature lands or a narrower canonical issue explicitly replaces this one.

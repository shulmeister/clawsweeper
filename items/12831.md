## Summary

Keep open. Current main has useful partial plumbing for live tool events, inline tool cards, and a resizable details sidebar, but it does not yet provide the requested separate activity/observer surface. Verbose remains session-scoped, and tool result payloads are still gated by verbose=full rather than an independent observer mode.

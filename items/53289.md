## Summary

Keep open. Current `main` does not contain the proposed clarification in `docs/tools/index.md`, and the PR as written documents an invalid sessions CLI form (`openclaw sessions list`). The underlying docs gap may still be worth fixing, but this PR should not be closed as already implemented or obsolete.

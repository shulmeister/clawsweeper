## Summary

Keep open. Current main does not implement Ralph Loop or a per-agent max Ralph iteration setting. There is an adjacent strict-agentic execution contract, but it is scoped to GPT-5-family plan-only retry behavior and has a fixed retry cap, so it does not satisfy the requested STOP-driven loop with per-agent iteration precedence. The issue is coherent, author is not protected, label is only enhancement, and the +1 comment supports demand rather than superseding it.

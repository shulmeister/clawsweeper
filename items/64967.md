## Summary

Current `main` still ignores `models.mode="replace"` in the CLI picker and gateway catalog, which matches issue #64950. PR #64967 is therefore not implemented on `main`, and the duplicate/open-PR plus review-discussion state is not strong enough for a conservative auto-close.

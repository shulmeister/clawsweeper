## Summary

Keep open. Current `main` still ships and documents the Docker path as `node dist/index.js`, and that path remains a separate legacy entrypoint from the newer `openclaw.mjs`/`dist/entry.js` fast-path wrapper. There is startup-performance work on `main`, but the checked-in benchmark coverage targets `openclaw.mjs`, not the reported Docker `dist/index.js` path, so there is not strong evidence that issue #65032 is fixed or obsolete.

## Summary

Keep open. Current main still defaults to the builtin memory backend and that backend still depends on node:sqlite providing FTS5. When FTS is unavailable, keyword/BM25 results are skipped and search degrades to vector-only or empty results. The related #62338 PR is still open and only covers doctor visibility, while the #64776 vector-extension fix is a separate sqlite-vec loading issue.

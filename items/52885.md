## Summary

Current main already addresses the reported external WeChat plugin compatibility failure: it documents `@tencent-weixin/openclaw-weixin` 2.x as the supported line for OpenClaw `>=2026.3.22`, links the host `openclaw` package into external plugins that declare it as a peer dependency, and includes packaged-install/plugin-sdk runtime resolution fixes and tests. The reported `v1.0.3` incompatibility is obsolete on main.

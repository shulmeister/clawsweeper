## Summary

Keep open. Current main still has the reported pre-GA Anthropic 1M behavior: `context1m: true` adds `context-1m-2025-08-07`, OAuth-style `sk-ant-oat` auth strips that beta and warns, tests assert both behaviors, and the Anthropic docs still describe 1M context as beta-gated. The linked PRs #45592 and #45613 are relevant follow-up work, but they are open/unmerged in the provided context, so the issue is not implemented on main and there is no clear merged superseding fix to close against.

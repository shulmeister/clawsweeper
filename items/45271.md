## Summary

Keep open. Current main has later safeguards for phased commentary/final_answer text, but the reported Ollama/GLM shape is an unphased assistant message containing natural-language text before a structured toolCall. I did not find the requested delivery-time strip guard or config flag on main, and the generic channel delivery path still emits unphased visible assistant text.

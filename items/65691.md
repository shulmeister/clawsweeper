## Summary

Keep #65691 open. Current main at 631552c554095b0c520376db9131dac0f5427f94 still does not implement Slack message search: Slack action discovery does not advertise `search`, Slack dispatch/runtime have no `searchMessages` path, the Slack Web API wrapper has no `search.messages` helper, and CLI/docs still describe message search as Discord-only. The PR is an active rebuild of closed unmerged #58143 with relevant implementation and tests; the posted review comments are actionable merge-review items, not evidence that the PR is obsolete.

## Best Possible Solution

Keep #65691 open for maintainer review. The best path is to finish this PR or a successor in the bundled Slack plugin: add a `search.messages` wrapper, wire Slack runtime and message-action dispatch, advertise `search` only when runnable for the selected Slack account, update CLI/docs for Slack token requirements, and keep focused tests for Slack search plus Discord search validation.

## Summary

Keep open. Current `main` still does not implement Slack message search, and this rebuilt PR from #58143 contains useful work but still has unresolved correctness and regression issues in review.

## Evidence

## Risks / Open Questions

- Merging as-is would regress Discord CLI validation by removing the immediate `--guild-id` requirement without an equivalent channel-specific guard.
- Slack searches scoped with raw channel IDs may return broader-than-requested results because the query syntax does not match Slack’s documented channel scoping form.
- Slack accounts configured only with a user token could execute search at runtime but still never advertise the capability, creating a discovery/runtime mismatch.

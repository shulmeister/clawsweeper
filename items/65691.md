## Summary

Keep this PR open. Current main still does not implement Slack message search: Slack action discovery does not advertise `search`, Slack dispatch has no `search` mapping, the Slack runtime has no `searchMessages` action, the Slack Web API wrapper has no `search.messages` helper, and the CLI/docs still present message search as Discord-only. The PR is an active unmerged implementation candidate rebuilt from closed PR #58143, and the latest discussion says the posted account-scoping, channel-scoping, and Discord UX review comments were addressed on PR head `52b5e79d3f0f50e9c147eab17b8ef7fc3e5ffd50`.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to finish and land this PR or a successor that adds Slack `search.messages` support inside the bundled Slack plugin: action discovery gated by runnable user-token accounts, dispatch/runtime support for `searchMessages`, a Slack Web API wrapper that uses the correct user token and channel scoping behavior, shared CLI/docs updates, and focused Slack plus Discord regression tests.

## Summary

Keep open. #11623 is a clear contributor feature request for a macOS app-level floating agent bubble UI. Current main has menu bar status, menu-anchored WebChat, hover HUD, Canvas, and Talk Mode overlays, but not per-agent floating desktop bubbles with names, drag/reposition, mouse-follow, and inline chat. A ClawHub close would be premature because current plugin surfaces do not expose a native macOS window/UI extension point for this.

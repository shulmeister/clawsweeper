## Summary

Keep open. Current main still has the reported one-Enter failure mode for `/status`: `/status` is intentionally not `executeLocal`, the slash menu intercepts Enter while open, and selecting a non-local command only fills `/status ` instead of sending. The related one-line `executeLocal: true` approach is not sufficient because the local slash executor has no `status` handler, and current tests explicitly assert `/status` stays on the agent path.

## Summary

Keep open. Current main has plugin hooks for model/tool/message/session/subagent/gateway/install surfaces, but no skill invocation/success/failure lifecycle hook. Skill slash commands are either rewritten into a normal model request or directly dispatch a tool, and the direct dispatch path does not emit a skill-level before/after event. The commenter’s enterprise permission-check use case points to a real plugin/core API gap rather than optional ClawHub content.

## Evidence

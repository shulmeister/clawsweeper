## Summary

Keep open. Current main has strong adjacent work: `openclaw update` now refreshes the installed gateway service environment and restarts by default. But the issue specifically asks for auto-restart after a raw package-manager upgrade such as `npm i -g openclaw@latest`; that path still has no lifecycle hook that runs `gateway install` or `gateway restart`, and the docs still present manual post-update restart steps for package-manager updates.

## Evidence

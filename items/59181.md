## Summary

Current `main` no longer matches the issue’s reported channel-validation model: bundled gateway channels are catalog-driven instead of a fixed 9-item list, plugin channel lookup uses the pinned channel registry, and regression tests now cover `openclaw-weixin`/`qqbot` resolution across registry churn. That addresses the reported `sessions_spawn` -> `unknown channel` failure mode for the cited bundled and plugin channels.

## Evidence

## Summary

Current main already has an explicit Control UI image-upload path that reads pasted/dropped/file-picked images, sends them through `chat.send` as attachments, and the gateway persists them as agent-visible media. The same implementation is present in the shipped `v2026.4.23` release.

## Evidence

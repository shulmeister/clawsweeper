## Summary

Keep open. Current main at a57d681d still injects the blanket group-chat instruction forbidding same-chat `message` tool sends, and the tests/fixtures still assert that old prompt. The linked bug #43146 remains the canonical open tracker, while #48196 was closed as a duplicate pointing back to #43146/#43276, so this PR still represents useful unresolved maintainer work rather than cleanup-close material.

## Best Possible Solution

Keep #43276 open or have a maintainer land an updated equivalent: change `buildGroupChatContext` so normal text replies are direct while files/images/attachments may use the `message` tool to the same group, update the current prompt tests/fixtures, and close #43146 when the fix ships.

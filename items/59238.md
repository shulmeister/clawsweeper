## Summary

Keep open. Current `main` clearly wires Slack per-channel `systemPrompt` into `GroupSystemPrompt`, carries it through `extraSystemPrompt`, and appends that text into the final agent system prompt, but the issue's actual claim is that live agent behavior still ignores those instructions. I did not find a fixing commit, changelog entry, or reproducible proof that the behavioral problem is resolved on `main`, so there is not strong enough evidence to close conservatively.

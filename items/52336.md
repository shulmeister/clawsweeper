## Summary

Keep open. Current main has several nearby browser/CDP hardening fixes, but I did not find strong evidence that issue #52336's specific intermittent `gateway closed (1000 normal closure)` path is implemented-on-main or obsolete. The current CLI still sends browser actions through `browser.request` over the gateway, and the generic 1000-close formatter is still the same broad surface.

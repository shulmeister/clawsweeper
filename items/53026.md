## Summary

Issue #53026 is already fixed on current `main`, and the released `v2026.4.22` tag already contains the same fix: `scripts/ui.js` now bootstraps missing UI deps with plain `pnpm install`, not `pnpm install --prod`, which preserves `ui/package.json` devDependencies like `vite`.

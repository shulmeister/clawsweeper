## Summary

The reported Chrome for Testing 145.0.7632.6 macOS arm64 download failure no longer reproduces against current evidence: current main and the latest release use Playwright Core 1.59.1, whose upstream browser manifest targets Chrome for Testing 147.0.7727.15, and both the old 145.0.7632.6 mac-arm64 artifacts and the current 147.0.7727.15 mac-arm64 artifacts now return HTTP 200. Normal OpenClaw install/update also does not eagerly download Playwright browsers by default.

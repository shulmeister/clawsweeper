## Summary

Keep this PR open. Current main still renders the Control UI chat transcript as `role="log"` with hardcoded `aria-live="polite"` while streaming, and the focused live-region regression tests proposed by the PR are not present on main. The related issue #65538 remains open, and the alternate aria-busy PR #67424 was closed unmerged, so this PR is still a live implementation candidate rather than obsolete.

## Best Possible Solution

Keep the PR open and review it against #65538. The likely maintainer path is to land a focused Control UI fix with regression tests, possibly adapting this PR to the more robust `aria-busy` approach raised in review, then close the linked bug once the implementation is merged.

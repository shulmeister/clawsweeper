## Summary

Keep #25493 open. Current main still hardcodes the requested Safety and OpenClaw CLI Quick Reference prompt sections, and the available customization paths are either whole-prompt replacement or narrow provider/runtime section overrides that do not expose Safety, system tips, or CLI reference. The related PR #25938 directly attempted this feature but was closed unmerged.

## Best Possible Solution

Keep #25493 open. The best maintainer path is to add documented and tested section/template overrides for selected OpenClaw-owned prompt sections such as Safety, tool-call/system tips, and CLI reference, or make an explicit product decision that those sections remain intentionally fixed and document `systemPromptOverride` plus hooks/provider overlays as the supported escape hatches.

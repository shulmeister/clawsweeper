## Summary

Keep #9607 open. Current main still ships the bundled Himalaya skill with stale ID-first move/copy examples, the composition reference still omits the required message ID in its export tip, and the requested html-email/philosophy/plain-text/common-mistakes/save-copy guidance is still absent.

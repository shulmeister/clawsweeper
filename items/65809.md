## Summary

Keep open. Current `main` still embeds the Slack manifest JSON inside the setup note, so the underlying copy/paste bug is still present, but this PR's `console.log`-based fix is not safe because gateway/UI wizard flows only surface `WizardPrompter` steps and do not carry arbitrary stdout.

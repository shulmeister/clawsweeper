## Summary

Keep open. Current main has generic VPS, Docker, Kubernetes, GCP/Azure/Hetzner, and Amazon Bedrock provider documentation, but it still does not provide the requested AWS deployment guide for EC2/ECS/Lambda. The issue discussion contains concrete production requirements and an active AWS sample repo that should feed a docs PR, not close the request.

## Best Possible Solution

Keep #13597 open until OpenClaw ships an AWS deployment page, likely under docs/install/aws.md with install/vps/platform navigation. The guide should use Docker Compose as the primary EC2/Lightsail path, cover durable storage, gateway bind/reverse proxy/ALB/TLS, security groups, IAM and Secrets Manager, ECS/Fargate task/service/networking, Lambda limitations or non-applicability, monitoring/logging, backups, troubleshooting, multi-agent topology, and link the AWS sample repo from the issue discussion where appropriate.

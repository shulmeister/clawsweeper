## Summary

Keep #13597 open. Current main still has no AWS deployment guide or AWS install navigation entry. Existing docs cover useful pieces, including VPS, Docker, Kubernetes, gateway security, and Amazon Bedrock authentication, but they do not satisfy the requested EC2/ECS/Fargate/Lambda production deployment runbook. The issue discussion remains actionable and includes concrete production concerns plus an AWS sample repository link.

## Best Possible Solution

Keep the issue open until OpenClaw ships an AWS install page, likely under docs/install/aws.md and linked from docs/docs.json, docs/install/index.md, and relevant VPS/provider pages. The guide should make Docker Compose the primary EC2/Lightsail path, cover durable state and backups, gateway bind/reverse proxy/ALB/TLS/auth/origin settings, security groups, IAM roles and task roles, Secrets Manager, logging/monitoring, troubleshooting, performance and hardening, ECS/Fargate task/service/networking, Lambda applicability or non-support, multi-agent topology, and evaluate or link the AWS sample repo from the issue discussion.

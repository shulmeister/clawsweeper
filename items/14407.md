## Summary

Keep open. Current main has partial web support for gateway/config sensitive values, but the requested cross-platform password visibility toggle is not implemented across all named surfaces. The related PR #7124 was closed unmerged, and at least the Skills API key field plus iOS password fields still lack a reveal toggle.

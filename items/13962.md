## Summary

Keep open. Current `main` already covers the context-window half of the request with pending-only group history (`historyLimit`) that is injected when a mention-gated group message finally triggers a reply, but there is no evidence of distinct mentioned-vs-background model routing or per-group `mentionModel`/`backgroundModel` config. This is partial overlap, not an implemented fix for issue #13962.

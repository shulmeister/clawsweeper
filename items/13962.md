## Summary

Keep open. Current main implements part of the requested group context behavior by storing skipped unmentioned WhatsApp group messages and injecting them into the next processed group turn, but it does not implement the requested per-mention model routing or the proposed per-group mention context size. The existing plugin hooks also do not appear sufficient to move this to ClawHub because WhatsApp returns before dispatch for skipped unmentioned group messages.

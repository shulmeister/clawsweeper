## Summary

Keep open. Current main partially addresses the trust/audit problem with an Agents > Files panel that can load, edit, preview, and save allowlisted workspace files including USER.md and MEMORY.md, but the requested Memory-specific surface is not complete: the UI tab is generic Files, and the gateway only lists/allows bootstrap files plus MEMORY.md, not today's memory/YYYY-MM-DD.md daily log.

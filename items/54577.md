## Summary

Keep this open. Current main still lacks the requested `openclaw timeline` command, `/timeline` chat command, Control UI or TUI activity timeline panel, and JSON-exportable cross-agent lifecycle timeline. Existing agent events, trajectory export, subagent status, and task records are adjacent implementation material but do not satisfy the user-facing timeline view. Related #60814 remains an open draft web dashboard candidate and has not shipped this issue's full scope.

## Best Possible Solution

Keep this issue open until maintainers scope and ship the activity timeline product. The best path is to either broaden and land #60814 with coverage for the CLI command, `/timeline`, Control UI/TUI surface, event persistence/export contract, filters, tests, and docs, or split a focused core/gateway activity-timeline implementation over the existing agent event, task, subagent, session, and trajectory data.

## Summary

Keep open. Current `main` still reloads chat history immediately after `sessions.reset` in both clear-history code paths, so the reported race is not implemented away on `main`. This PR also appears incomplete because it only targets `ui/src/ui/app-chat.ts` while the Control UI button path in `ui/src/ui/app-render.ts` still does the same immediate reload.

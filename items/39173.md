## Summary

Current `main` already has the reported heartbeat fix: the `requests-in-flight` path no longer advances the schedule, the wake layer retries after 1s, and regression tests cover both the retry path and the no-drift scheduling case. The same implementation is present in release `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).

## Summary

Current main covers the reported Ollama failover/compaction loop symptoms: native Ollama simple-completion paths now register the provider-owned stream, context-overflow errors stay in bounded compaction handling instead of cascading into model fallback, and v2026.4.24 includes timeout propagation for slow local Ollama calls that previously hit the default 60s body timeout.

## Summary

Current `main` already strips assistant `<final>` tags through the shared extraction/sanitization path, but the exact end-to-end regression coverage proposed by PR #65187 is not present in `ui/src/ui/chat/message-extract.test.ts`. The review discussion says the missing cached assertions were addressed in follow-up PR #65336, but this checkout does not provide strong enough proof that equivalent message-extraction tests landed on `main`, so the conservative cleanup decision is to keep this PR open.

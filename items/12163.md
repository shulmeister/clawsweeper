## Summary

Keep #12163 open. Current main partially implements OpenAI TTS speed plumbing for config/talk/runtime API calls, but the reported model directive path still is not implemented for OpenAI: the OpenAI speech provider directive parser only handles voice/model tokens, while generic directive parsing depends on provider parse handlers. The related user PRs #12177 and #22618 are useful context but were closed unmerged, so they do not supersede the issue.

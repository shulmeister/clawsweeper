## Summary

Keep open. Current main does not implement the raw-mode pending changes diff requested by PR #48621 / issue #39831: the config view still computes and renders granular diffs only for form mode, while raw mode only shows a generic unsaved-changes state and the raw textarea. The PR also has maintainer engagement and no protected cleanup close reason applies.

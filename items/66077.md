## Summary

Current main resolves memory embedding providers through plugin-declared capability providers when a requested id is not already registered, and the bundled Ollama plugin declares/registers `ollama` as a memory embedding provider. The exact v2026.4.12 early-return behavior reported in #66077 is gone by v2026.4.20 and remains fixed on current main.

## Evidence

## Summary

Keep open. Current main still supports WhatsApp sending reactions, but not querying/listing message reactions, storing searchable reaction history, or surfacing WhatsApp reaction events to agents. The related PR #13641 was closed unmerged and only attempted incoming event surfacing, not the requested historical/query path.

## Best Possible Solution

Keep the issue open until a maintainer either adds the safe-mode recovery instructions to the generated TOOLS.md/public docs, or identifies the external installer/safe-mode workspace repo that owns setup-safe-mode-workspace.sh and moves/links the work there. The implemented docs should cover both token sources, visible OAuth re-auth without detached tmux, and the required safe-mode flag/reset exit procedure.

## Summary

Keep open. Current main appears to have the workspace bootstrap-file injection portion covered as a generic agent-runtime behavior, but the requested global Slack `channels.slack.systemPrompt` remains missing. Slack only accepts/uses `systemPrompt` under per-channel config entries, and the related Slack file PR tracks media ingestion rather than this prompt/config feature.

## Evidence

- **Slack config schema lacks top-level global prompt:** `SlackChannelSchema` includes `systemPrompt`, but the strict `SlackAccountSchema`/`SlackConfigSchema` fields do not include a top-level `systemPrompt`, so the requested `channels.slack.systemPrompt` shape is not accepted by current config schema.

## Summary

Keep open. Current `main` already injects workspace bootstrap files into the model system prompt generically, so the Slack "zero context" part is obsolete on current code. But the requested global Slack override is still not implemented: Slack only supports `systemPrompt` on `channels.slack.channels.<id>`, not on top-level `channels.slack` or per-account Slack config.

## Summary

Current main and the v2026.4.24 release contain fixes for both reported Google Chat failure modes: the webhook account lifecycle stays pending until abort instead of restarting, add-on JWT principal failures are surfaced with specific auth reasons and startup warnings, appPrincipal is part of the config contract, and status uses flattened audience fields from the account snapshot.

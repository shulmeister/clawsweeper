## Summary

Keep #13304 open. Current main supports static binding-based agent routing and a separate content-based model/provider hook, but it does not implement content-based agent routing in `bindings`, nor a documented plugin-hook decision that reroutes an inbound message to a different existing agent with prefix stripping.

## Evidence

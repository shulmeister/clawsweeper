## Summary

Current `main` already treats `apply_patch`, `memory_search`, and `memory_get` as built-in coding-profile core tools, suppresses the false `tools.profile (coding)` unknown-entry warning for baseline profile entries, and the changelog says that fix shipped in release `2026.3.13`.

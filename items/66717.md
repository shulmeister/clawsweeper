## Summary

PR #66717 is a draft bundled plugin for optional ADP/discovery-verification behavior. Current main’s product scope and plugin docs route optional plugin capabilities to externally maintained ClawHub/npm packages, and the needed hook plus guarded-fetch surfaces already exist for an external implementation.

## Best Possible Solution

Do not merge this as a bundled core plugin. Externalize the PoC as a maintained ClawHub/npm plugin or bundle using the existing plugin SDK `before_tool_call` hook and guarded-fetch surfaces, and keep any future core adoption discussion in the canonical discovery/trust issues once there is a stable standard or a concrete missing SDK seam.

## Summary

Current `main` already implements visible queuing for messages sent during an active run, explicit stop/abort handling in Control UI, reconnect-safe pending abort replay, and gateway/session abort plumbing for active runs. The reported behavior matches older builds, but not current `main`.

## Evidence

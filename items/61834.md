## Summary

Keep open. Current main still does not expose QMD 2.1.0 no-rerank through either `memory.qmd` config or the `memory_search` request schema, and the QMD manager does not pass `--no-rerank` or MCP `rerank: false` in the direct or mcporter query paths.

## Summary

Keep open. Current main supports Cloudflare AI Gateway onboarding only for the Anthropic Gateway endpoint and the default Claude model. There is still no Workers AI provider-type/model-selection path for `--auth-choice cloudflare-ai-gateway-api-key`; the generic `custom-api-key` path can serve as a workaround, but it does not implement the requested Cloudflare onboarding flow.

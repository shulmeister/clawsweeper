## Summary

Keep #46810 open. Current main partially implements the Control UI exec approval modal, including queued pending requests, a command block, security/ask metadata, timeout text, and Allow once / Always allow / Deny buttons. The title/body request is broader, though: command syntax highlighting and a requested-vs-effective command preview/diff are still not present in the Control UI path, and the UI parser ignores richer approval fields that the gateway already exposes.

## Summary

Keep open. Current main still does not emit the requested internal `gateway:shutdown` or `gateway:pre-restart` hook events in the gateway close path, and the related feature issue remains open. The existing `gateway_stop` plugin hook is adjacent but not a full substitute for the PR's requested internal lifecycle events and restart metadata.

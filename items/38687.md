Keep open. Current main at 46b9044c3f9d8519a56024f032b4b6e0b42aaf2b still does not implement the PR's target-aware proxy bypass for media backend baseUrl/NO_PROXY matches, and the PR discussion shows the branch addressed substantive review feedback rather than being obsolete.

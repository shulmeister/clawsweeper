## Summary

Keep #70629 open. Current main still maps `reasoning: "minimal"` to `thinkingBudget: 128` for Gemini 2.5 Flash-Lite via the broad `2.5-flash` branch, while current Google Gemini API docs list Flash-Lite's budget range as 512-24576. The PR is not implemented on main or superseded by the related #57683 zero-budget issue, though the exact patch should be reviewed because current docs only require the 512 floor for Flash-Lite, not necessarily for 2.5 Pro or 2.5 Flash. ([ai.google.dev](https://ai.google.dev/gemini-api/docs/thinking))

## Best Possible Solution

Keep the PR open and turn it into a docs-backed, tested fix: at minimum avoid sending `thinkingBudget: 128` for `gemini-2.5-flash-lite` minimal reasoning, add the regression test from the PR, and only change 2.5 Pro/Flash floors if live Google API evidence confirms the docs are stale for those models.

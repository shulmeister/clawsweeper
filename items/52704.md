## Summary

Keep open. Current main has gateway health probes and OpenTelemetry export, but it does not implement the requested Pulse feature set: there is no Prometheus-style `/metrics` endpoint, no bundled Prometheus/Grafana/Alertmanager setup, and no `infra/pulse` dashboard assets. This is still a coherent, in-scope observability discussion rather than something already implemented or safe to auto-close.

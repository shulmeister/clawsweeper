## Summary

Keep open. Current `main` still computes `vectorScore` and `textScore` inside hybrid memory search, but `mergeHybridResults` returns only the combined `score`, the public `MemorySearchResult` type still exposes only `score`, and the `memory_search` tool response spreads those typed results directly. The request is therefore not implemented on `main` yet. The issue is also recent and already linked to two open PRs in the provided timeline.

## Summary

PR #68364 is a narrower stable-session-key patch that is superseded by open PR #70464. Current main has the earlier #67023 cleanup and release note, but it still does not contain #68364's stable key/idempotency-key changes; #70464 carries those same changes and adds the missing scoped gateway delete-session permission path for memory-core dreaming sessions.

## Best Possible Solution

Close #68364 as superseded and keep review focused on #70464 or its successor, because that path combines the stable workspace+phase narrative session key, per-run idempotency key, pre-run cleanup, and the narrow gateway permission needed for memory-core to delete its own transient dreaming sessions.

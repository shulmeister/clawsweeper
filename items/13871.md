## Summary

Current `main` already provides browser voice input in the Control UI/WebChat chat composer: a mic button is shown when browser speech recognition is available, interim listening text is displayed, and final transcripts are appended into the draft. The same implementation is present in the shipped `v2026.4.22` release.

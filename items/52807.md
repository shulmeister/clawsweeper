## Summary

Current `main` no longer uses the reported broad runtime-loading path for implicit provider/model discovery. The `ensureOpenClawModelsJson` path now resolves discovery providers through a manifest/discovery-entry flow, and any fallback full plugin load is explicitly bounded to selected plugin IDs rather than cold-loading the whole provider runtime tree.

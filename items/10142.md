## Summary

Keep open. Current main has related plugin lifecycle hooks, but it does not implement the requested internal `session:end` hook event for HOOK.md/internal hook handlers. The internal hook catalog and source only show session compaction and session patch events, while `session_end` is a separate plugin hook surface. Related PRs in the provided context are either closed unmerged or still open, so there is not enough evidence to close this as implemented or superseded.

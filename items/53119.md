## Summary

Keep open. Current `main` does not contain the proposed helper change, and the repo still intentionally treats `runLevelError` as fatal even when later payload text exists. The PR also carries unresolved concerns: its semantics conflict with current tests, and the GitHub context shows unrelated `skills/gary-scripts/*.py` files bundled into the branch.

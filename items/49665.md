## Summary

Keep open. Current `main` still has the reported lazy-load failure mode: `createLazy()` caches the first `import()` promise without any rejection handling, and `lazyRender()` renders `nothing` whenever the module getter returns `null`. The same code is present on current `main` (`320d52a23e41a326287826219a20f1f29f334554`) and in the latest release commit (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`), so this is not implemented on main or obsolete.

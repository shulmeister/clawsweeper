## Summary

Keep #7834 open. Current main at 631552c554095b0c520376db9131dac0f5427f94 still does not contain the requested observability ingestor QA suite, synthetic JSONL generator, JSONL-to-SQLite integration coverage, file rotation/watch-mode tests, or large-log coverage. Existing diagnostics-otel/prometheus coverage and the OTEL QA smoke are adjacent telemetry-export coverage, not the requested ingestor pipeline tests. Draft PR #69106 appears to track this work separately and has not landed on main.

## Best Possible Solution

Keep #7834 open until PR #69106 or a maintainer-approved replacement lands deterministic observability ingestor coverage on main: synthetic channel/session JSONL fixtures, parser and ingest-to-SQLite assertions, file rotation and live watch update tests, plus a bounded large-log performance case. If maintainers decide the ingestor is not core OpenClaw, close this with an explicit ClawHub/plugin-scope decision and track the tests in that plugin.

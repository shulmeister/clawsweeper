## Summary

Keep open. The core Docker request is still not implemented on current main: the workspace is mounted only at `/home/node/.openclaw/workspace`, and the runtime Dockerfile does not create a `/workspace` symlink. The linked PR #12764 was an exact attempted fix, but it was closed unmerged by the stale bot, so it does not supersede or resolve this issue. Current AGENTS/SOUL templates no longer contain the cited `/workspace` wording, which makes the docs-friction part partly obsolete, but the requested container compatibility path remains missing.

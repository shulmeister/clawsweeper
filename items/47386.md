## Summary

Keep open. Current main has per-tool collapsible rows and a persisted show/hide toggle, but it does not implement the requested WebChat summary mode that replaces multiple tool messages with one per-turn "N tool calls" disclosure or a detailed-vs-summary setting. The related #54838 context is narrower and was also kept open because the remaining persisted tool-output expansion preference is still missing, so it does not supersede this issue.

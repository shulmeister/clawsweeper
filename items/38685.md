## Summary

Current `main` already documents the underlying Telegram group behaviors, but it does not implement this PR's actual docs ask: a consolidated multi-agent Telegram group setup example in `docs/concepts/multi-agent.md`. The existing material is split across pages, so there is not strong evidence to close this as already implemented or obsolete.

## Summary

This is best closed as a plugin/ClawHub-scope item, not an OpenClaw core issue. Current `main` treats Feishu as a plugin surface, the repo scope document says optional capability should usually ship as plugins and be promoted via ClawHub, and the current Feishu plugin/docs do not define `channels.feishu.footer.elapsed` or `channels.feishu.footer.status`. The existing issue discussion already points to a separate Lark plugin update path.

## Summary

Keep #13617 open. Current main has useful cron debugging/admin improvements, including `cron run`, `cron runs`, `cron show`, delivery previews, and `cron edit`, but it does not implement the requested dry-run/test framework for cron jobs or skills. The related closed #13605 appears to cover part of the cron CLI quality-of-life work, while the related open #41418 tracks a different global agent-tool dry-run layer and does not supersede the cron/skills testing request.

## Summary

Keep open. Current `main` still wires `message_sending` through outbound channel delivery, but the webchat final-delivery path in `src/gateway/server-chat.ts` broadcasts directly without calling the hook. The issue is also recent and the provided timeline already points to related open work in PR #38340, so there is no strong basis to auto-close it.

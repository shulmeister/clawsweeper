## Summary

Keep #71733 open. Current main does not implement the requested host-owned durable next-turn injection queue or an `agent_turn_prepare` plugin hook, and the linked RFC PR is open/docs-only rather than a merged implementation. This also does not fit a `clawhub` close because the request is explicitly for a missing core plugin API seam that feature plugins would need before they can provide Plan Mode parity.

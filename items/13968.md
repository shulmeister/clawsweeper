## Summary

Keep open. Current main has improved heartbeat bootstrap controls and the related lightContext heartbeat path appears addressed, but #13968’s requested default behavior is still not implemented: HEARTBEAT.md is not loaded strictly on-demand for heartbeat turns only. Normal/default runs can still include HEARTBEAT.md when heartbeat guidance is enabled, which is the default cadence path.

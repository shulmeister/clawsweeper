## Summary

Current main has adjacent support, but it does not implement the reported Claude Max/browser-session Opus 4.6 1M behavior well enough to close. Opus 4.6 still requires an explicit `params.context1m: true` path for 1M accounting, and the Anthropic wrapper strips the 1M beta for OAuth/legacy token auth, falling back to the standard context window. Opus 4.7/Claude CLI 1M support exists, and one commenter reported a custom-provider workaround, but that does not close the original core request.

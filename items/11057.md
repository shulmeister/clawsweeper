## Summary

Keep open. Current `main` has partial 1M-context support for Anthropic, but it does not implement the specific request in #11057 for Claude Max/browser-session style auth: Opus 4.6 still needs opt-in `params.context1m`, automatic 1M normalization is only wired for Opus 4.7, and OAuth/subscription tokens (`sk-ant-oat-*`) explicitly have the `context-1m-*` beta removed.

## Evidence

- **Automatic 1M normalization is limited to Opus 4.7:** `applyAnthropicOpus47ContextWindow` returns early unless the model is an Opus 4.7 variant, and both `resolveDynamicModel` and `normalizeResolvedModel` route through that helper.
  - file: [extensions/anthropic/register.runtime.ts:329](https://github.com/openclaw/openclaw/blob/86f69ba5a06ae256906c3a26308801e0b84e0091/extensions/anthropic/register.runtime.ts#L329)

  - file: [extensions/anthropic/index.test.ts:201](https://github.com/openclaw/openclaw/blob/86f69ba5a06ae256906c3a26308801e0b84e0091/extensions/anthropic/index.test.ts#L201)

## Summary

Keep open. Current main has partial fixes: `plugins list` now uses the cold registry snapshot and bundled runtime dependency repair checks for missing sentinels before installing. But the reported `agents bind` surface is still covered by the broad `agents` CLI plugin-load policy, and the loader still performs bundled runtime dependency installation during enabled bundled plugin loads when deps are missing. That means current main can still execute npm from `openclaw agents bind` on a fresh or missing runtime-deps stage, so the request is not fully implemented.

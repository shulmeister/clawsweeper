## Summary

Keep open. Current main has partially addressed the cron timeout dimension, but the item still has meaningful unimplemented core work: per-model timeout configuration remains absent from the strict model schema, and RAM-pressure/local process termination still lacks the requested diagnostics or RAM-aware model-loading behavior. Related #44310 remains the better tracker for the per-model timeout subproblem, but it does not cover the RAM-pressure diagnostics added by this issue.

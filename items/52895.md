## Summary

Current main does not contain this PR’s mention-pattern hardening, and the unsafe path is still reachable through runtime config overrides even though file-backed config schema validation rejects non-string `mentionPatterns`. That is not strong evidence for closure; the item should stay open for maintainer judgment or a revised patch.

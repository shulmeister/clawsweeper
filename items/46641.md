## Summary

Keep open. Current main has a useful partial implementation, `agents.defaults.compaction.notifyUser`, but it only emits brief start/end status notices and is disabled by default. It does not implement the requested configurable pre-threshold user prompt, does not wait for the user to respond with what should be preserved, and the silent memory-flush path remains model-only before the normal turn proceeds.

## Evidence

## Summary

The reported packaging bug was real for `openclaw@2026.3.28`, but current `main` now enforces bundled-plugin pack artifacts during release checks, and the shipped `v2026.4.23` npm tarball contains the previously missing files. The old `2026.3.28` artifact is immutable on npm, but the underlying packaging issue is fixed.

## Evidence

- **Published npm tarball comparison:** Streaming the published tarballs from npm showed that `openclaw@2026.3.28` lacks `package/dist/extensions/diffs/runtime-api.js`, `package/dist/extensions/image-generation-core/runtime-api.js`, and `package/dist/extensions/image-generation-core/package.json`, while `openclaw@2026.4.23` contains those paths.
  - command: `for v in 2026.3.28 2026.4.23; do url=$(npm view openclaw@$v dist.tarball); curl -Lsf "$url" | tar -tzf - | rg 'package/dist/extensions/(image-generation-core|diffs|deepgram|deepseek|groq)/(runtime-api\\.js|package\\.json)$'; done`
- **Release check now requires bundled plugin pack artifacts:** `requiredPathGroups` includes `...listBundledPluginPackArtifacts()`, so release verification now fails if packaged bundled-plugin entrypoints or package manifests are missing from the npm pack output.
  - file: [scripts/release-check.ts:54](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/scripts/release-check.ts#L54)
  - sha: [78a431ed224b](https://github.com/openclaw/openclaw/commit/78a431ed224b6f5b9d58672346551a5b411ad200)
- **Artifact enumeration covers package.json and public JS entrypoints:** The pack artifact generator adds each bundled plugin's `openclaw.plugin.json`, `package.json`, and every top-level public source entry compiled to `.js`, which is exactly the class of files missing in the broken 2026.3.28 tarball.
  - file: [scripts/lib/bundled-plugin-build-entries.mjs:148](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/scripts/lib/bundled-plugin-build-entries.mjs#L148)
  - sha: [78a431ed224b](https://github.com/openclaw/openclaw/commit/78a431ed224b6f5b9d58672346551a5b411ad200)
- **Current main still contains the missing plugin public surfaces:** The bundled plugin sources that correspond to the missing packed files are present on `main`, including the `runtime-api` entrypoints for `diffs` and `image-generation-core` and the `image-generation-core` package manifest.
  - file: [extensions/diffs/runtime-api.ts:1](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/extensions/diffs/runtime-api.ts#L1)
  - command: `rg -n "^" extensions/diffs/runtime-api.ts extensions/image-generation-core/runtime-api.ts && nl -ba extensions/image-generation-core/package.json | sed -n '1,20p'`
  - sha: [78a431ed224b](https://github.com/openclaw/openclaw/commit/78a431ed224b6f5b9d58672346551a5b411ad200)

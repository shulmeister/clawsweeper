## Summary

Keep open. Current main does not implement Anthropic server-side compaction: the config/schema surface has no `serverSide`/Anthropic compaction option, the Anthropic provider wrapper only handles beta headers for configured `anthropicBeta`/`context1m` plus service-tier wrappers, and the only `context_management` injection on main is scoped to OpenAI Responses. The related PR #10214 was closed unmerged, and the related issue #13752 reinforces the same remaining request rather than superseding it with a shipped fix.

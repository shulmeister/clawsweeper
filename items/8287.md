## Summary

Keep open. Current main still does not implement node-registered agent tools: nodes can advertise caps/commands and can be invoked through the static `nodes` tool, but there is no protocol field, registry state, or agent-tool materialization path for per-node tool definitions. The related PRs cited in discussion are adjacent plumbing only and do not supersede this request.

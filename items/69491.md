## Summary

Keep PR #69491 open. Current main still does not forward default Anthropic beta flags through the standard Amazon Bedrock Converse stream payload, so the PR is not obsolete. The branch is relevant, but it needs revision before merge because existing review comments identify real edge cases around merging existing betas and avoiding heuristic false positives.

## Best Possible Solution

Keep #69491 open and revise it narrowly in the bundled Amazon Bedrock provider: add Bedrock Converse `additionalModelRequestFields.anthropic_beta` forwarding for definitive Claude/Anthropic targets, merge with any existing beta array, preserve guardrail composition, leave non-Anthropic models untouched, and add regression tests for no-guardrail, guardrail composition, existing-beta merge, and the non-Anthropic/app-profile boundary.

## Summary

Close. Current `main` already contains a narrower implementation of this fix: source `.ts` plugin/runtime entries avoid native Jiti loading, while built `dist/*.js` entries still use native loading to preserve the canonical module graph. The same behavior is present in release `v2026.4.22`, so this PR is obsolete.

## Evidence

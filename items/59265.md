## Summary

Current `main` already contains a shipped Control UI chat fix that matches this v2026.4.1 regression: it defers same-session history reloads during an active run, keeps optimistic chat state visible, and reloads only after terminal chat events. The changelog says this shipped in `v2026.4.15`, and current tests lock that behavior in.

## Summary

Keep #72179 open. Current main at e672b61417af5c45b0431df6d9109a1f4b618ef5 still lacks the public `cron_lifecycle` hook, a hook-runner method, docs, and gateway emission for cron start/finish lifecycle events. The PR remains the canonical active implementation direction for #8209; #8855 was closed unmerged and does not supersede it.

## Best Possible Solution

Keep #72179 open as the canonical implementation for #8209. The best path is to land a focused, additive `cron_lifecycle` plugin hook that adds the typed catalog/handler/runner surface, emits gateway cron `started` and `finished` events for both `agentTurn` and `systemEvent` jobs, documents the public hook, and keeps focused gateway plus hook-runner tests. Do not close it as implemented, ClawHub-only, or superseded unless another merged PR ships the same public contract first.

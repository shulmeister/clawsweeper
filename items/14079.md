## Summary

Keep open. Current `main` still does not implement Linux desktop notifications: the Linux docs say native Linux companion apps are only planned, the node docs limit `system.notify` to the macOS node, the CLI labels `nodes notify` as mac-only, and the headless node-host dispatcher has no `system.notify` handler. Existing `system.notify` implementations are present for macOS, iOS, and Android only.

## Evidence

## Summary

Keep open. Current main still documents and implements Discord guild channels as agent-scoped session buckets, and the request for shared cross-agent Discord channel history remains missing. `allowBots="mentions"` only gates whether mentioned bot-authored messages are processed; it does not merge session stores or inject another agent's transcript. The closest multi-agent/shared-buffer feature is documented as WhatsApp-only, with Discord still planned.

## Evidence

## Summary

Current `main` does not implement this request: Android still targets `minSdk = 31`, and multiple Android code paths still assume API 29+/31 behavior. The PR therefore is not `implemented_on_main`, not obsolete, and should stay open for maintainer judgment rather than be conservatively closed.

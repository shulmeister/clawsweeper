## Summary

Keep #13607 open. Current main still supports `reasoningDefault` only as a per-agent `agents.list[]` setting; the requested global `agents.defaults.reasoningDefault` is absent from the defaults type, strict schema, generated schema/help metadata, docs, and runtime resolution path. Related implementation PRs were closed unmerged in the provided context, and the latest issue comment also explicitly kept this open.

## Best Possible Solution

Keep #13607 open until OpenClaw either implements schema-backed `agents.defaults.reasoningDefault` with precedence `directive > session > per-agent default > global default > model/default fallback`, or a maintainer explicitly decides reasoning visibility should remain per-agent only. The likely implementation should update `src/config/types.agent-defaults.ts`, `src/config/zod-schema.agent-defaults.ts`, generated schema/help/labels, `src/agents/agent-scope-config.ts`, `src/auto-reply/reply/get-reply-directives.ts`, `src/auto-reply/reply/directive-handling.levels.ts`, status/directive tests, and `docs/tools/thinking.md` / config docs.

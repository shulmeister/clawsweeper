## Summary

Keep open. Current `main` still implements `reasoningDefault` only as a per-agent setting under `agents.list[]`; the requested global `agents.defaults.reasoningDefault` is not present in the defaults type, strict Zod schema, generated config schema/help, docs, or runtime resolution path. The related PRs in the provided context were closed unmerged, and the latest maintainer review comment also explicitly kept this open.

## Best Possible Solution

Keep #13607 open until OpenClaw either implements schema-backed `agents.defaults.reasoningDefault` with the same precedence style as `thinkingDefault`, updates runtime/status/docs/tests/generated config metadata, or a maintainer explicitly decides the product should remain per-agent only. The likely code path is `src/config/types.agent-defaults.ts`, `src/config/zod-schema.agent-defaults.ts`, generated schema/help metadata, `src/agents/agent-scope-config.ts`, `src/auto-reply/reply/get-reply-directives.ts`, status/directive helper code, and `docs/tools/thinking.md`.

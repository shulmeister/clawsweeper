## Summary

Keep #13607 open. Current main at 82262387651ab270b565f71533ce4937c9c00dc6 still lacks a schema-backed global `agents.defaults.reasoningDefault`; it only supports per-agent `agents.list[].reasoningDefault`, and the main reply/status paths still resolve reasoning from session to per-agent default to `off` without consulting global defaults. The open related PR #71817 is explicitly partial per-agent Telegram/Feishu work and does not supersede this global-default request.

## Best Possible Solution

Keep #13607 open until OpenClaw adds a real `agents.defaults.reasoningDefault` contract with clear precedence: inline `/reasoning` and session override > per-agent `agents.list[].reasoningDefault` > global `agents.defaults.reasoningDefault` > model-capability/off fallback. The implementation should update config types, Zod schema, generated schema/help metadata, docs, main reply resolution, status/current-level helpers, relevant channel/plugin status paths, and regression tests. Treat #71817 as useful partial per-agent channel work, not the canonical global-default implementation.

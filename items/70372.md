## Summary

Keep PR #70372 open. Current `main` does not implement the Anthropic/Bedrock unphased-thinking tracking proposed here, and the same leak path is still plausible in source. Related PR #70344 is still open and broader, so it is not a shipped superseding fix. ([github.com](https://github.com/openclaw/openclaw/pull/70372))

## Best Possible Solution

Keep this open until maintainers either land a scoped fix here or consolidate it into #70344. The best implementation should add a regression test for unphased Anthropic/Bedrock thinking plus regular text with reasoning visibility off, preserve the existing OpenAI phase-pending guard, and scope any derived final-answer behavior to the providers/event shape that actually lack phase metadata.

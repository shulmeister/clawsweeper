## Summary

Keep #10960 open. Current main still implements steer as a conditional streaming fast path: it injects only when the active run is streaming and otherwise queues a follow-up. The requested clean pause/abort, context injection, and resume behavior for mid-generation or long tool waits is not implemented on main. Related maintainer comments closed #13207 and #13675 as duplicates of #10960, so #10960 is the canonical tracker; open PR #64758 appears to be the current implementation attempt, but it is still open and unmerged.

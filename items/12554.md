## Summary

Keep #12554 open: current main has scoped background exec supervision and per-process `process` tool controls, but still lacks the requested session-wide operator/RPC/CLI surface to list and terminate all exec children for a session.

## Best Possible Solution

Keep #12554 open and implement an additive Gateway/CLI/RPC operator surface: resolve the canonical session key, list managed exec runs for that session scope, terminate matching runs through process-supervisor cancellation plus process-tree fallback, return per-process status, document the contract, and test same-session kill, unrelated-session isolation, background exec, HTTP/RPC/CLI validation, and subagent cancellation interactions.

## Summary

Current `main` does not reproduce the PR’s stated regression for explicit isolated cron delivery targets. The fallback channel-selection path only runs when `resolveSessionDeliveryTarget(...)` failed to produce a channel, and existing tests on `main` already cover that explicit `channel`/`to` are preserved and explicit `accountId` overrides session/binding-derived values.

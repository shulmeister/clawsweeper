## Summary

PR #69232 should close as ClawHub/plugin scope. The requested `/runs` alias is for the PR's own optional Sense Worker/NemoClaw integration, not a bundled OpenClaw core surface on current main. Current plugin command APIs already support plugin-owned command aliases, and docs describe plugin commands as coming from installed/enabled plugins.

## Best Possible Solution

Close this PR from OpenClaw core and keep the Sense Worker/NemoClaw work as an external plugin path: publish or maintain it through ClawHub/npm/local plugin loading using `api.registerCommand(...)` and `nativeNames` for aliases. If an actual missing plugin API appears, open a focused SDK/API issue or PR for that seam.

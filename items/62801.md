## Summary

Keep PR #62801 open. Current main still does not expose `runtime.subagent.spawnDetached(...)` or its public SDK types/docs, and the existing native `spawnSubagentDirect`/`sessions_spawn` machinery is only available through internal agent/tool paths, not the plugin runtime API this PR proposes. The review discussion also shows the bot findings were addressed in the PR rather than proving the work is obsolete.

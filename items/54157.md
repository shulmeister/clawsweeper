## Summary

Current main still has the reported zero context-cache pricing for the bundled Volcengine/Doubao and BytePlus provider catalogs. The linked PR #55913 is open, unmerged, and uses GitHub closing syntax for this issue, so this issue should remain open until that implementation candidate is reviewed and landed or closed.

## Best Possible Solution

Keep this issue open and review PR #55913 as the implementation candidate. The fix should update `extensions/volcengine/models.ts` and `extensions/byteplus/models.ts` with source-backed nonzero `cacheRead` and `cacheWrite` pricing, correct related per-million-token prices where vendor docs support it, and add focused regression coverage in the bundled provider plugin tests.

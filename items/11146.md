## Summary

Keep #11146 open. Current main still has only the existing command-palette shortcut and lower-level agent creation APIs/CLI flow; it does not implement the requested dashboard-wide keyboard navigation or a Control UI New Agent wizard.

## Best Possible Solution

Keep #11146 open, or split it into two maintainer-tracked dashboard issues. Keyboard work should add a dashboard-level shortcut dispatcher with focus guards, wire the requested shortcuts to existing chat/navigation actions, and cover them with focused UI tests. Agent creation work should add a Control UI New Agent wizard that reuses agents.create plus agent file/config/session APIs for workspace bootstrap, SOUL.md/USER.md customization, model/session defaults, and tool/skill allowlists, with UI tests and user-facing docs.

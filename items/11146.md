## Summary

Keep #11146 open. Current main still only has the existing Cmd/Ctrl+K command palette shortcut and partial backend/CLI support for agent creation; the requested dashboard keyboard navigation and dashboard New Agent wizard are not implemented.

## Best Possible Solution

Keep #11146 open, or split it into narrower tasks: add a dashboard-level shortcut dispatcher with focus guards for chat/navigation actions, then add an Agents dashboard creation wizard that uses `agents.create`, updates workspace/bootstrap files such as SOUL.md and USER.md when requested, and optionally creates/spawns the initial session via the existing sessions APIs. Cover the new UI behavior with focused Control UI tests and update user-facing docs.

## Summary

Keep #11146 open. Current main has a partial backend `agents.create` RPC and the existing Cmd/Ctrl+K command palette shortcut, but the requested dashboard keyboard navigation is still missing and the Agents dashboard still lacks a New Agent creation wizard that generates workspace/files and spawns an initial session.

## Best Possible Solution

Keep #11146 open, or split it into narrower maintainer-owned tasks: add a dashboard-level shortcut dispatcher with focus guards for the requested chat/navigation actions, then wire a New Agent dashboard flow to the existing `agents.create` RPC and any needed session creation/spawn path, with focused UI tests and docs for the new controls.

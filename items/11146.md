## Summary

Keep open. Current `main` only partially overlaps the request: the dashboard has a `Cmd/Ctrl+K` command palette and the gateway exposes an `agents.create` RPC, but the Control UI still does not implement the requested dashboard keyboard-navigation shortcuts or a dashboard-side agent creation flow.

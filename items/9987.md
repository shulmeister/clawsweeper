## Summary

Keep #9987 open. Current main still has backend support for pending exec approval lookup via `exec.approval.list`, but the requested user-facing `openclaw approvals pending` command is absent from the approvals CLI, CLI tests, and CLI docs. The related implementation PRs #10013 and #10341 were closed unmerged, so there is no shipped CLI surface that resolves the issue.

## Best Possible Solution

Keep #9987 open until OpenClaw adds `openclaw approvals pending` in `src/cli/exec-approvals-cli.ts`, backed by the existing `exec.approval.list` gateway method. The first implementation should include human-readable table output with command details, `--json` output for monitoring, focused tests in `src/cli/exec-approvals-cli.test.ts`, and docs updates in `docs/cli/approvals.md` plus `docs/cli/index.md`; treat `--watch` as a scoped follow-up unless maintainers explicitly want it in the initial command.

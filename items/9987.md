## Summary

Keep #9987 open. Current main still does not implement the requested user-facing `openclaw approvals pending` command. The lower-level `exec.approval.list` gateway plumbing exists and is tested, but it is only used for approval replay and is not wired into CLI command registration, CLI tests, or CLI docs. The related implementation PRs #10013 and #10341 were closed unmerged.

## Best Possible Solution

Keep #9987 open until OpenClaw adds a real `openclaw approvals pending` CLI command in `src/cli/exec-approvals-cli.ts` that calls the existing `exec.approval.list` gateway method, renders useful command details in a table, supports `--json`, and adds focused CLI docs/tests. Treat `--watch` as either part of that change or an explicit follow-up if maintainers want an incremental rollout.

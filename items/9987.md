## Summary

Keep #9987 open. Current main has backend `exec.approval.list` support that can return pending approval records with command payloads, but the requested user-facing `openclaw approvals pending` CLI command, CLI tests, and CLI docs are still missing. The prior implementation attempts #10013 and #10341 are closed unmerged, so this issue remains the canonical tracker rather than a cleanup close candidate.

## Best Possible Solution

Keep #9987 open until OpenClaw adds `openclaw approvals pending` in the approvals CLI, backed by the existing `exec.approval.list` gateway method. The implementation should include human-readable table output with command details, `--json` output for monitoring/automation, focused CLI tests, and docs updates at https://docs.openclaw.ai/cli/approvals and https://docs.openclaw.ai/cli. Treat `--watch` as either part of the first implementation or a clearly linked follow-up.

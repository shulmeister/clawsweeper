## Summary

Keep #9987 open. Current main has a low-level gateway RPC, `exec.approval.list`, that can enumerate pending exec approvals, but the requested user-facing CLI surface is still missing: `openclaw approvals` only registers `get`, `set`, and `allowlist` commands, and the approvals CLI docs only document those commands. The related implementation PRs were closed unmerged, with #10341 explicitly superseding #10013 but not landing.

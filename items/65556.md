## Summary

Current `main` already implements the core request: OpenClaw ships a native bundled `github-copilot` provider with device-code auth, default setup/auth choices, automatic GitHub-to-Copilot token exchange with cached expiry handling, Copilot-specific headers, dynamic model selection, and public docs for the built-in no-proxy path.

## Evidence

- **Bundled native provider is enabled by default:** The `github-copilot` plugin is a bundled provider, enabled by default, with a first-class auth choice and env-token support (`COPILOT_GITHUB_TOKEN`, `GH_TOKEN`, `GITHUB_TOKEN`).
  - file: [extensions/github-copilot/openclaw.plugin.json:2](https://github.com/openclaw/openclaw/blob/c150110e02550eaf6c56db0c851dc62fc0ca88e1/extensions/github-copilot/openclaw.plugin.json#L2)

- **Provider owns login, setup, and runtime auth exchange:** The provider registers `github-copilot` with device-code auth, setup-wizard metadata, dynamic model resolution, and `prepareRuntimeAuth()` that exchanges the GitHub token for a Copilot API token and returns `expiresAt`/`baseUrl`.
  - file: [extensions/github-copilot/index.ts:90](https://github.com/openclaw/openclaw/blob/c150110e02550eaf6c56db0c851dc62fc0ca88e1/extensions/github-copilot/index.ts#L90)

  - file: [extensions/github-copilot/login.ts:119](https://github.com/openclaw/openclaw/blob/c150110e02550eaf6c56db0c851dc62fc0ca88e1/extensions/github-copilot/login.ts#L119)

  - file: [src/agents/github-copilot-token.ts:105](https://github.com/openclaw/openclaw/blob/c150110e02550eaf6c56db0c851dc62fc0ca88e1/src/agents/github-copilot-token.ts#L105)

  - file: [src/agents/copilot-dynamic-headers.ts:55](https://github.com/openclaw/openclaw/blob/c150110e02550eaf6c56db0c851dc62fc0ca88e1/src/agents/copilot-dynamic-headers.ts#L55)

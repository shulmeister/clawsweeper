## Summary

Keep #12198 open. Current main has some adjacent safety work for generated Skill Workshop content and installer code scanning, but it does not implement the requested Markdown validation for prompt files, does not disallow HTML comments in SKILL.md or other prompt Markdown, and ClawHub skill install still installs downloaded skill folders without that Markdown lint path.

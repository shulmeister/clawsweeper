## Summary

Keep open. Current main still does not expose or thread a `tools.exec.shell` option, and the PR discussion identified real PTY and gateway-approval integration gaps that still need design/code before the feature is merge-safe or obsolete.

## Best Possible Solution

Keep this PR or a canonical follow-up open until maintainers decide the product contract and implement shell override consistently across direct exec, PTY supervisor launches, and gateway approval follow-up runs, with config schema/types, docs, and focused regression tests updated together.

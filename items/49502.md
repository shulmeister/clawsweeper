## Summary

Current `main` does not implement the proposed `agent.wait` usage/cost metadata. The gateway wait snapshot type, dedupe reader, live agent completion payload, and `agent.wait` response path still expose only terminal status/timestamps/error, so this PR remains substantive and should stay open.

## Summary

Keep open. Current main still implements only a global `agents.defaults.models` model allowlist/catalog; per-agent config supports primary/fallback model selection and per-agent params, but not a per-agent model allowlist. Related #9010 was closed by a maintainer as a duplicate of #7424, so #7424 is the canonical remaining request, not superseded.

## Summary

Current main still lacks an ordinary per-agent model allowlist. It has global `agents.defaults.models` allowlist/catalog behavior, per-agent primary/fallback defaults, and plugin subagent `allowedModels`, but no `agents.list[]` model allowlist schema or agent-scoped model listing/switch enforcement. Related #9010 is closed as a duplicate of #7424, so #7424 remains the canonical tracker.

## Best Possible Solution

Keep #7424 open as the canonical feature request. Add an agent-scoped model allowlist, likely under `agents.list[].model.allowlist` or `agents.list[].models`, then enforce it anywhere models are listed or selected while preserving existing global `agents.defaults.models` catalog semantics and documenting the inheritance/intersection rule.

## Summary

Keep #7424 open. Current main still supports a global `agents.defaults.models` model catalog/allowlist and per-agent `agents.list[].model` primary/fallback overrides, but it does not provide an ordinary per-agent model allowlist or fully enforce/list models by agent. The related #9010 is already closed as a duplicate, so #7424 remains the canonical tracker.

## Best Possible Solution

Keep #7424 open as the canonical feature request. Add an agent-scoped model allowlist, define how it composes with `agents.defaults.models`, and enforce it across model listing and selection paths: gateway `models.list`, `/model` and `sessions.patch`, OpenAI-compatible `x-openclaw-model` overrides, Control UI model pickers, config schema/types/generated docs, and regression tests for visibility plus rejection of disallowed models.

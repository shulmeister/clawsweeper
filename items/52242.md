## Summary

Keep open. Current `main` still lacks parent-progress reporting for `runtime: "subagent"`. The only built-in progress relay is ACP-only `streamTo: "parent"`, subagent spawns explicitly reject that parameter, and default subagent tool policy still denies `sessions_send`, so the requested callback/session-key path is not implemented.

## Evidence

## Summary

Keep #57976 open. Current main does not implement Docker GPU passthrough for agent sandboxes: there is no `sandbox.docker.gpus` type/schema field, config resolution does not carry it, Docker create args never emit `--gpus`, and docs/tests only cover the existing Docker sandbox knobs. The linked PR #58124 and broader #29384 context are active related work, not shipped behavior or a conservative reason to close this focused issue early.

## Best Possible Solution

Keep #57976 open until a maintainer-reviewed implementation lands, either via PR #58124 or the broader Docker sandbox parameter work tracked by #29384. The best implementation should add an explicit `sandbox.docker.gpus` field through the public type, strict Zod schema, resolver/hash path, Docker create-arg builder, docs/config reference, and targeted tests, with clear security and host-prerequisite documentation.

## Summary

The report targets the `v2026.4.9` Dreams page layout, but current `main` and the shipped `v2026.4.12+` Dreams UI use a simplified Scene/Diary split, move grounded review into a separate Advanced surface, and now use wrap-safe layout/text styling that addresses the reported truncation/overlap mode.

## Summary

Current `main` does not implement the reported Control UI fix. The Control UI still resolves exec approvals with only `{ id, decision }`, the UI approval model still drops turn-source channel data, and the gateway schema still does not accept channel/sourceChannel fields for `exec.approval.resolve`. This issue should stay open.

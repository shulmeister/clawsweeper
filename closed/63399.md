## Summary

Current main now routes bundled MCP tools through the same effective tool-policy pipeline before they are added to an agent’s runtime tool list, with targeted tests covering `tools.allow` and `deny` behavior for bundled MCP tools. This matches the issue’s requested fix closely enough to close as implemented on main.

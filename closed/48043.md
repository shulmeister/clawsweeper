## Summary

Issue #48043 was filed against OpenClaw 2026.3.13, but current main already contains the Windows/existing-session browser-attach path that superseded the old relay flow, including explicit `userDataDir` support for Chromium profiles and attach-specific error handling. The shipped changelog marks that support in release 2026.3.22.

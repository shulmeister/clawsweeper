## Summary

Current `main` resolves embedded-run defaults to concrete `provider/model` refs before execution, and the only documented `model=default` path is a session-override reset token. I could not find a current embedded execution path that would send `openai-codex/default` to runtime.

## Evidence

- **Configured default resolves to explicit model ref:** `resolveDefaultModelForAgent()` resolves agent/default config through `resolveConfiguredModelRef(...)`, returning a concrete `provider/model` pair rather than the literal string `default`.
  - file: [src/agents/model-selection.ts:199](https://github.com/openclaw/openclaw/blob/fd65caf4b0bad1b0ac27a82d349c22e92167e7bc/src/agents/model-selection.ts#L199)

  - file: [src/auto-reply/reply/get-reply.ts:193](https://github.com/openclaw/openclaw/blob/fd65caf4b0bad1b0ac27a82d349c22e92167e7bc/src/auto-reply/reply/get-reply.ts#L193)

  - file: [src/cron/isolated-agent/model-selection.ts:45](https://github.com/openclaw/openclaw/blob/fd65caf4b0bad1b0ac27a82d349c22e92167e7bc/src/cron/isolated-agent/model-selection.ts#L45)

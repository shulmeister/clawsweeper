## Summary

Current `main` already contains LM Studio preload coordination and idempotent load checks that address the repeated model-load path discussed on the issue, with regression tests covering both concurrent preload dedupe and skip-when-already-loaded behavior. The change is on `main` but not in a tagged release yet.

## Evidence

## Summary

The issue asks for feed-list changes to an in-tree "News Collector" that does not exist on current `main`. Current project scope pushes optional RSS/feed aggregation into skills/plugins published via ClawHub, and the repo already ships an RSS-oriented `blogwatcher` skill rather than a core feed catalog.

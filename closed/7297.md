## Summary

Current main now wires both tool-call hooks, passes structured exec results through `after_tool_call`, and exposes the runtime primitives needed for a plugin to enqueue a follow-up retry turn. That matches the issue’s preferred hook-based solution well enough to close as implemented on main.

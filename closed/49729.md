## Summary

Current `main` already implements Telegram agent/media delivery: the Telegram send path accepts `mediaUrl`, sends images via `sendPhoto`, the Telegram outbound adapter forwards agent `mediaUrls`, and docs/tests cover the behavior. The implementation is also present in release `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).

## Evidence

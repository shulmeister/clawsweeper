## Summary

Current `main` no longer uses the ACPX Codex auth-bridge path this PR modifies, and the remaining env-leak concern is already covered by the generic ACP spawn path with case-insensitive secret stripping. The PR discussion usefully identified mixed-case env and sanitized-base-env edge cases, but OpenClaw now addresses those concerns through a different implementation.

## Summary

Current main already implements operator-visible Telegram feedback for tool-heavy turns and compaction, and the same behavior is present in the shipped v2026.4.22 release: tool events start/refresh typing, Telegram preview streaming reuses a live "Working…" message for tool/progress updates by default, and compaction drives explicit compacting/thinking feedback.

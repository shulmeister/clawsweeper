## Summary

PR #44798 targets legacy Chrome extension relay docs, but current `main` no longer ships that path. OpenClaw removed the extension relay/`driver: "extension"` flow and replaced host-local attach with Chrome MCP `existing-session`/`user` profiles, so this docs change no longer matches current behavior or docs structure.

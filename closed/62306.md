## Summary

Keep open. Current main has a visible-history/display mitigation for internal runtime envelopes, but the broader #62306 request is not fully implemented: the subagent announce path still builds a raw internal-context trigger, forwards it through `agent`, and the transcript persistence fallback still records the ingress `message` when no clean `transcriptMessage` is provided. The changelog also explicitly describes the current Control UI/WebChat fix as leaving the underlying transcript intact.

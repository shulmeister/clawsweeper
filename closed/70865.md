## Summary

Keep open. Current `main` still has a QQBot reminder flow that tells the model to call `cron`, while `cron` remains an owner-only core tool filtered out for non-owner senders. The linked PR #70937 in the issue timeline also suggests the fix is still in progress rather than already shipped.

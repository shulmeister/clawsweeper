## Summary

Keep open. Current `main` still hardcodes plugin/runtime default provider selection to `openai`, while OpenClaw explicitly documents `openai-codex/gpt-5.5` as a supported default-model setup. The relevant SDK and bundled-plugin paths still feed that hardcoded provider into bare-model parsing or plugin agent defaults, so there is not strong evidence the report is already fixed, non-reproducible, or incoherent.

## Evidence

## Summary

Current main routes normal model-provider HTTP requests through OpenClaw's guarded transport fetch with an explicit strict/direct dispatcher by default, so ambient NODE_USE_ENV_PROXY/HTTP_PROXY settings and CIDR-only NO_PROXY matching no longer determine ordinary LAN provider calls. Env proxy use remains an explicit request.proxy override.

## Evidence

## Summary

Current `main` already implements the Feishu multi-account/account-routed behavior requested here, including `channels.feishu.accounts`, account-scoped routing via `bindings[].match.accountId`, per-account webhook paths, and Feishu broadcast handling across multiple accounts. The implementation appears to have landed after this issue was filed and is not in the latest tagged release yet.

## Evidence

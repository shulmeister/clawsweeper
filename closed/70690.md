## Summary

The request is for a separate Python Claude Code-like CLI in OpenClaw core, but current `main` explicitly pushes optional integrations and new CLI backends to external plugins/ClawHub. OpenClaw already exposes CLI-backend and plugin extension points for this shape of integration, while the core project direction remains TypeScript-based orchestration.

## Summary

Current main already routes session-start model selection through `runWithModelFallback`, and the all-profiles-in-cooldown branch emits structured `model_fallback_decision` entries with `decision: "skip_candidate"`, reason, next candidate, and run correlation before falling through to the fallback model.

## Summary

Keep #65856 open. Current main still lacks the requested manifest metadata for Runway and Alibaba, so it is not implemented on main. The PR also has a real unresolved product/behavior question: today the auth-choice picker and CLI help only request the `text-inference` setup scope, so changing these choices to only `image-generation` would hide them from the current setup path until an image/media setup caller exists. The linked issue #65854 remains a valid, concrete bug report rather than a stale or incoherent item.

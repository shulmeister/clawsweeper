## Summary

Current `main` already contains the webchat stream-preservation, deferred history reload, and markdown fallback paths that cover disappearing structured replies, and the same implementation was already shipped in `v2026.4.23`.

## Evidence

- **Final-event fallback keeps streamed reply:** When a `chat` final event arrives, webchat appends the normalized assistant message; if that payload is empty, it falls back to the buffered `chatStream` text instead of dropping the reply.
  - file: [ui/src/ui/controllers/chat.ts:456](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/ui/src/ui/controllers/chat.ts#L456)
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Active-run history reloads are deferred:** `session.message` reloads are skipped while a run is active, then replayed after terminal chat events; final events also trigger a history reload. This avoids mid-run state resets that can make the visible reply disappear.
  - file: [ui/src/ui/app-gateway.ts:422](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/ui/src/ui/app-gateway.ts#L422)
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Regression coverage for deferred and final reloads:** Node tests cover skipping `session.message` reloads during an active run, replaying them after terminal states, and reloading chat history after final tool-output events.
  - file: [ui/src/ui/app-gateway.node.test.ts:724](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/ui/src/ui/app-gateway.node.test.ts#L724)
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Sequence state preserves later visible messages:** Gateway SSE history tests explicitly seed raw sequence state from startup snapshots, so hidden transcript entries do not desynchronize later visible assistant messages.
  - file: [src/gateway/sessions-history-http.test.ts:563](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/gateway/sessions-history-http.test.ts#L563)
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)

## Summary

Current main and shipped release v2026.4.23 already contain the agent two-stage final-response handling and one-shot gateway-client teardown changes that match this hang report.

## Evidence

  - file: [CHANGELOG.md:208](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/CHANGELOG.md#L208)

  - file: [src/gateway/call.ts:95](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/gateway/call.ts#L95)

  - file: [src/gateway/client.ts:352](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/gateway/client.ts#L352)

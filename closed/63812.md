## Summary

Current `main` already routes legacy provider-owned `tools.web.search.*` config through read-time compatibility migration and shared config persistence, so the WebUI/skills toggle path no longer has a separate stale write path that would reintroduce deprecated `tools.web.search` provider config. The shipped `v2026.4.23` tree also contains the same migration and persistence flow.

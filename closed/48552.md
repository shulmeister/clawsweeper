## Summary

Current main no longer matches the reported Docker self-build failure. Docker/package runtime plugin discovery now prefers compiled bundled plugin artifacts, the missing `plugins/runtime/index` dist entry is part of the build graph, and bundled plugin entries/runtime dependencies are produced independently of the old empty `OPENCLAW_EXTENSIONS` failure mode.

## Evidence

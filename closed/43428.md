## Close Comment

Closing after Codex review because this PR is aimed at the old Chrome extension relay flow, and current `main` no longer has that path. On `28fc03c386e14db502429bbe7e062cbe90439ca1`, `extensions/browser/src/browser/server-lifecycle.ts:9` makes the relay helper a no-op with an explicit note that the extension relay path was removed, `extensions/browser/src/browser/config.test.ts:57` asserts `chrome-relay` is no longer created, and `extensions/browser/src/browser/server.agent-contract-core.test.ts:547` rejects `driver: "extension"` as unsupported. The current supported host-local path is `existing-session` / Chrome MCP instead (`docs/gateway/doctor.md:205`). Since the code this PR modifies is obsolete relative to current `main`, this no longer matches the current behavior and is being closed. If the same redirect/tab continuity problem still exists on today’s browser paths, please open a new issue or PR against the current `existing-session` or remote CDP implementation with reproduction steps on current `main`.

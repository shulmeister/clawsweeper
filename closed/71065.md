## Summary

Current `main` already hardens the embedded runner against missing/partial assistant usage snapshots in the replay/context-accounting path and separately stabilizes isolated heartbeat session routing, which matches both the reported `TypeError` symptom and the follow-up heartbeat session-family observations.

## Summary

Current main already provides the requested lifecycle hook capability: compaction hooks (`before_compaction`/`after_compaction`), session lifecycle hooks (`session_start`/`session_end`), and documented internal compaction hook events for scriptable automation. The issue's exact JSON syntax is not the shipped API, but the underlying feature request is implemented well enough on main and is present in the latest release tree.

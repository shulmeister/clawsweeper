## Summary

Current main no longer installs the Windows gateway as a PowerShell-script Scheduled Task. The shipped Windows service path writes a `gateway.cmd` wrapper, registers the Scheduled Task against that file, uses hidden detached fallback launch on Windows, and documents the native-Windows install behavior. That makes this PowerShell-specific flashing-window report obsolete on current main.

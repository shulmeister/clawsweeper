## Summary

Current `main` already implements non-bundled skill discovery and prompt injection through a shared local loader. The loader scans `skills.load.extraDirs`, `~/.openclaw/skills`, `~/.agents/skills`, `<workspace>/.agents/skills`, and `<workspace>/skills`; `openclaw skills list/check` uses that same entry set; and both CLI and embedded runs pass the resolved skills prompt into the system prompt. The shipped `v2026.4.22` release contains this implementation.

## Summary

Current `main` already has a native in-process subagent runtime behind `sessions_spawn`'s default `runtime="subagent"` path, plus documented context/workspace inheritance controls and a dedicated in-process queue lane.

## Evidence

- **`sessions_spawn` already exposes runtime selection:** The tool schema only accepts `runtime` values `subagent` and `acp`, and execution defaults to native `subagent` when `runtime` is omitted.
  - file: [src/agents/tools/sessions-spawn-tool.ts:26](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/agents/tools/sessions-spawn-tool.ts#L26)

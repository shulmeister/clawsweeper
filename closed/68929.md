## Summary

Current `main` already implements mid-run stop/steer/redirect behavior for active chat runs, including tool-boundary steering and hard interrupt-and-restart paths, with matching docs and tests added in commit `0ec3b79c07973e746328a1d6e1be5057096029af`.

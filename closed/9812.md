## Summary

The requested per-agent request/token limiter is not implemented on current main, but the remaining capability is optional resource policy that fits the existing plugin/ClawHub path. Current plugin hooks and plugin-owned config are sufficient for an agent-scoped limiter, and the linked cost-guard plugin proposal already explored this direction.

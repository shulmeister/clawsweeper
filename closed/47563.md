## Summary

Current main already avoids the macOS arm64 sharp native-build updater failure by keeping bundled plugin runtime dependencies lazy during package postinstall and by retrying npm global updates with --omit=optional. The linked PR #47617 identified the same failure mode, but the current implementation fixes the user-facing update blocker without adding node-addon-api to root dependencies.

## Evidence

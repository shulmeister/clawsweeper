## Summary

Current `main` no longer has a Bash-based A2UI build step on Windows, so this PR's requested Git Bash note is obsolete and would now document behavior that is not in the checked-in build path.

## Evidence

  - file: [scripts/build-all.mjs:12](https://github.com/openclaw/openclaw/blob/aa21d4ea7e9aab034acbf0b13f8c3575057c5c54/scripts/build-all.mjs#L12)

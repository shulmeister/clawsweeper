## Summary

Current `main` already contains the browser/CDP loopback SSRF fix for this failure class: loopback CDP endpoints are explicitly allowed during strict SSRF validation, managed local loopback CDP control is kept outside the browser navigation SSRF policy, and regression tests cover strict-policy loopback reachability plus `browser open` `/json/new` fallback.

## Evidence

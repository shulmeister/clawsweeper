## Summary

Current main already exposes the requested native-command, skill-command, and plugin-command helpers through supported plugin-sdk subpaths, while the bare `openclaw/plugin-sdk` root is intentionally kept narrow. The PR's exact root-barrel re-export is no longer the intended SDK contract.

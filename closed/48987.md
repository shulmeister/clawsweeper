## Summary

Current `main` has partial mitigation for this report, but it does not clearly implement the requested behavior. `package.json` still keeps `node-llama-cpp` as an optional peer dependency, and the docs still describe extra manual setup for local embeddings instead of an automatic install/GPU-ready path. New doctor/status code now surfaces missing local-embedding setup instead of leaving it silent, so the issue looks narrowed, not resolved.

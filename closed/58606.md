## Summary

Issue #58606 is already implemented on current `main` and was shipped by `v2026.4.23`: sandbox browser containers now publish loopback host ports, OpenClaw resolves the mapped host CDP port before attaching, and noVNC observer access is exposed through the documented token URL flow.

## Summary

Keep open. Current main still does not implement the requested ACP transient OpenAI/Codex retry behavior, but the submitted PR is not safe to merge as-is because it bundles unrelated Slack/Agent Hub/script changes, a committed bearer credential, and review-flagged retry correctness bugs.

## Best Possible Solution

Keep the item open, but replace or heavily reduce the PR. The right fix belongs in `src/acp/control-plane/manager.core.ts` with a narrow transient classifier, abortable bounded backoff, correct runtime-handle cleanup, and replay-safe gating that refuses to retry after text/tool output or side effects. Add focused ACP manager tests and classifier tests; remove all unrelated Slack, Agent Hub, script, docs, and credential changes from this PR.

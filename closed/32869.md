## Summary

Issue #32869 requests adding or recognizing a Chinese translation of the official skills catalog. Current main already defines skill discovery, publishing, and promotion as ClawHub-owned rather than OpenClaw core, and repo docs explicitly keep localized docs output out of this source repo. The linked Chinese skills repository is useful external/community material, but it does not require a new core API or bundled core change.

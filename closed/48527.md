## Summary

Current main already hardens GitHub Copilot endpoint resolution and runtime auth so Copilot calls use the token-resolved API base URL instead of relying on stale/default host routing. That matches the reported 2026.3.13-era symptom closely enough that there is no remaining maintainer action on main.

## Evidence

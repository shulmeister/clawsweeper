## Summary

Keep #50799 open. Current main still implements the reported macOS paperclip picker as image-only: the macOS NSOpenPanel is titled for image attachments and restricts allowed content types to `.image`, and the shared chat view model rejects any attachment whose UTType does not conform to image with the user-facing error "Only image attachments are supported right now."

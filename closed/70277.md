## Summary

Current `main` contains a post-`v2026.4.23` fix for the reopened Signal replay/absolute-path variant: managed inbound files under `<stateDir>/media/inbound` are now recognized as valid inbound media references before allowed-root checks, and regression tests cover both the runner replay path and the image tool path.

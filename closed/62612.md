## Summary

Current main already resolves the observable problem behind PR #62612: the normal `pnpm check`/`pnpm tsgo` path no longer performs declaration emit for core/extensions unless explicitly requested, so the xAI TS2883 declaration-portability failures from 2026.4.6 are obsolete. The PR's exact xAI annotations are not present on main, but the failing check path has been fixed in the shared tsgo runner policy and covered by tests.

## Best Possible Solution

Close PR #62612 as obsolete on current main, keep the shared tsgo runner policy and its tests as the maintained fix for the reported check failure, and open a separate issue only if declaration-true emits for bundled extension sources become a product requirement.

## Summary

Current main implements the app-backed macOS node surface separately from the headless node-host path. The macOS app starts a node session, connects as openclaw-macos, and advertises/handles canvas, notification, screen, browser, camera, and system commands where enabled. The v2026.4.24 release tag also contains this implementation.

## Summary

Keep open. Current main still routes Telegram groups to isolated group/topic session keys, and the Telegram group config schema does not expose a per-group option to collapse a group into the agent main DM session. The related #56692 is relevant discussion about group context/routing safety, but it does not supersede this requested capability.

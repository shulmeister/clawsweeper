## Summary

Current `main` and the shipped `v2026.4.23` release already install a startup warning filter that suppresses the reported `DEP0040` punycode warning on CLI paths; the transitive `node-fetch -> whatwg-url -> tr46` chain still exists in the lockfile, but the user-visible behavior this issue reported is implemented/fixed well enough to close.

## Evidence

## Summary

Current main already handles bundled/installed plugin runtime entry resolution for Feishu/Lark-style plugins: bundled metadata is rewritten to built `.js` entrypoints, package entry validation is anchored to the plugin directory, and discovery tests cover installed package runtime loading instead of rejecting `./index.js` as escaping.

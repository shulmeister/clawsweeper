## Summary

The request is better handled as plugin/ClawHub scope, not OpenClaw core: current main explicitly assigns optional memory capability to plugins, documents memory as a plugin-owned slot, and already points users needing cross-session/channel memory plus multi-agent awareness to the Honcho plugin.

## Evidence

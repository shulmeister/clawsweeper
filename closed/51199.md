Current main and the shipped npm package no longer match the reported failure surface. The root package no longer ships the old root extensions tree or the dist/plugin-sdk/config path from the warning, and @whiskeysockets/baileys is no longer a root openclaw dependency; WhatsApp owns it as a plugin-local dependency. Release tooling now packs, installs, and post-publish verifies the npm package path, and v2026.4.20 shipped the plugin runtime dependency repair that keeps plugin dependency trees out of core installs.

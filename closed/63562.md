## Summary

Current `main` addresses the reported update-path failures: post-update plugin work is respawned from the freshly installed entrypoint, completion-cache generation no longer crashes on missing QA scenario assets, the restart path now waits for gateway health and retries stale-PID cases, and a later main-only config-reload fix removes false restart pressure from plugin update writes.

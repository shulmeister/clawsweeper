## Summary

Current main already documents and implements the Claude CLI subscription path. The remaining user problem from 2026.4.15 is covered by the Anthropic plugin's `claude-cli` auth flow, synthetic auth resolution, and docs that point users to `openclaw models auth login --provider anthropic --method cli --set-default` rather than hand-editing `auth-profiles.json`.

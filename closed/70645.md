## Summary

Close: current `main` rewrites `approval-pending` exec results into a deterministic approval payload, routes approval deliveries through a typed outbound hook, and has regression coverage for that path. The linked fix is on `main` after `v2026.4.23`, so the issue is implemented on `main` but not yet in a tagged release.

## Evidence

- **Deterministic approval payload replaces raw tool output:** `emitToolResultOutput` detects `approval-pending`, builds an exec-approval reply from structured fields (`approvalId`, `approvalSlug`, `command`, `cwd`, `host`, `nodeId`, `expiresAtMs`, `warningText`), sends it via `onToolResult`, and returns before generic tool-output emission.

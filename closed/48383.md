## Summary

Current main already provides the requested session-start lifecycle coverage and a supported pre-model context injection path. The shipped plugin API uses `session_start` plus `before_prompt_build` rather than the proposed internal `session:start` shape, but it solves the reported need for reliable runtime-owned startup context injection.

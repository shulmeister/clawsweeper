## Summary

Keep open. Current `main` mitigates the fallback side of the report by no longer classifying bodyless HTTP 400/422 responses as `format`, but it still does not strip Gemini JSON Schema conditional keywords (`if`/`then`/`else`) in `cleanSchemaForGemini()`. The issue’s proposed two-file fix is still outstanding in open PR #70497, so the underlying Gemini tool-schema incompatibility remains plausible on current `main`.

## Summary

Current `main` already implements the reported handshake-timeout fix: the server and client now share a 10s pre-auth/connect-challenge timeout, runtime overrides exist for both sides, and the changelog says this shipped in `2026.3.22` to stop busy local gateways from dropping healthy CLI connections.

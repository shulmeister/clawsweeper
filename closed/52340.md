## Summary

Current main routes community plugin discovery through ClawHub/native plugin install flows, and repo policy keeps optional community plugin discoverability out of core unless there is a stronger product/security reason. This PR's hard-coded onboarding catalog entry is better handled as a community plugin published and discovered through ClawHub.

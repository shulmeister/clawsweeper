## Summary

Close the PR as stale: current `main` already treats Feishu as a bundled channel id, so plugin auto-enable now resolves `feishu` through `channels.feishu` instead of falling back to `plugins.entries.feishu`. The same behavior is present in release `v2026.4.22`.

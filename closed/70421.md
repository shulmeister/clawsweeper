## Summary

Current `main` already handles the reported `endedAt`/`status` mismatch for subagent-style sessions: session rows now distinguish live runs from stale orphaned run records and fall back to persisted terminal session metadata instead of continuing to report `running`. The added regression test covers the exact case of a stale registry run plus persisted `endedAt`/terminal state.

## Evidence

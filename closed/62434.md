## Summary

Keep open. Current `main` does not implement this PR's backend-DOM-id aria ref resolution: `refLocator` still hard-rejects `ax...` refs from `format=aria` snapshots, and the snapshot route does not persist backend-based aria ref metadata. The PR discussion also shows a remaining P1 review concern on the latest PR commit, so there is not strong evidence for a conservative close.

## Evidence

## Summary

Current `main` already suppresses isolated cron announce delivery for exact/whitespace-only `NO_REPLY`, prevents the queued fallback summary, and documents/tests that behavior. The implementation is present on `main` in commit `1e8dc2389e424ef90eab001c17c54f0a86d50d5d`, so this PR is superseded.

## Evidence

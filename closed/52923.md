## Summary

Keep open: current `main` still does not handle numeric `usage.cost` values in `src/infra/session-cost-usage.ts`, so there is no strong basis to close this PR as already implemented or obsolete. The PR still appears to address a real gap, and the earlier review concern about the test was addressed in the PR's follow-up commit.

## Summary

Keep open. Current `main` already lets provider SDKs honor short `Retry-After` 429s, but the embedded runner still rotates on runner-classified rate-limit failures instead of retrying the same profile, and the PR discussion surfaced remaining edge cases rather than showing the work is obsolete.

## Summary

Current main implements the proxy handling requested by #44891. Gemini memory embedding calls still flow through `withRemoteHttpResponse`, and that helper now opts into `trusted_env_proxy` whenever the target URL would use `HTTP_PROXY`/`HTTPS_PROXY` and is not excluded by `NO_PROXY`. Regression tests cover both proxied and non-proxied cases. The implementation is on main after v2026.4.24, so no shipped fixed release tag was identified.

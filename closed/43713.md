## Summary

Close as `clawhub`. PR #43713 is a useful Feishu/Lark calendar integration, but it adds an optional service-specific tool to the bundled Feishu plugin. Current main does not ship `feishu_calendar`, and the current plugin API already supports external agent-tool plugins published through ClawHub/npm. The linked feature request #43696 has already been closed on the same scope basis, and related discussion pointed to `https://github.com/m1heng/clawdbot-feishu` as an existing Feishu/Lark extension source.

## Best Possible Solution

Close this OpenClaw core PR and move the useful Feishu calendar work into a maintained external plugin/ClawHub package, ideally building from the already discussed Feishu/Lark extension work and using the current `api.registerTool` plugin surface. If maintainers later want calendar in the official `@openclaw/feishu` package, reopen with explicit owner direction and address the multi-account/cache and changelog review findings first.

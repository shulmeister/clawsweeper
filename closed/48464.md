## Summary

Current main no longer matches the failure mode described in #48464: the reported tests are no longer coupled to the Windows lane, core node shards fetch the exact PR SHA before running, and the Windows pnpm cache path now uses an exact lockfile key with restore-key fallback disabled. This makes the PR #48018-specific stale-SHA/stale-cache report obsolete against current main.

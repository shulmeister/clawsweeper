## Summary

Close as ClawHub/plugin scope. The request is clear and useful, but it is optional per-conversation policy, current plugin hooks and TTS runtime surfaces are sufficient to build it outside core, and current main does not expose a core ttsByChannel-style config. The related #38673 is complementary per-agent TTS work, not a canonical duplicate.

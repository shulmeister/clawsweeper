## Summary

Current main implements the requested cross-session context path through the optional Active Memory plugin plus opt-in session transcript memory search. Active Memory runs before eligible chat replies, can be scoped to direct/group/channel sessions, and injects recalled context into the prompt. Session transcript indexing lets memory_search recall prior conversations across sessions when enabled. The feature shipped in 2026.4.12.

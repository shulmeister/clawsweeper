## Summary

Current `main` already wires plugin `before_tool_call` handling into embedded/local agent execution, preloads the plugin registry for `openclaw agent --local`, documents the behavior, and has focused embedded-mode tests. The security-motivated local hook gap described in #65025 appears implemented after the issue was filed.

## Summary

Current main already prevents the reported `doctor --fix` regression: the auto-enable path skips candidates disabled through `plugins.entries.<id>.enabled=false`, and doctor uses that same auto-enable mutation path before writing repair changes. The latest available release tag also contains this behavior.

## Evidence

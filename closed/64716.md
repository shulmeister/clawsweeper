## Summary

Current `main` no longer carries `@lancedb/lancedb` in the root `openclaw` package install path. Bundled plugin runtime dependencies are kept off npm install and repaired only when the owning plugin activates, and `memory-lancedb` now bootstraps LanceDB into user state when needed. The reported Android/Termux global install failure is therefore already addressed on `main` and documented as shipped in the changelog.

## Evidence

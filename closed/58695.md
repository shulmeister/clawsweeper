## Summary

Current `main` already filters non-self-chat `is_from_me` iMessage rows before dispatch, keeps a narrower self-chat exception only for genuine self-chat traffic, strips wrapped imsg text fields, and carries regression coverage for both behaviors. The changelog shows this shipped in `v2026.4.10`.

## Evidence

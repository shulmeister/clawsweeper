## Summary

Close as duplicate/superseded. Current main already renders base64 `data:image/...` markdown images, but still intentionally flattens remote `http(s)` markdown images to alt text and enforces a Control UI CSP that blocks remote image fetches. The remaining remote-image request is already tracked more precisely by open issue #40945, with related PR #41025.

## Evidence

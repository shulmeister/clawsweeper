## Summary

The requested prompt-risk guardrail is better scoped as a plugin/ClawHub package, not OpenClaw core. Current main already exposes plugin hooks to inspect prompts before execution and to block or require approval before tool calls, which covers the needed implementation seam without expanding core policy surface.

## Summary

The request is for ClawHub registry/CLI package lifecycle controls, not an OpenClaw core change. Current `main` only consumes ClawHub packages for install/update, and the issue is already cross-referenced to the canonical ClawHub tracker.

## Evidence

- **OpenClaw only documents plugin install/update from ClawHub:** `openclaw plugins` supports ClawHub-first install flows and recorded updates, but no owner lifecycle controls for packages.
  - file: [docs/cli/plugins.md:57](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/docs/cli/plugins.md#L57)

  - file: [docs/tools/clawhub.md:12](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/docs/tools/clawhub.md#L12)
  - sha: [7d1891e6e63d](https://github.com/openclaw/openclaw/commit/7d1891e6e63dc35dd9dcb688ba308049e68fe695)

  - file: [docs/tools/clawhub.md:223](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/docs/tools/clawhub.md#L223)
  - sha: [7d1891e6e63d](https://github.com/openclaw/openclaw/commit/7d1891e6e63dc35dd9dcb688ba308049e68fe695)

  - file: [src/infra/clawhub.ts:457](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/src/infra/clawhub.ts#L457)
  - sha: [7d1891e6e63d](https://github.com/openclaw/openclaw/commit/7d1891e6e63dc35dd9dcb688ba308049e68fe695)

## Summary

Keep PR #49770 open. Current main still has the Feishu `@_all` shortcut that marks every bot as mentioned, and the proposed regression coverage is not present. The PR remains a focused, plausible fix for the linked Feishu group-chat bug rather than something implemented or superseded on main.

## Best Possible Solution

Keep #49770 open, or land the same minimal fix: remove the raw `@_all` early return from `extensions/feishu/src/bot-content.ts` and add regression tests proving that `@_all` alone or with another user does not set `mentionedBot`, while an explicit bot `open_id` mention still does. If maintainers want opt-in broadcast behavior later, track that separately from this bug fix.

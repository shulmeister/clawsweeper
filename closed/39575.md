## Summary

Close as ClawHub/plugin scope. This PR adds an optional Feishu-specific spreadsheet reader tool to the bundled Feishu plugin, but current OpenClaw policy and SDK surfaces support shipping that as a native plugin through ClawHub/npm instead of adding another bundled tool. The PR discussion and related #43868/#65865 reviews are useful implementation signal, including the multi-account registration concern, but they do not show a missing core/plugin API seam.

## Best Possible Solution

Close this core-repo PR as ClawHub/plugin scope.

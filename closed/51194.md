## Summary

Current `main` has already moved ACPX turn execution out of the repo-local code path this PR patches. The bundled plugin now delegates `runTurn()` to the embedded `acpx/runtime` package, and the changelog shows that the extra external ACP CLI hop was removed in `v2026.4.5`, so this repo-side fix is already superseded on `main`.

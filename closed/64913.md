## Summary

Current `main` already contains a stronger version of this CI guard, so PR #64913 is superseded by work already landed on `main`. The workflow now guards both missing base commits and missing `.pre-commit-config.yaml`, falls back to `origin/${BASE_REF}`, and only falls back to the checked-out config after a warning without exporting a trusted config path.

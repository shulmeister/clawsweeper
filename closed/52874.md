## Summary

Current main already eliminates the generated bundled-plugin metadata mismatch this PR was fixing. The generated metadata pipeline is gone, bundled metadata now comes from scanning the actual bundled directories, and the copy/build paths share the same optional-cluster gate in shipped code.

The PR’s target files are no longer part of the implementation.

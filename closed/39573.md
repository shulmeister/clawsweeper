## Summary

Current main already prunes processed historical image blocks before prompt assembly and compaction sizing, which addresses the reported cache-ttl image-bloat compaction loop. The implementation keeps only the three most recent completed turns intact and replaces older user/toolResult image blocks with a text marker, with focused regression tests and changelog/docs coverage.

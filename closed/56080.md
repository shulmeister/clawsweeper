## Summary

Keep open. Current `main` still has the systemd unit-name mismatch described by the PR, and the PR itself is mixed-scope because it also adds an unrelated `tools.exec.nodeCwd` feature commit. There is not enough evidence to auto-close this as already implemented, obsolete, or unreproducible.

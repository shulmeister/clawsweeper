## Summary

Keep #62772 open. Current main has meaningful partial work for local models, especially the experimental `agents.defaults.experimental.localModelLean` flag and safer compaction defaults, but it does not implement the requested one-switch `localMode` macro. The shipped flag only trims a few heavyweight tools and docs explicitly frame it as an experimental pressure-release valve, while timeout tuning, installer/config exposure, local embedding defaults, model-size guidance, and hot-reload behavior remain separate manual/product decisions.

## Best Possible Solution

Keep the issue open for maintainer product judgment. The best path is either a documented stable local-model profile/macro that composes the existing primitives (`localModelLean`, timeout/idle-timeout settings, compaction defaults, tool allow/deny policy, and memory embedding provider selection), or a maintainer decision to split the remaining work into narrower follow-ups while leaving #66495 as the shipped partial implementation.

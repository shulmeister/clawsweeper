## Summary

Keep open. Current `main` still builds changed-check lint commands without the changed-check child env, while `scripts/run-oxlint.mjs` still relies on `OPENCLAW_OXLINT_SKIP_LOCK=1` to skip reacquiring the local heavy-check lock. The PR is a small, targeted fix with matching regression coverage and is not already implemented on `main`.

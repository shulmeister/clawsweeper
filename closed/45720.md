## Summary

Current `main` already ships Telegram Local Bot API support, including `channels.telegram.apiRoot`, probe/audit/chat-lookup routing through that root, and guarded handling for absolute Bot API `file_path` values. The shipped implementation is safer than this PR in one key area: absolute local reads are gated by `channels.telegram.trustedLocalFileRoots` instead of happening on `apiRoot` alone.

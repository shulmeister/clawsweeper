## Summary

Issue #51953 is an exact duplicate/superseded tracker for the same remaining gateway auth fix now carried by open PR #69638. Current main still only reads `connectAuth.password` in password mode, so this is not implemented-on-main; the cleanup rationale is that #69638 is the canonical active fix with matching scope and regression coverage described in its PR body.

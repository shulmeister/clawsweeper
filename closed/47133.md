## Summary

Current main has not implemented the systemd/SIGTERM fix: it still renders KillMode=control-group and the gateway run loop still handles SIGTERM as an immediate stop/restart signal with no startup guard. The remaining work is already tracked by open PR #47222, which explicitly fixes #47133 and matches the two-part fix discussed by the reporter and reproducer.

## Summary

The reported WeChat image/heartbeat race belongs to the external `@tencent-weixin/openclaw-weixin` channel plugin, not OpenClaw core. Current `main` documents WeChat as an external plugin, keeps WeChat-specific runtime ownership outside this repo, and already contains generic heartbeat queue/session-busy guards in core.

## Evidence

- **WeChat runtime is explicitly external:** The WeChat docs say OpenClaw connects through Tencent's external `@tencent-weixin/openclaw-weixin` plugin, that the WeChat code does not live in the core repo, and that login, Tencent API calls, media upload/download, context tokens, and monitoring are owned by the external plugin.

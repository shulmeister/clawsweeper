## Summary

Current main already provides the requested theming support through a browser-local Control UI custom theme slot. It does not implement arbitrary gateway.webchat.customCss loading, but it covers the user-facing need for custom colors/fonts/themes without editing core CSS, and WebChat now routes through native UI or the Control UI chat tab rather than a separate static CSS surface.

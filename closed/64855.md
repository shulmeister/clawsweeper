## Summary

Close as `clawhub`. Current main already treats prompt/context assembly as plugin-owned via the `contextEngine` slot and prompt hooks, and project direction is to keep optional behavior like context routing out of core. A conversation-type router/lightweight injector fits the existing ClawHub plugin/skill path better than a new built-in core mode.

## Evidence

- **WeChat support is external-plugin-owned:** The WeChat docs state that OpenClaw connects through Tencent's external `@tencent-weixin/openclaw-weixin` plugin, and that WeChat login, Tencent iLink API calls, media upload/download, context tokens, and account monitoring are owned by that external plugin rather than core.
  - file: [docs/channels/wechat.md:10](https://github.com/openclaw/openclaw/blob/0e9156d2053022d208c3eea1d0fa1e377ed0a154/docs/channels/wechat.md#L10)
  - sha: [0e9156d20530](https://github.com/openclaw/openclaw/commit/0e9156d2053022d208c3eea1d0fa1e377ed0a154)
- **Docs already call out multi-account WeChat isolation:** The same WeChat docs tell users adding another WeChat account to isolate DMs by account, channel, and sender with `session.dmScope per-account-channel-peer`, matching the issue's own note that two bot accounts were active for the same recipient.
  - file: [docs/channels/wechat.md:77](https://github.com/openclaw/openclaw/blob/0e9156d2053022d208c3eea1d0fa1e377ed0a154/docs/channels/wechat.md#L77)
  - sha: [0e9156d20530](https://github.com/openclaw/openclaw/commit/0e9156d2053022d208c3eea1d0fa1e377ed0a154)
- **Current main implements per-account DM isolation generically:** `per-account-channel-peer` is implemented in current main by including the account id in direct-message session keys, so the core side already has the generic isolation mechanism the docs point WeChat users to.
  - file: [src/routing/session-key.ts:157](https://github.com/openclaw/openclaw/blob/0e9156d2053022d208c3eea1d0fa1e377ed0a154/src/routing/session-key.ts#L157)
  - sha: [0e9156d20530](https://github.com/openclaw/openclaw/commit/0e9156d2053022d208c3eea1d0fa1e377ed0a154)
- **Project scope keeps optional channel capabilities in plugins:** VISION says optional capability should usually ship as plugins, with plugin hosting/maintenance outside core and plugin promotion/ownership living in ClawHub rather than the core repo.
  - file: [VISION.md:54](https://github.com/openclaw/openclaw/blob/0e9156d2053022d208c3eea1d0fa1e377ed0a154/VISION.md#L54)
  - sha: [0e9156d20530](https://github.com/openclaw/openclaw/commit/0e9156d2053022d208c3eea1d0fa1e377ed0a154)

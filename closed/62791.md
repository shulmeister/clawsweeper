## Summary

Current main contains fixes for both reported crash causes. The AJV/provider stack-overflow path is covered by the provider-hook reentrancy guard and regression tests; the late stdout/stderr `Agent listener invoked outside active run` path is covered by the exec update suppression fix and regression tests. Both fixes shipped by v2026.4.10.

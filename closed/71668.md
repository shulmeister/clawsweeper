## Summary

Keep open. Current main still has the two code shapes this PR claims to fix: the Bonjour plugin entry does not pass `registerUnhandledRejectionHandler` into the advertiser, and the unhandled-rejection registry is still a module-local `Set`. The PR discussion includes a concrete Docker/macOS crash-loop reproduction and a bot review agreeing the patch is targeted, so this looks like useful active bug-fix work rather than a conservative cleanup close candidate.

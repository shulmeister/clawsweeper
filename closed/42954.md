## Summary

Current `main` no longer has the static provider env-var map this PR edits, and it no longer exposes a `bailian` provider surface. Auth env resolution is now manifest-driven for the bundled `alibaba` provider, so this PR's `bailian`-specific patch no longer matches current behavior or code structure.

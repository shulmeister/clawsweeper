## Summary

Keep open. Current `main` does not implement the proposed `claude-cli` `context1m` override, and the existing runtime/docs/tests indicate that Claude CLI auth uses oauth/token credentials where OpenClaw strips the Anthropic `context-1m` beta and falls back to the standard window. This PR is therefore an unresolved behavior change request with concrete review feedback, not an already-fixed, obsolete, or safely auto-closeable item.

## Summary

Current `main` already supports overriding the bundled Claude CLI backend command via `agents.defaults.cliBackends.claude-cli.command`, and that support is present in the shipped `v2026.4.23` release with docs, doctor guidance, and regression coverage.

## Evidence

- **Config overrides are merged for bundled CLI backends:** `resolveCliBackendConfig` reads `cfg.agents.defaults.cliBackends`, picks the matching backend override, and merges it over the registered backend config with `...base, ...override`, which makes `command` overrideable for `claude-cli`.
  - file: [src/agents/cli-backends.ts:143](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/agents/cli-backends.ts#L143)

  - file: [src/agents/cli-backends.test.ts:576](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/agents/cli-backends.test.ts#L576)

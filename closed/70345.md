## Summary

Current `main` already has the bundled ComfyUI plugin, matching public docs, a config schema that accepts `mode` and nested `image`/`video`/`music` workflow blocks, built-in `workflow` model registration, and legacy `models.providers.comfy` fallback. The changelog shows this config-validation fix shipped in `v2026.4.23`, which post-dates the reporter's `v2026.4.21` install.

## Evidence

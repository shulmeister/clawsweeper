## Summary

Current `main` already implements the Ollama `thinkingLevel` to top-level `think` mapping through the bundled Ollama extension, with docs, regression tests, and a changelog entry covering the same behavior. This PR's old `src/agents/*` target path is stale; the live implementation now lives under `extensions/ollama/*`.

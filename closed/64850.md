## Summary

Close as implemented: current `main` and shipped release `v2026.4.23` contain the ACPX `probeAgent` fix for the backend-wide `ACP runtime backend is currently unavailable` failure when the default `codex` probe is missing or unauthenticated, which matches this qwen/qwen-code report's failure mode.

## Evidence

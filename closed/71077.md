## Summary

Current `main` contains a targeted heartbeat-session stability fix that canonicalizes repeated isolated heartbeat keys, removes stale orphaned siblings, and adds regression tests for exec-event wake re-entry and route drift. That matches the reported symptom where an async heartbeat prompt takes over one branch and later periodic heartbeats continue on a different branch.

## Evidence

## Summary

Rocket.Chat integration fits OpenClaw's external plugin/ClawHub model rather than core. Current main has no in-tree Rocket.Chat implementation, and the repo's vision/docs explicitly direct optional channel integrations to community plugins.

## Evidence

- **Vision scope keeps optional capability out of core:** OpenClaw's vision says core stays lean, optional capability should usually ship as plugins, the preferred path is npm distribution plus local extension loading, and plugin authors should host and maintain plugins in their own repositories.
  - file: [VISION.md:55](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/VISION.md#L55)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **Community plugins are the canonical path:** Community plugins can add channels, are published on ClawHub or npm, and ClawHub is the canonical discovery surface rather than core/docs PRs for discoverability.
  - file: [docs/plugins/community.md:9](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/docs/plugins/community.md#L9)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **External plugin install flow is first-class:** OpenClaw documents external community plugins as a supported model and installs them via `openclaw plugins install`, with ClawHub-first resolution.
  - file: [docs/tools/plugin.md:11](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/docs/tools/plugin.md#L11)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **Channel-specific runtime is expected to live in plugins:** The WeChat channel docs explicitly state the code does not live in the core repo, that OpenClaw provides the generic channel contract, and that the external plugin owns the channel-specific runtime while core stays channel-agnostic.
  - file: [docs/channels/wechat.md:27](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/docs/channels/wechat.md#L27)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **No Rocket.Chat implementation on current main:** A repo search on current main found no `rocketchat`, `Rocket.Chat`, or `rocket chat` matches under `docs`, `extensions`, `src`, `package.json`, or `pnpm-lock.yaml`.

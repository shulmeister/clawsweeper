## Summary

Current `main` already covers the substance of PR #56372. The allowlist mutation bug is addressed by recompiling allowlists per call with regression coverage, the git-commit context/cache bug is addressed by per-search-dir resolution with broader tests, and the PR gate flow has been superseded by the current `scripts/pr-lib/gates.sh` logic that keeps full `pnpm test` for non-docs changes and docs-only reuse. These implementations are already present in release `v2026.4.22`.

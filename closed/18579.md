## Summary

The requested Google Cloud Text-to-Speech/WaveNet/Polyglot integration is a useful optional speech provider, but current main has generic speech-provider plugin APIs, generic `messages.tts.providers.<id>` config, public media/transcode helpers, and ClawHub as the preferred discovery/review path for optional plugins. The existing bundled Google TTS is Gemini API TTS and explicitly not the separate Cloud Text-to-Speech API, so this is not implemented in core; it is plugin-scope rather than core-scope.

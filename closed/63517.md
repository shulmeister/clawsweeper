## Summary

Current `main` already supports per-agent tool restrictions via `agents.list[].tools.allow/deny/profile/byProvider`, applies that policy in the tool-filtering pipeline before tools are exposed to the model, ships regression coverage for the exact restricted-agent case, and the latest release tag `v2026.4.23` already contains the same implementation.

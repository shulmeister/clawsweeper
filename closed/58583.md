## Summary

Keep open. Current main still implements the reported 500 MB suppression behavior, the regression test explicitly asserts suppression after the cap, and the public config docs still document external log rotation as the workaround. The issue is recent, actionable, non-maintainer-authored, and has an open cross-referenced PR (#58621), so conservative cleanup should not close it.

## Summary

Current main implements durable `allow-always` exec approvals: the Control UI sends `allow-always`, gateway resolution accepts and validates it, the gateway/node execution paths persist allowlist entries with `source: "allow-always"`, and regression coverage verifies that repeated gateway commands no longer request approval. The fix shipped after the reporter's 2026.3.31 build in v2026.4.5, with the relevant merged PR #59880.

## Evidence

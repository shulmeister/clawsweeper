## Summary

Close: current `main` and the shipped `v2026.4.23` release already moved Venice model allowance to live `/models` discovery with static fallback, so newly advertised Venice model IDs are no longer limited by the bundled static catalog alone.

## Evidence

  - file: [extensions/venice/provider-catalog.ts:4](https://github.com/openclaw/openclaw/blob/bc21f500d4c14d8ecbacb992c39c7ff0954b04f2/extensions/venice/provider-catalog.ts#L4)

The static catalog is used only as fallback on discovery failure.

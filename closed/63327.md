## Summary

Current `main` no longer has the async `enqueuePersist()` / silent `.catch(() => {})` path from this PR. The bindings persistence code was replaced with direct synchronous `saveJsonFile(...)` writes that let real filesystem errors surface, so the requested fix is already superseded on `main` and there is nothing left to merge from this branch.

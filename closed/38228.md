## Summary

PR #38228 addresses a real remaining Control UI bug, but it is superseded by open PR #38876, which tracks the same #38223 tool-card array-content issue and has a broader patch that handles both array `text` and array `content` while joining all text fragments. Current main still needs the fix, so this should close as superseded, not implemented.

## Best Possible Solution

Close PR #38228 as superseded by open PR #38876. Keep #38876, or an equivalent maintainer patch, as the canonical implementation path and add regression coverage in `ui/src/ui/chat/tool-cards.node.test.ts` for array `content`, array `text`, invalid non-string text values, and multi-fragment joining.

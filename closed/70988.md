## Summary

Current `main` already reworks bundled plugin runtime-deps staging so packaged channels load from an external staged root with a regenerated `openclaw/plugin-sdk` alias, and the test suite now covers the Feishu/update path plus the reported `Cannot find package 'openclaw'` failure mode.

## Evidence

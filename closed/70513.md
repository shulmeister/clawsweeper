## Summary

Current `main` already ships the requested capability set: persistent cross-session memory files, semantic recall tools, automatic pre-compaction memory flush, scheduled multi-phase memory consolidation (`dreaming`), and a bundled `memory-wiki` knowledge layer with structured claims/evidence. This reads as a duplicate of shipped functionality rather than an unimplemented core gap.

## Evidence

It also states that `memory-wiki` adds a provenance-rich knowledge layer beside active memory.

## Summary

Current main fixes the reported stale/cumulative context-percentage problem by carrying a fresh prompt/context snapshot through session state and using that snapshot for Control UI context displays and /usage. Related unmerged PRs #51231 and #49917 proposed the same class of fix, and the prior KeepMyClaw comment correctly identified the operator-risk of stale UI state; main now addresses that in core.

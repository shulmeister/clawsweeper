## Summary

Current `main` already covers the issue’s three asks: `sessions_spawn` accepts arbitrary user labels, subagents have built-in cleanup/auto-archive lifecycle handling, and `thread: true` is no longer limited to Discord because Matrix and Feishu register subagent thread-binding hooks.

## Summary

Current main implements the requested guard for the reported `openclaw agents delete` CLI data-loss path. The command now retains same, nested, parent, or realpath-overlapping workspaces still referenced by another agent, reports that in JSON output, and has unit plus Docker CLI smoke coverage. The fix shipped in v2026.4.24 via PR #70897, which also closed later duplicate #70890.

## Evidence

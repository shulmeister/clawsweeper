## Summary

Current main makes this a plugin/memory-surface feature, not a core command change: plugin commands and skill commands can provide `/learn`, memory plugins own pre-compaction flush behavior, and plugins can observe reset/compaction lifecycle hooks.

## Risks / Open Questions

- There is not a ready-made dedicated `/learn` plugin on main today, so closing this core PR still leaves follow-up work for a plugin or skill author.
- Auto-learning before `/new`/`/reset` would still need plugin-side implementation and UX decisions around latency/noise.

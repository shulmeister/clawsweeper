## Summary

Current `main` already changes Dreaming's default storage from `inline` to `separate`, so the light/REM phase blocks that were polluting `memory/YYYY-MM-DD.md` no longer go into daily memory files unless an operator explicitly opts back into inline mode.

## Evidence

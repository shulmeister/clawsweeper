## Summary

Current `main` does not support the PR’s claim that `appPrincipal` is universally required at runtime. The code only consults it for add-on `app-url` tokens, while normal Chat issuer `app-url` traffic and `project-number` traffic work without it. Making the field globally required would reject currently valid configs and break the existing Google Chat setup flow/docs.

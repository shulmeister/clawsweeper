## Summary

Keep open. Current main still does not expose `browser.ssrfPolicy.allowRfc2544BenchmarkRange`: core SSRF supports the policy field, but browser config type/schema/help/docs and the browser resolver omit it. The PR branch also appears rewrite-only per maintainer discussion, but no allowed close reason applies because the requested capability remains valid and unsuperseded.

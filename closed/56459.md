## Summary

Current `main` already preserves cron `lightContext` bootstrap flags across embedded-agent retries, and the same implementation is present in released tag `v2026.4.22`.

## Evidence

- **Cron lightContext still maps to lightweight bootstrap:** The cron isolated-agent executor sets `bootstrapContextMode: "lightweight"` when `agentPayload.lightContext` is true and always sets `bootstrapContextRunKind: "cron"` before calling the embedded runner.
  - file: [src/cron/isolated-agent/run-executor.ts:196](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/src/cron/isolated-agent/run-executor.ts#L196)
  - command: `nl -ba src/cron/isolated-agent/run-executor.ts | sed -n '180,210p'`
  - sha: [7d1891e6e63d](https://github.com/openclaw/openclaw/commit/7d1891e6e63dc35dd9dcb688ba308049e68fe695)
- **Retry path is loop-based on current main:** `runEmbeddedPiAgent` now retries inside a `while (true)` loop, so retries reuse the same outer `params` object instead of constructing a separate recursive call that could drop fields.
  - file: [src/agents/pi-embedded-runner/run.ts:633](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/src/agents/pi-embedded-runner/run.ts#L633)
  - command: `nl -ba src/agents/pi-embedded-runner/run.ts | sed -n '620,705p'`
  - sha: [7d1891e6e63d](https://github.com/openclaw/openclaw/commit/7d1891e6e63dc35dd9dcb688ba308049e68fe695)
- **Every attempt forwards both bootstrap flags:** At the attempt call site, `bootstrapContextMode: params.bootstrapContextMode` and `bootstrapContextRunKind: params.bootstrapContextRunKind` are explicitly forwarded into `runEmbeddedAttemptWithBackend`, so retry iterations keep the same light-context settings.
  - file: [src/agents/pi-embedded-runner/run.ts:887](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/src/agents/pi-embedded-runner/run.ts#L887)
  - command: `nl -ba src/agents/pi-embedded-runner/run.ts | sed -n '840,930p'`
  - sha: [7d1891e6e63d](https://github.com/openclaw/openclaw/commit/7d1891e6e63dc35dd9dcb688ba308049e68fe695)
- **Regression coverage exists for the exact flags:** The forwarding test asserts that `bootstrapContextMode: "lightweight"` and `bootstrapContextRunKind: "cron"` are included in the params passed to the attempt runner.
  - file: [src/agents/pi-embedded-runner/run.attempt-param-forwarding.test.ts:21](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/src/agents/pi-embedded-runner/run.attempt-param-forwarding.test.ts#L21)

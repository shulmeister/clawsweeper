OpenClaw's shipped Control UI only exposes chat, channels, instances, sessions, usage, cron, skills, nodes, config, dreams, debug, and logs surfaces; there is no GitHub issue card/project-board editor or matching Gateway RPC in this repository, so the issue is not actionable against `openclaw/openclaw` as filed.

## Summary

Keep open. Current main still does not implement the legacy `CLAWDBOT_*`/`MOLTBOT_*` startup warning or the improved gateway auth error text, so PR #53667 remains a plausible focused fix for the still-open bug #53482 rather than cleanup that can be closed.

## Best Possible Solution

Keep this PR open or supersede it with an equivalent gateway diagnostic. The best path is a small main-branch fix that scans startup environment keys for `CLAWDBOT_*`/`MOLTBOT_*`, emits a clear migration warning listing key names and `OPENCLAW_*` replacements without values, updates the relevant auth/no-auth errors, and adds focused tests for warning emission, deduplication, suppression in tests, and no-warning behavior for valid `OPENCLAW_*` env vars.

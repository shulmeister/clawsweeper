## Summary

Current `main` already ships a first-class `openclaw wiki compile` surface via the bundled `memory-wiki` plugin, with docs, command wiring, bridge support for `MEMORY.md`/daily notes, and release-changelog evidence. The issue’s core claim that there is no built-in wiki-compile path is now obsolete.

## Summary

Current main already routes plugin trust checks through the source config snapshot, suppresses provenance warnings for allowlisted plugin ids, and includes a regression test for the exact runtime-vs-source allowlist mismatch described in the report. The shipped v2026.4.23 release also contains that implementation.

## Summary

Current main and the shipped v2026.4.23 release already subscribe to Telegram `callback_query` updates, handle them immediately, and intentionally forward `callback_data` into the agent as text per the documented Telegram channel contract.

## Evidence

- **Allowed updates include callback queries:** Telegram allowed-update resolution includes `callback_query` in both fallback update lists, so the bot is configured to receive inline-button callback updates.
  - file: [extensions/telegram/src/allowed-updates.ts:16](https://github.com/openclaw/openclaw/blob/86f8c826e20a01d1bba67c0ba6e73182fbf0455f/extensions/telegram/src/allowed-updates.ts#L16)

## Summary

Issue #8336 requests an optional synchronous delegated-agent tool. Current main does not implement `agent_task`, and `sessions_spawn` remains non-blocking, but the current plugin/runtime surfaces already allow this to be built as an external ClawHub plugin without a new core API.

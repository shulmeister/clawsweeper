## Summary

Close as ClawHub/plugin-scope. The useful part of this PR is an optional runtime classifier/guardrail layer, and current main already exposes the needed before_tool_call hook and plugin approval surface to build it externally. Core exec safety remains in the existing exec approvals/tool-policy paths, and the PR’s standalone helper shape was not wired into those paths and had unresolved review findings.

## Best Possible Solution

Close this core PR and route the OpenGuardrails/runtime classifier idea to a ClawHub code plugin using before_tool_call plus plugin approvals. Keep built-in exec safety in the existing exec approvals, tool policy, sandbox, and security audit paths; open a focused core/API issue only if an external plugin cannot implement a needed guardrail with the current hooks.

## Summary

Current `main` now hardens the embedded Anthropic HTTP path that this report narrowed down: embedded runs install global Undici connect tuning with a 300 ms `autoSelectFamilyAttemptTimeout` and disable `autoSelectFamily` on WSL2 to avoid IPv6 stalls. That directly matches the Windows/WSL2 slow-call pattern described in the issue comments, and the path is covered by focused tests.

## Summary

Current `main` already provides a supported direct-local `gateway-client`/`backend` path that authenticates with the shared gateway secret and retains requested operator scopes without device pairing. The report matches older `2026.3.x` behavior, but the specific gap is no longer present on `main` and is covered by current docs and regression tests.

## Summary

Issue #54438 is the same remaining feature request as canonical open issue #18160: add a direct shell/exec cron payload that bypasses LLM/agentTurn execution and can deliver output. Current main at be1d656514aa485073287cea50e81e1d0e4163cb has not implemented it; cron payloads, protocol schema, CLI, docs, and runtime dispatch still only support main-session system events or agent turns. The right cleanup is to close #54438 as a duplicate and keep discussion/implementation focused on #18160 and the open implementation attempt #47628.

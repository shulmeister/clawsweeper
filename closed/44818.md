## Summary

Current main now routes Gemini memory embedding HTTP calls through the shared memory-host remote HTTP helper, which opts into trusted env-proxy mode when HTTP_PROXY/HTTPS_PROXY will be used for the target URL. This addresses the reported Gemini memory embedding proxy gap; the change is recorded in the unreleased 2026.4.25 changelog entry for #71506.

## Evidence

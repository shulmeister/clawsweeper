## Summary

Current main still renders duplicated local-path text in media notes, but the reported functional bug for duplicate image injection does not reproduce: prompt image detection only extracts the path portion before the `|` delimiter, and main already has a regression test for that exact duplicated-path bracket expecting a single image reference.

## Evidence

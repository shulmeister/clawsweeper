## Summary

Current `main` already fixes the reported Control UI leak: the chat text extraction path now strips internal runtime-context blocks before rendering, chat history hides user messages that become empty after that sanitization, and new regression tests cover the exact leaked subagent payload scenario. The fix appears to be on `main` only and not in the latest tagged release yet.

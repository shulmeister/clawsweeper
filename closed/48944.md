## Summary

Keep open. Current main still skips sub-1024-byte audio as a `tooSmall` media-understanding skip with no synthetic transcript output, and the preflight path collapses missing transcripts to `undefined`. The current regression tests also assert the existing skipped/no-output behavior. The linked PRs #49131, #54991, and #55111 are useful and directly address the issue, but they are still open and the fix is not present on main.

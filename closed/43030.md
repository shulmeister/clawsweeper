## Summary

Closing PR #43030 because the reported downstream bug no longer reproduces on current `main`: duplicate local-path note text is still emitted, but current image-ref parsing only extracts one attachment from that bracket form and has an explicit regression test for it.

## Evidence

- **Current media-note output still includes duplicate local URL text:** `buildInboundMediaNote` still appends ` | ${urlRaw}` whenever a URL is present, and the current tests still expect `[media attached: /tmp/a.png (image/png) | /tmp/a.png]` plus the multi-file equivalents.
  - file: [src/auto-reply/media-note.ts:29](https://github.com/openclaw/openclaw/blob/41c5ffc5d5f4dc0f1df6f2c9a7efd8ce6a627f00/src/auto-reply/media-note.ts#L29)
  - command: `nl -ba src/auto-reply/media-note.ts | sed -n '1,40p'`
  - sha: [41c5ffc5d5f4](https://github.com/openclaw/openclaw/commit/41c5ffc5d5f4dc0f1df6f2c9a7efd8ce6a627f00)
- **Current parser only extracts the attachment path from `[media attached: ...]` brackets:** `detectImageReferences` parses `[media attached ...]` content, skips the `N files` header, then extracts only the path before the MIME/`|` delimiter and adds that single path ref through the deduping `addPathRef` helper.
  - file: [src/agents/pi-embedded-runner/run/images.ts:229](https://github.com/openclaw/openclaw/blob/41c5ffc5d5f4dc0f1df6f2c9a7efd8ce6a627f00/src/agents/pi-embedded-runner/run/images.ts#L229)
  - command: `nl -ba src/agents/pi-embedded-runner/run/images.ts | sed -n '229,296p'`
  - sha: [41c5ffc5d5f4](https://github.com/openclaw/openclaw/commit/41c5ffc5d5f4dc0f1df6f2c9a7efd8ce6a627f00)
- **Regression test covers the exact duplicate-local-path bracket form:** Current `main` includes a test named `does not double-count path and url in same bracket` that feeds `[media attached: /cache/IMG_6430.jpeg (image/jpeg) | /cache/IMG_6430.jpeg]` into `detectImageReferences` and expects exactly one image reference.
  - file: [src/agents/pi-embedded-runner/run/images.test.ts:165](https://github.com/openclaw/openclaw/blob/41c5ffc5d5f4dc0f1df6f2c9a7efd8ce6a627f00/src/agents/pi-embedded-runner/run/images.test.ts#L165)

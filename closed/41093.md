## Summary

Current `main` has removed the Chrome extension relay path that this PR modifies, migrated legacy `driver: "extension"` configs to Chrome MCP `existing-session`, and no longer contains `assets/chrome-extension/background.js`. The requested idle relay cleanup is therefore obsolete on `main` rather than pending review in this PR.

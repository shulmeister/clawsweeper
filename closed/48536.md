## Summary

Current main already has the build and loader changes that prevent the plugin runtime from falling back to source TypeScript on first use: the core dist graph emits `plugins/runtime/index`, the resolver prefers `dist` in production/dist contexts, and regression tests cover both behaviors. The latest locally verifiable release tag also contains the same implementation.

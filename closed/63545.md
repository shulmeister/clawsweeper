## Summary

Close as `clawhub`: the PR is a downstream-specific AIWork/PKOS bridge that is already shaped as a plugin, and current OpenClaw docs/vision say optional integrations like this should be maintained as external plugins/ClawHub packages rather than added as new bundled core plugins. The posted scaffold is useful, but it belongs outside the main repo; the current PR also still has unresolved docs/contract issues.

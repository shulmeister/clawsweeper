## Summary

Keep open. Current main still has the stale logger shape described by #62381: subsystem loggers cache a child file logger indefinitely, and the root file transport writes to the date-resolved `settings.file` captured when that logger was built. Related fix PRs exist, but the visible ones are open or closed unmerged, so this is not implemented on main.

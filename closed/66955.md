## Summary

PR #66955 should remain closed as ClawHub scope. It adds an optional third-party SenseAudio music-generation provider, and current main already exposes the plugin install, manifest contract, and registration APIs needed to publish it outside core.

## Best Possible Solution

Keep this PR closed as third-party extension scope. The best path is to publish the SenseAudio music provider as a ClawHub/npm plugin using the existing `registerMusicGenerationProvider` API and `musicGenerationProviders` manifest contract; reopen core work only if a concrete missing plugin API is identified or maintainers decide to own SenseAudio music as an official bundled capability.

## Summary

Current `main` implements subagent model selection with the expected precedence, applies the chosen model to the child session before the child run starts, and has regression tests covering both explicit `sessions_spawn.model` and `agents.defaults.subagents.model`. The same implementation is present in the released `v2026.4.23` tag.
